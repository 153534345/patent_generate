标题title
一种扩增引物、油莎豆黑色茎豆种皮鉴定用试剂盒
摘要abst
本发明涉及分子生物学及遗传育种技术领域，具体涉及一种扩增引物、油莎豆黑色茎豆种皮鉴定用试剂盒。本发明通过对油莎豆种质资源中的代表性茎豆种皮的基因组测序，选取黑色、中圆粒、大粒、中长粒为筛选样本，发掘了大批油莎豆SSR序列位点，经过筛选单拷贝位点，设计单拷贝位点扩增引物对现有的油莎豆进行筛选鉴定，发现可以特异性用于区分黑色茎豆种皮油莎豆的分子标记，并确定靶向扩增该分子标记的扩增引物，组装形成能够鉴定区分出黑色茎豆种皮油莎豆品种的快速分子检测产品。经过本发明实施例验证，本发明试剂盒能够从多样混合的油莎豆品种中准确筛选出黑色茎豆种皮品种。本发明试剂盒成本低、用时短、重复性高、结果准确。
权利要求书clms
1.一种扩增引物，其特征在于，靶向扩增如SEQ ID NO.1所示序列。2.如权利要求1所述的扩增引物，其特征在于，由正向引物和反向引物组成，正向引物序列为：5’-ATCCATTTGCATCCAACTCC-3’；反向引物序列为：5’-TCCGAAATGAGCTCAAAAGG-3’。3.一种油莎豆黑色茎豆种皮鉴定用试剂盒，其特征在于，包括如权利要求1~2任一项所述扩增引物。4.如权利要求3所述的油莎豆黑色茎豆种皮鉴定用试剂盒，其特征在于，还包括扩增反应用缓冲液。5.如权利要求4所述的油莎豆黑色茎豆种皮鉴定用试剂盒，其特征在于，所述扩增反应用缓冲液包括DNA聚合酶、dATP、dCTP、dTTP、dGTP。
说明书desc
技术领域本发明涉及分子生物学及遗传育种技术领域，具体涉及一种扩增引物、油莎豆黑色茎豆种皮鉴定用试剂盒。背景技术油莎豆起源于非洲，是近年来我国大力推广种植的新兴油料作物，它具有亩产高、健康功效好、抗逆性强、不与主粮争地等突出优点。2019年以来，为了充分保障我国植物食用油的自主供给，在不占用主粮播种面积的前提下提高油料的产量，油莎豆成为我国科技部和农业农村部推荐的新兴油源作物，科技部于2019年设立了我国首个油莎豆国家重点研发专项，进行油莎豆产业化全产业链关键技术研发，奠定了种质资源、育种和加工的基础。现有的油莎豆种质资源中，其茎豆种皮基本可以分为中圆粒、中长粒、中长粒。茎豆颜色常见为棕黄色，黑色较为少见。油莎豆不同的茎豆粒型和茎豆种皮颜色区分是油莎豆品质分类的主要表型依据之一，也是专收、专储、分类加工等产业化环节的基础。但由于油莎豆的研究基础非常薄弱，遗传学及基因组学发展滞后，至今仍没有参考基因组和重复性高的引物，不同茎豆种皮的油莎豆种质资源鉴定基本完全依赖于表型和考种鉴定。而表型鉴定存在时间周期长，鉴定结果易受环境和人为影响等问题。以DNA分子标记为代表的分子生物学技术已充分小麦、水稻、玉米等主要粮食作物的性状检测中得以运用，而油莎豆遗传学研究起步晚、基础薄。目前，油莎豆仅有2个油分合成相关基因通过同源克隆的方法被成功发掘，吉林农科院开发了数个RAPD标记对种质资源进行分类，河南省农业科学院经济作物研究所开发了部分KASP标记，以上标记尚未得到在种质资源中的充分验证，同时也无法和油莎豆的茎豆种皮性状建立联系。目前的黑色茎豆这一类重要性状的分子检测技术尚处于缺失状态，为了更好完成国家制定的科研和产业化任务，亟需通过筛选确定油莎豆黑色茎豆种皮的分子检测标记，开发出能够用于区分出油莎豆黑色茎豆种皮的分子诊断产品。发明内容为了克服现有技术的缺陷，本发明的目的之一在于提供一种扩增引物，靶向扩增如SEQ ID NO.1所示序列。本发明目的之二在于提供一种油莎豆黑色茎豆种皮鉴定用试剂盒，包括本发明提供的扩增引物，靶向扩增如SEQ ID NO.1所示序列，用于区分出黑色茎豆种皮油莎豆。为了实现上述发明目的，本发明采用的技术方案如下：一种扩增引物，靶向扩增如SEQ ID
