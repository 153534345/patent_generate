标题title
检测ATP7B基因的引物探针组合及其应用
摘要abst
本发明公开了检测ATP7B基因的引物探针组合及其应用。检测ATP7B基因的引物探针组合由引物组和探针组组成，所述引物组包括序列如SEQ ID NO:1～11所示的DNA分子、依次为以上11个DNA分子反向互补序列的DNA分子，其序列如SEQ ID NO:12～22所示；所述探针组包括序列如SEQ ID NO:23～61所示探针中的至少两种。本发明的试剂盒操作简单、敏感性高、特异性强，能准确区分不同突变类型的纯合子、杂合子、复合杂合子及正常等位基因，结果容易判断；使用简便省时，无需特殊设备，检测仅需4h，且检测结果分析相对简单，不限于专业人员使用，利于在基层单位开展。

说明书desc
技术领域本发明涉及医学诊断技术领域，尤其涉及检测ATP7B基因的引物探针组合及其应用。背景技术肝豆状核变性又称为Wilson病，是一种代谢性、遗传性疾病，属常染色体单基因隐性遗传。资料显示，此病在世界范围内不同人群的发病率为15～30/100万，患病率为1/3万，携带者频率约为1/90。此病因WD基因缺陷致ATP7B蛋白异常，肝脏向胆道排泄铜障碍，导致铜在体内聚集。由于该病可以累积神经、肝脏、血液、骨关节等全身多个系统，临床表现主要为急、慢性神经/精神症状、肾功能障碍和角膜K-F环等，既复杂又缺乏特异性。所以这些指标均存在不同程度的假阳性和假阴性，单凭任何一项指标不能确定或排除该病。因此，该病的诊断尤其是非典型病例的诊断都较为困难，也存在较高的误诊率。肝豆状核变性致病基因是ATP7B基因，ATP7B基因位于13q14.3，全长80Kb，含21个外显子和20个内含子，共编码1465个氨基酸，其编码一个P型铜转运ATP酶。该ATP酶有6个铜离子结合区、8个跨膜功能区、1个转导区、1个离子通道及磷酸化区、1个核酸结合域和1个ATP结合区。WD基因突变的位点几乎分布于基因的各个外显子，甚至有的还分布在内含子或剪接区内，突变类型包括插入突变、缺失突变、移码突变、剪接突变等，其中以点突变为主。目前已发现300余种基因突变，因此对每一个病人进行如此多的突变位点检测是比较困难的。而且相较于对整个ATP基因的检测的高昂成本，挑选致病率以及突变频率高的位点进行检测是比较科学的，也能够减少不必要的资源浪费。由于存在误诊，基因诊断就显得尤为重要。目前，ATP7B基因突变的检测方法有：RFLP/STR/SNP-家系连锁分析、PCR-SSCP分析、PCR-RFLP分析DNA测序技术等。虽然家系连锁分析可以对症状前患者进行早期诊断但却受家族史的限制，只能应用于存在肝豆状核变性先证者的家系，并且必须获得先证者父母的DNA资料，因此在应用和推广上具有一定的局限性；而且作为一种间接的诊断方法并不能检出致病突变，并受到重组风险及多态信息含量等因素的影响。PCR-SSCP分析只能作为一种突变检测方法，要最后确定突变的位置和类型，还需进一步测序；电泳条件要求较严格；另外，由于SSCP是依据点突变引起单链DNA分子立体构象的改变来实现电泳分离的，这样就可能会出现当某些位置的点突变对单链DNA分子立体构象的改变不起作用或作用很少时，再加上其他条件的影响，使聚丙烯酰胺凝胶电泳无法分辨造成漏检。PCR-RFLP分析对样品纯度要求较高，样品用量大，且RFLP多态信息含量低，多态性水平过分依赖于限制性内切酶的种类和数量，加之PCR-RFLP分析技术步骤繁琐、工作量大、成本较高，而且并不是所有的基因突变都导致酶切位点的改变，所以其应用受到了一定的限制。DNA序列测定一般是基于Sanger双脱氧链终止法的原理，准确性受PCR产物纯度影响，对嵌合体病人可能无法检测出突变。可见，传统的基因检测方法受限较多，本发明旨在突破检测人群，检测成本，检测环境的限制，使ATP7B基因的检测得到普及。发明内容本发明的首要目的在于克服现有技术的缺点与不足，提供检测ATP7B基因的引物探针组合及其应用。本发明的目的通过下述技术方案实现：检测ATP7B基因的引物探针组合，由引物组和探针组组成，所述引物组包括序列如SEQ
