标题title
一种猪繁殖与呼吸综合征病毒RT-qPCR检测引物、试剂盒及其应用
摘要abst
本发明提供了一种猪繁殖与呼吸综合征病毒RT‑qPCR检测引物、试剂盒及其应用，属于病毒检测技术领域。本发明的猪繁殖与呼吸综合征病毒RT‑qPCR检测引物如SEQ ID NO.1、SEQ ID NO.2所示，探针如SEQ ID NO.3所示。本发明的猪繁殖与呼吸综合征病毒RT‑qPCR检测试剂盒的反应体系是根据本发明扩增目的基因片段及特定引物探针序列设计的，具有更高的检测灵敏度，30min内完成RT‑qPCR扩增，检测灵敏度高达0.312copies/μL。本发明的试剂盒具有定量线性范围好、检测的抗干扰能力强、检测精密度高、稳定性强的优点。

说明书desc
技术领域本发明涉及病毒检测技术领域，尤其涉及一种猪繁殖与呼吸综合征病毒RT-qPCR检测引物、试剂盒及其应用。背景技术猪繁殖与呼吸综合征也称猪蓝耳病，是一种由猪繁殖与呼吸综合征病毒感染猪后引起病毒性传染病，该病是当前危害养猪业最为严重的疾病之一。其感染可引起猪繁殖与呼吸综合征，主要表现为母猪流产、产木乃伊胎和弱仔等繁殖障碍以及各年龄猪的呼吸系统症状，感染猪可以通过粪便、尿液、唾液和鼻腔分泌物等多种途径排毒，从而污染饲料、饮水和周围环境等，给全球养猪业造成巨大经济损失。PRRSV是动脉病毒科门动脉病毒属的一种包膜单链阳性RNA病毒，病毒基因组大小约15.4 Kb，有明显的遗传和抗原的多样性。PRRSV的出现和流行迄今已近30年，不断变异产生新毒株，但是迄今为止对PRRSV的控制在全球范围内仍未取得实质性进展，其影响和危害依旧。在实际生产中，对PRRSV不同谱系毒株进行诊断对疫情防控策略的制定十分重要。实时荧光定量PCR方法由于操作简单、检测快速是目前PRRSV检测中最为常用的方法。如王方洲等，建立了PRRSV一步法多重RT-PCR检测方法，可同时检测欧洲型和美洲型毒株。查帆等建立了检测美洲型猪繁殖与呼吸综合征病毒的荧光定量PCR方法，同时检测经典PRRSV、高致病性PRRSV和NADc30样PRRSV。与此同时，为了满足PRRSV检测的市场需求，一些企业利用荧光定量PCR技术，开发了用于PRRSV检测的商品化试剂盒。但目前的PRRSV qPCR检测技术和检测试剂盒技术水平参差不齐，在一定程度上存在操作过程复杂、检测耗时长、检测准确性低以及抗污染能力差等缺陷，很难满足PRRSV核酸检测的需求。因此，开发建立一种超快速、高精度RT-qPCR检测方法对控制和阻断PRRSV的传播有着重要意义。发明内容本发明的目的在于提供一种猪繁殖与呼吸综合征病毒RT-qPCR检测引物、试剂盒及其应用。本发明的猪繁殖与呼吸综合征病毒RT-qPCR检测试剂盒具有操作简单、检测时间短、检测准确性高以及抗污染能力强的特点。为了实现上述发明目的，本发明提供以下技术方案：本发明提供了一种猪繁殖与呼吸综合征病毒RT-qPCR检测引物和荧光探针组合，包括正向引物PRRSV-F1、反向引物PRRSV-R1和荧光探针PRRSV-P1；正向引物PRRSV-F1的核苷酸序列如SEQ ID NO.1所示；反向引物PRRSV-R1的核苷酸序列如SEQ ID NO.2所示；荧光探针PRRSV-P1的核苷酸序列如SEQ ID NO.3所示。优选的，所述荧光探针5’末端携带报告荧光基团，所述报告荧光基团为6-FAM、HEX、TET、TAMRA、 Cy3、Cy5、Cy5 .5、Texas Red、Yakima Yellow或VIC中的一种；所述荧光探针的3’末端携带淬灭荧光基团，所述淬灭荧光基团为自MGB、BHQ-1、BHQ-2或BHQ-3。本发明还提供了一种猪繁殖与呼吸综合征病毒RT-qPCR检测试剂盒，在所述RT-qPCR检测试剂盒中，包括RT-qPCR检测引物、荧光探针和qPCR酶预混液。优选地，所述qPCR酶预混液包括PCR buffer和酶混合液。PCR buffer和酶混合液可以采用本领域技术人员能够理解和使用的配方进行配制。并且，在本发明的RT-qPCR检测试剂盒中，RT-qPCR检测引物、荧光探针与qPCR酶预混液以全预混的形式存在于试剂盒中。在实际检测时，只需要将待测RNA模板直接采用具有全预混液的试剂盒进行RT-qPCR反应即可，无需将引物和qPCR酶预混液中的各成分按照次序先后放入。因此，本发明的这一RT-qPCR检测试剂盒大大简化了操作步骤，缩短了检测时间。优选地，在所述RT-PCR检测试剂盒中，每条引物的浓度独立地为0.20~0.80 μmol/L，特异性探针的浓度为0.10~0.30 μmol/L。在实际操作过程中，整个反应过程为：将待检测病毒的RNA或DNA加入上述RT-qPCR检测试剂盒中即可进行检测。反应体系为：待检测病毒的RNA或DNA 5 μL，RT-qPCR检测引物、荧光探针和qPCR酶预混液的总量为20 μL。具体地，RT-qPCR检测引物和荧光探针组中，两条引物在25μL的反应体系中的浓度各自为10 μmol/L，用量各自为1.0 μL；特异性探针的浓度为10 μmol/L，用量为0.25 μL。qPCR酶预混液包括7.5 μL的PCR buffer和4.75 μL的酶混合液，以及用于补齐余量的ddH2O。其中，更为具体地，所述PCR Buffer包括以下浓度的组分：Tris 0.2~0.3 M，KCl 45~55mM，MgCl2 23~25 mM，dNTP 0.2~0.5 mM，dUTP 0.2~0.3 mM，牛凝血酶0.5~5 μg/mL，四氢嘧啶0.4~0.6 M，单链结合蛋白1~8 ng/μL，二硫苏糖醇6~10 mM，TAA0.5~2.5 mM，蔗糖0.25~1%，DMSO 2~3% ；所述DMSO、蔗糖的百分含量为体积百分含量。所述酶混合液包括以下浓度的组分：RNA酶抑制剂2.3~2.5 U/μL；尿嘧啶糖基化酶0.4~0.6U/μL，高效逆转录酶15~17 U/μL；快速Taq酶0.1~0.2 U/μL。上述各原料可以在一定范围内进行调节，本发明并不局限于这里的具体的浓度。本发明的RT-qPCR全预混液包括RT-qPCR检测引物、荧光探针组合和qPCR酶预混液。在检测过程中，采用“一步法”即可实现RT-qPCR扩增，缩短了检测时间，具有很好的检测效果。本发明的全预混液操作简便，稳定性好，且扩增灵敏度不低于常规的分步操作。在试剂盒的实际使用过程中，RT-qPCR全预混液的体积是阴性质控品和阳性质控品的三倍。例如，一种具体的试剂盒中，RT-qPCR全预混液的规格为1100 μL/管×1管，阴性质控品的规格为350 μL/管×1管，阳性质控品的规格为350 μL/管×1管。所述试剂盒组装如下：表1试剂盒的组成本发明还提供了所述的猪繁殖与呼吸综合征病毒RT-qPCR检测引物和荧光探针组合在检测该病毒试剂盒中的应用。优选的，所述应用包括以下步骤：使用所述的RT-qPCR检测引物和荧光探针组合，以待测样本的RNA为模板，进行RT-qPCR扩增，根据扩增曲线和反应Ct值对结果进行判断，判断标准为：当FAM荧光通道中Ct≤40时，判断样品为PRRSV阳性；当FAM荧光通道中40
