标题title
从鸡卵巢组织中分离的长链非编码RNA及其应用
摘要abst
本发明公开了从鸡卵巢组织中分离的长链非编码RNA及其应用。本发明提供的长链非编码RNA的核苷酸序列为SEQ ID NO.1所示。本发明发现，将调控该长链非编码RNA表达的siRNA转入鸡卵巢颗粒细胞后干扰长链非编码RNA的表达后，调控卵巢颗粒细胞功能的基因FSHR的表达量显著降低；将其过表达后，FSHR表达量显著升高，推定其与FSHR存在靶向关系；干扰长链非编码RNA的表达后，鸡卵巢颗粒细胞凋亡数量增加，过表达后，卵巢颗粒细胞凋亡比例明显降低，表明长链非编码RNA对鸡卵巢颗粒细胞凋亡具有抑制作用。本发明所提供的长链非编码RNA可作为检测鸡卵巢颗粒细胞凋亡情况或FSHR表达的分子标志物。
权利要求书clms
1.一种从鸡卵巢颗粒细胞中分离的长链非编码RNA，其特征在于，其核苷酸序列为SEQID NO.1所示。2.干扰权利要求1所述的长链非编码RNA表达的siRNA，其特征在于，所述siRNA的正义序列的核苷酸序列为SEQ ID NO.2所示，其反义序列的核苷酸序列为SEQ ID NO.3所示。3.含有权利要求1所述的长链非编码RNA的过表达载体。4.根据权利要求3所述的表达载体，其特征在于，所述的过表达载体是慢病毒过表达载体。5.干扰或抑制权利要求1所述的长链非编码RNA表达的基因编辑载体或干扰载体。6.权利要求1所述的长链非编码RNA在调控鸡卵巢颗粒细胞的凋亡中的应用，包括：将所述的长链非编码RNA在鸡卵巢颗粒细胞中过表达抑制鸡卵巢颗粒细胞的凋亡；或者抑制或降低所述的长链非编码RNA在鸡卵巢颗粒细胞中的表达促进鸡卵巢颗粒细胞的凋亡。7.权利要求1所述的长链非编码RNA作为FSHR表达的分子标志物的应用。8.根据权利要求7所述的应用，其特征在于，包括：通过检测鸡LncRNA12078.1基因的表达量确定鸡中FSHR表达情况；如果鸡LncRNA12078.1基因的表达量高，则鸡中FSHR表达量高；如果鸡LncRNA12078.1基因的表达量低，则鸡中FSHR表达量也低。9.权利要求1所述的长链非编码RNA作为鸡卵巢颗粒细胞的凋亡状况的分子标志物的应用，包括：检测鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量；如果鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量高，则鸡卵巢颗粒细胞的凋亡比例较低；如果鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量低，则鸡卵巢颗粒细胞的凋亡比例较高。10.一种检测权利要求1所述的长链非编码RNA的荧光定量PCR试剂盒，包括：正向引物，反向引物，SYBR Green Master和ddH2O；其特征在于，所述的正向引物的核苷酸序列为SEQID NO.6所示，所述的反向引物的核苷酸序列为SEQ ID NO.7所示。
说明书desc
技术领域本发明涉及鸡长链非编码RNA，尤其涉及从鸡卵巢组织中分离的长链非编码RNA及其在调控鸡卵泡发育中的应用，属于鸡长链非编码RNA及其应用领域。背景技术卵巢是家禽重要的繁殖器官，其主要功能是产生并排出成熟的卵子、分泌性激素。卵巢的生长发育情况决定着家禽的繁殖能力和使用年限。卵泡是卵巢的基本功能单位，由卵母细胞及其周围的颗粒细胞和膜细胞组成，与鸡产蛋性能密切相关。卵泡发育成熟并排卵的关键是颗粒细胞的增殖、分化。颗粒细胞可通过分泌性腺激素及生长因子调控卵泡膜细胞和卵母细胞的生长、分化和成熟，进而精准调控卵泡生长发育，影响个体产蛋能力。据此，掌握调控卵泡发育的影响因子更有助于有效改善家禽产蛋性能。长链非编码RNA是一类长度超过200个核苷酸，具有一定功能但无编码能力的RNA。研究表明，lncRNA可通过调控对基因表达，参与家禽免疫、生长和繁殖等过程。近期研究发现，lncRNA可通过影响卵巢颗粒细胞增殖、分化、凋亡等过程影响家禽卵泡发育和排卵。因此，lncRNA的研究有助于进一步了解卵泡发育、排卵的分子机制。迄今为止，关于调控鸡卵巢颗粒细胞的lncRNA研究未见报道。发明内容本发明的目的之一是提供一种从鸡卵巢颗粒细胞中分离的长链非编码RNA；本发明的目的之二是提供干扰所述长链非编码RNA表达的siRNA；本发明的目的之三将所述的长链非编码RNA应用于包括调控鸡卵巢颗粒细胞的凋亡、或者作为FSHR的表达或鸡卵巢颗粒细胞的凋亡的分子标志物。本发明的上述目的是通过以下技术方案来实现的：本发明的一方面在于提供了一种从鸡卵巢颗粒细胞分离的长链非编码RNA，命名为LncRNA12078.1，其核苷酸序列如SEQ ID NO.1所示。本发明的另一方面是提供了干扰LncRNA12078.1表达的siRNA，所述siRNA的正义序列的核苷酸序列为SEQ ID NO.2所示，其反义序列的核苷酸序列为SEQ ID NO.3所示。本发明的另一方面是提供了含有LncRNA12078.1的过表达载体。作为本发明的一种优选的具体实施方案，所述的含有LncRNA12078.1的过表达载体是含有LncRNA12078.1的慢病毒过表达载体；本领域技术人员可以按照本领域常规的慢病毒过表达载体的构建方法构建获得含有LncRNA12078.1的慢病毒过表达载体。本发明的另一方面是提供了干扰LncRNA12078.1基因表达的基因编辑载体或干扰载体。作为本发明一种优选的具体实施方式，本领域人员可以通过常规的基因编辑技术或RNAi干扰载体的构建方法构建LncRNA12078.1基因的基因编辑载体或RNAi干扰载体。本发明的再一方面是提供了LncRNA12078.1基因在包括调控鸡卵巢颗粒细胞的凋亡、或者作为FSHR的表达或鸡卵巢颗粒细胞的凋亡的分子标志物中的应用。作为本发明的一种优选的具体实施方案，本发明提供了LncRNA12078.1基因在调控鸡卵巢颗粒细胞的凋亡中的应用，包括：将LncRNA12078.1基因在鸡卵巢颗粒细胞中过表达抑制鸡卵巢颗粒细胞的凋亡；或者抑制或降低LncRNA12078.1基因在鸡卵巢颗粒细胞中的表达促进鸡卵巢颗粒细胞的凋亡。作为本发明的一种优选的具体实施方案，本发明提供了LncRNA12078.1基因作为FSHR表达的分子标志物中的应用，包括：通过检测鸡LncRNA12078.1基因的表达量确定鸡中FSHR表达情况；如果鸡LncRNA12078.1基因的表达量高，则鸡中FSHR表达量高；如果鸡LncRNA12078.1基因的表达量低，则鸡中FSHR表达量也低。作为本发明的一种优选的具体实施方案，本发明提供了LncRNA12078.1基因作为鸡卵巢颗粒细胞的凋亡状况的分子标志物中的应用，包括：检测鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量；如果鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量高，则鸡卵巢颗粒细胞的凋亡比例较低；如果鸡卵巢颗粒细胞中LncRNA12078.1基因的表达量低，则鸡卵巢颗粒细胞的凋亡比例较高。本发明的再一方面是提供了一种检测LncRNA12078.1基因的荧光定量PCR试剂盒，包括：正向引物，反向引物，SYBR Green Master和ddH2O；其中，所述的正向引物的核苷酸序列为SEQ ID NO.6所示，所述的反向引物的核苷酸序列为SEQ ID NO.7所示。本发明在前期完成的鸡卵巢转录组测序基础上，通过对不同品种卵巢组织进行测序结果分析筛选得到的一个新的lncRNA，将其命名为鸡LncRNA12078.1；本发明通过实验分析发现，将调控 LncRNA12078.1表达的siRNA转入鸡卵巢颗粒细胞后，与对照组相比，干扰LncRNA12078.1的表达后，调控卵巢颗粒细胞功能的基因FSHR的表达量显著降低，而过表达LncRNA12078.1后，卵巢颗粒细胞中的FSHR表达量显著升高，由此推定LncRNA12078.1与FSHR存在靶向关系；另外，干扰LncRNA12078.1的表达后，鸡卵巢颗粒细胞的细胞凋亡数量增加，过表达LncRNA12078.1后，卵巢颗粒细胞凋亡比例明显降低，表明LncRNA12078.1对鸡卵巢颗粒细胞的凋亡具有抑制的调控作用，证明该lncRNA12078.1可以作为分子标志物应用于调控检测鸡卵巢颗粒细胞中细胞凋亡情况。附图说明图
