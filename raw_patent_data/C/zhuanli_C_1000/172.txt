标题title
抑制ST3GAL1基因表达的shRNA及其应用
摘要abst
本发明涉及基因工程技术领域，涉及一种抑制ST3GAL1基因表达的shRNA及其应用。所述抑制ST3GAL1基因表达的shRNA，的靶核苷酸序列如SEQ ID No:4所示。上述的shRNA能够持续、稳定、高效且特异性地抑制ST3GAL1基因表达。
权利要求书clms
1.一种抑制ST3GAL1基因表达的shRNA，其特征在于，其靶核苷酸序列如SEQ ID No:4所示。2.如权利要求1所述的shRNA，其特征在于，表达此shRNA的DNA的正义链序列如SEQ IDNo:17所示。3.一种双链寡核苷酸，其特征在于，其正义链序列如SEQ ID No:17所示，其反义链序列如SEQ ID No:18所示。4.一种抑制ST3GAL1基因表达的shRNA表达载体，其特征在于，其包括表达权利要求1或2所述的shRNA的DNA序列。5.如权利要求4所述的shRNA表达载体，其特征在于，所述的shRNA表达载体是将表达权利要求1或2所述的shRNA的DNA序列插入慢病毒载体得到的重组载体。6.如权利要求5所述的shRNA表达载体，所述慢病毒载体为pCHD-shRNA载体。7.如权利要求4～6任一项所述的shRNA表达载体，其特征在于，表达权利要求1或2所述的shRNA的DNA为权利要求3所述的双链寡核苷酸。8.一种重组细菌，其特征在于，所述细菌包含权利要求4～7任一所述的shRNA表达载体。9.如权利要求1或2所述的抑制ST3GAL1基因表达的shRNA、如权利要求4～7任一项所述的shRNA表达载体、如权利要求8所述的重组细菌在制备治疗ST3GAL1基因异常表达的药物中的应用。10.一种药物组合物，其特征在于，包括如权利要求1或2所述的抑制ST3GAL1基因表达的shRNA、或如权利要求3所述的双链寡核苷酸、或如权利要求4～7任一项所述的shRNA表达载体、或如权利要求8所述的重组细菌。
说明书desc
技术领域本发明涉及基因工程技术领域，更具体地说，它涉及一种抑制ST3GAL1基因表达的shRNA及其应用。背景技术免疫细胞疗法在癌症治疗中具有巨大的潜力，然而，实体肿瘤的复杂性和免疫抑制微环境给免疫细胞的迁移和浸润带来了困难，影响了这些疗法的临床效果。首先，实体肿瘤的微环境复杂得多，包括缺氧、低pH、营养缺乏等，这些都可能影响免疫细胞的生存和功能。此外，肿瘤细胞释放的特定分子，如免疫抑制细胞因子或血管生成因子，可以进一步抑制免疫反应。再者，免疫细胞在非靶器官的非特异性聚集也是影响免疫疗法效果的一个重要因素。在临床试验中，免疫细胞经常被注入患者体内以攻击肿瘤。然而，这些细胞往往不会直接迁移到肿瘤部位，而是首先聚集在肝脏、肺等非靶器官。这种非特异性聚集不仅降低了有效细胞的数量，还可能导致严重的副作用。最后，即使免疫细胞能够到达肿瘤，实体肿瘤的异质性和进化也使得单一的免疫疗法很难取得长期效果。部分肿瘤细胞可能具有逃逸机制，使得它们能够避免被免疫细胞识别和攻击。因此，为了提高免疫细胞疗法的临床效果，需要开发更有效的策略来克服这些挑战。这可能包括改进现有的免疫疗法，开发新的组合疗法，或者通过更深入的理解肿瘤微环境和免疫细胞的相互作用来开发新的治疗方法。ST3GAL1是一种转移性调节因子，在生物学中发挥着重要的作用。它是一种酶，能够催化唾液酸转移反应，从而改变糖链的结构。这种糖链结构的改变可以影响细胞的识别和相互作用，从而调节免疫细胞的活性。最近的研究表明，ST3GAL1对免疫细胞，特别是T细胞的迁移具有特异性调节作用。在炎症或感染的部位，ST3GAL1可以催化糖链结构的改变，使得T细胞能够更好地识别和攻击目标细胞。此外，ST3GAL1还可以调节T细胞的活化、增殖和凋亡过程，从而维持免疫系统的平衡。除了对T细胞的调节作用，ST3GAL1还可能对其他免疫细胞的功能产生影响。例如，它可以通过调节糖链的结构来影响抗原提呈细胞的功能，从而影响免疫应答的强度和特异性。此外，ST3GAL1还可能影响细胞因子的产生和信号传导，从而影响免疫细胞的活化和分化过程。值得注意的是，ST3GAL1的生物学功能可能不仅限于免疫系统。研究表明，它还可能参与肿瘤的发生和发展过程。在肿瘤细胞中，ST3GAL1可以催化糖链结构的改变，从而改变细胞的识别和相互作用，促进肿瘤细胞的迁移和侵袭。总之，ST3GAL1是一种重要的转移性调节因子，对免疫细胞的活化和分化具有重要调节作用。同时，它还可能参与肿瘤的发生和发展过程。深入了解ST3GAL1的生物学功能和作用机制将有助于我们更好地理解免疫系统和肿瘤的发生发展过程，为相关疾病的诊断和治疗提供新的思路和方法。敲除ST3GAL1可以显著增加CAR-T细胞在脾脏和淋巴结中的聚集，并促进T细胞向肿瘤组织迁移，增强TIL、TCR-T、CAR-T或CAR-NK免疫细胞疗法向肿瘤部位迁移，从而增强抗肿瘤作用。发明内容基于此，本发明的目的是提供一种可以有效、稳定地抑制ST3GAL1表达的shRNA、表达载体及表达方法，此举具有重要的应用价值。一种抑制ST3GAL1基因表达的shRNA，其靶核苷酸序列如SEQ
