标题title
抑制青花菜和花椰菜叶片发育的分子标记、引物及应用
摘要abst
本发明公开抑制青花菜和花椰菜叶片发育的分子标记、引物及应用。本发明基于青花菜和花椰菜和其他甘蓝类蔬菜CKX3基因3’非翻译区的结构变异位点开发了一种结构变异类型的分子标记，其是青花菜和花椰菜中特异的结构变异，通过分子标记检测甘蓝类蔬菜是否存在该结构变异，可快速鉴定检测材料是否为青花菜和花椰菜，进而可用于生物技术辅助育种。
权利要求书clms
1.一种抑制青花菜和花椰菜叶片发育的分子标记，其特征在于，所述分子标记为结构变异分子标记，其位于CKX3基因下游的3’非翻译区。2.根据权利要求1所述的抑制青花菜和/或花椰菜叶片发育的分子标记，其特征在于，所述分子标记具有SEQ ID NO.1所示的序列。3.一种引物组合，其特征在于，其用于扩增权利要求1或2所述的分子标记。4.根据权利要求3所述的引物组合，其特征在于，包括正向引物和反向引物。5.根据权利要求4所述的引物组合，其特征在于，所述正向引物具有SEQ ID NO.2所示的序列，所述反向引物具有SEQ ID NO.3所示的序列。6.一种用于鉴定青花菜和/或花椰菜的试剂盒，其特征在于，包括权利要求3-5任一项所述的引物组合。7.根据权利要求1或2所述的分子标记、权利要求3-5任一项所述的引物组合，或权利要求6所述的试剂盒在青花菜和/或花椰菜遗传学分析中的应用。8.根据权利要求7所述的应用，其特征在于，所述遗传学分析包括遗传多样性分析、种质鉴定或辅助育种。9.一种用于鉴定植物是否包含青花菜和/或花椰菜的方法，其特征在于，包括在待鉴定植物的DNA中检测是否存在位于CKX3基因下游3’非翻译区的结构变异分子标记的步骤，其中，所述分子标记具有SEQ ID NO.1所示的序列。10.根据权利要求9所述的用于鉴定植物是否包含青花菜和/或花椰菜的方法，其特征在于，包括以下步骤：从待鉴定植物样品中提取样品DNA；利用正向引物、反向引物通过PCR从所述样品DNA中扩增含有所述分子标记的核酸的步骤，其中所述正向引物具有SEQ ID NO.2所示的序列，所述反向引物具有SEQ ID NO.3所示的序列；和通过电泳检测样品PCR产物的长度来鉴定所述植物。
