标题title
玉米谷蛋白源抗氧化活性肽及其制备方法与应用
摘要abst
本发明提供了玉米谷蛋白源抗氧化活性肽及其制备方法与应用，属于生物活性肽技术领域。本发明提供的玉米谷蛋白源抗氧化活性肽，包括氨基酸序列如SEQ ID NO.1、SEQ ID NO.2、SEQ ID NO.3所示的肽中的至少一种。本发明提供的玉米谷蛋白源抗氧化活性肽是首次从玉米谷蛋白中分离得到的，为一种新的玉米谷蛋白源抗氧化活性肽，具有良好的抗氧化活性，可以作为天然抗氧化剂用于食品、药品和/或化妆品中。
权利要求书clms
1.玉米谷蛋白源抗氧化活性肽，其特征在于，所述玉米谷蛋白源抗氧化活性肽包括氨基酸序列如SEQ ID NO.1-3所示的肽中的至少一种。2.一种玉米谷蛋白酶解液，其特征在于，所述玉米谷蛋白酶解液含有权利要求1所述的玉米谷蛋白源抗氧化活性肽。3.权利要求2所述玉米谷蛋白酶解液的制备方法，其特征在于，包括如下步骤：用碱性蛋白酶酶解玉米谷蛋白，得到所述玉米谷蛋白酶解液；所述酶解的温度为60℃，所述酶解的时间为150min，所述酶解的pH为8.0；所述碱性蛋白酶为丹麦诺维信公司的Alcalase 2.4L；所述碱性蛋白酶的用量为每100g玉米谷蛋白添加3g碱性蛋白酶。4.权利要求1所述玉米谷蛋白源抗氧化活性肽、权利要求2所述玉米谷蛋白酶解液或权利要求3所述制备方法在制备抗氧化产品中的应用。5.根据权利要求4所述的应用，其特征在于，所述抗氧化产品包括抗氧化药品、食品抗氧化剂或化妆品。6.一种抗氧化产品，其特征在于，所述抗氧化产品的活性成分包括权利要求1所述玉米谷蛋白源抗氧化活性肽或权利要求2所述玉米谷蛋白酶解液。
说明书desc
技术领域本发明属于生物活性肽技术领域，尤其涉及玉米谷蛋白源抗氧化活性肽及其制备方法与应用。背景技术正常情况下，人体内的抗氧化防御系统能够有效的清除机体代谢产生的各种自由基，从而维持机体的动态平衡。当机体受到外界刺激时，平衡会被打破，导致机体内自由基水平升高，造成氧化应激反应。研究发现，通过外源补充具有抗氧化活性的物质就能够显著降低机体自由基水平，缓解氧化应激反应，增强机体的抗氧化能力。与合成抗氧化剂相比，天然来源的食用抗氧化剂具有易吸收、安全等优点，成为了研究热点。抗氧化肽是以蛋白质为原料，采用酶法水解的方式获得的，由多个氨基酸组成的一类物质，其具有安全、高效且价格低廉等优点。玉米谷蛋白富含疏水性氨基酸，且其蛋白序列结构中存在许多具有抗氧化活性的肽序列。因此，从玉米谷蛋白中分离具有抗氧化活性的玉米谷蛋白源抗氧化活性肽对丰富抗氧化功能食品种类，保护人类健康具有重要意义。发明内容有鉴于此，本发明的目的在于提供一种新的玉米谷蛋白源抗氧化活性肽。本发明的另一目的在于提供所述玉米谷蛋白源抗氧化活性肽的制备方法和应用。为了实现上述发明目的，本发明提供了以下技术方案：本发明提供了玉米谷蛋白源抗氧化活性肽，所述玉米谷蛋白源抗氧化活性肽包括氨基酸序列如SEQ ID NO.1、SEQ ID NO.2、SEQ ID NO.3所示的肽中的至少一种。本发明所述玉米谷蛋白源抗氧化活性肽分离自玉米谷蛋白水解物中，具有良好的抗氧化活性。本发明还提供了一种玉米谷蛋白酶解液，所述玉米谷蛋白酶解液含有上述的玉米谷蛋白源抗氧化活性肽。在本发明所述玉米谷蛋白酶解液中，优选的含有上述玉米谷蛋白源抗氧化活性肽中的两种以上。本发明还提供了上述玉米谷蛋白酶解液的制备方法，包括如下步骤：用碱性蛋白酶酶解玉米谷蛋白，得到所述玉米谷蛋白酶解液；所述酶解的温度为60℃，所述酶解的时间为150min，所述酶解的pH为8.0。本发明对于玉米谷蛋白和碱性蛋白酶的具体来源没有特殊限定，采用本领域常规市售产品即可。在本发明中，所述碱性蛋白酶的用量以玉米谷蛋白的重量计，为每100g玉米谷蛋白添加3g碱性蛋白酶。本发明还提供了一种玉米谷蛋白源抗氧化活性肽的制备方法，在上述制备方法的基础上，还包括如下步骤：对所述玉米谷蛋白酶解液进行分离纯化，收集小于1000Da且抗氧化活性最高的组分，进行质谱测序，经数据库序列比对，化学合成即得上述玉米谷蛋白源抗氧化活性肽。在本发明中，所述分离纯化的方式优选的包括凝胶色谱分离、离子交换色谱分离和反向色谱分离；所述凝胶色谱分离的填料为Sephadex G-25；所述离子交换色谱分离的填料为Q-Sepharose Fast Flow；所述反向色谱分离的色谱柱为Pronto SIL C18。本发明优选的采用LC-MS/MS进行质谱测序。本发明对质谱测序的过程和步骤没有特殊限定，采用本领域中常规质谱测序步骤即可。本发明还提供了上述玉米谷蛋白源抗氧化活性肽、上述玉米谷蛋白酶解液、上述玉米谷蛋白源抗氧化活性肽制备方法或上述玉米谷蛋白酶解液制备方法在制备抗氧化产品中的应用。在本发明中，所述抗氧化产品优选的包括抗氧化药品、食品抗氧化剂或化妆品。本发明还提供了一种抗氧化产品，所述抗氧化产品的活性成分包括上述玉米谷蛋白源抗氧化活性肽或上述玉米谷蛋白酶解液。在本发明中，所述抗氧化产品优选的还包括辅料和/或其他活性成分，本发明对于所述辅料和/或其他活性成分的种类没有特殊限定，采用所要制备的产品常规辅料或其他活性成分即可。本发明的有益效果：本发明提供的玉米谷蛋白源抗氧化活性肽是首次从玉米谷蛋白中分离得到的，为一种新的玉米谷蛋白源抗氧化活性肽，具有良好的抗氧化活性，可以作为天然抗氧化剂用于食品、药品和/或化妆品中。附图说明图1为本发明实施例的技术路线图。具体实施方式下面结合实施例对本发明提供的技术方案进行详细的说明，但是不能把它们理解为对本发明保护范围的限定。下述实施例中，如无特殊说明，均为常规方法。下述实施例中所用的材料、试剂等，如无特殊说明，均可从商业途径得到。实施例11、玉米谷蛋白酶解液的制备将玉米谷蛋白加入到pH
