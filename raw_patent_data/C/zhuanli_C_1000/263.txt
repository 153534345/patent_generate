标题title
与小麦穗长相关的分子标记及其应用
摘要abst
本发明公开了与小麦穗长相关的分子标记及其应用，属于作物选种培育技术领域。所述分子标记为MTA110‑MseI和MTA94‑AluI，二者分别标记与小麦穗长显著主效相关SNP位点AX‑111500116和AX‑111501563，前者的核苷酸序列如SEQ ID NO:9或SEQ ID NO:10所示，对应的AX‑111500116位点的基因型为AA或BB；后者的核苷酸序列如SEQ ID NO:12或SEQ ID NO:13所示，对应的AX‑111501563位点的基因型为AA或BB。此两个分子标记可广泛用于小麦穗长的分子标记辅助育种及全基因组选择育种，加快育种进程。

说明书desc
技术领域本发明涉及分子标记及其应用，具体涉及与小麦穗长相关的分子标记及其应用，属于作物选种培育技术领域。背景技术小麦作为全球最重要的粮食作物之一，其产量和质量一直备受农业科学家的关注。全基因组关联分析是当今遗传资源挖掘的重要方式，通过对大规模基因型和表型数据的整合，全基因关联分析帮助研究人员确定与数量显著关联的位点，揭示了各性状的复杂遗传基础。小麦穗长是影响产量的关键农艺性状之一，直接关系到每穗小麦的籽粒数量和分布，从而影响总产量。因此，了解小麦穗长的遗传基础、生长调控机制以及分子标记的应用成为小麦遗传改良和育种的关键研究领域。小麦的穗长性状受到多种因素调控，包括遗传和环境因素，需要更深入的研究来全面理解这一性状。目前，在小麦穗长研究中，关于小麦穗部相关分子标记的报道仍然较少。发明内容本发明的目的在于提供与小麦穗长相关的分子标记及其应用。为了实现上述目标，本发明采用如下的技术方案：与小麦穗长相关的分子标记，所述分子标记为MTA110-MseI和MTA94-AluI，二者分别标记与小麦穗长显著主效相关SNP位点AX-111500116和AX-111501563，AX-111500116位于4A染色体，物理位置为41899333，基因型为AA/BB，AX-111501563位于3B染色体，物理位置为767340200，基因型为AA/BB，其中：所述MTA110-MseI是由SEQ ID NO:1和SEQ ID NO:2所示的一次PCR引物以及SEQID NO:3和SEQ ID NO:4所示的二次PCR引物对待测小麦进行一次PCR扩增和二次PCR扩增所得产物经限制性内切酶MseI酶切后得到的，核苷酸序列如SEQ ID NO:9所示，对应的AX-111500116位点的基因型为AA，或者核苷酸序列如SEQ ID NO:10和SEQ ID NO:11所示，对应的AX-111500116位点的基因型为BB；所述MTA94-AluI是由SEQ ID NO:5和SEQ ID NO:6所示的一次PCR引物以及SEQ IDNO:7和SEQ ID NO:8所示的二次PCR引物对待测小麦进行一次PCR扩增和二次PCR扩增所得产物经限制性内切酶AluI酶切后得到的，核苷酸序列如SEQ ID RefSeq v2.1，AX-111500116位于4A染色体，物理位置为41899333，基因型为AA/BB；AX-111501563位于3B染色体，物理位置为767340200，基因型为AA/BB。二、分子标记开发基于AX-111500116、AX-111501563两个位点变异，结合中国春小麦的参考基因组序列 RefSeq v2.1，使用小麦联盟的CAPS/dCAPS引物设计工具，对前面所检测到的2个SNP位点分别设计引物。针对AX-111500116，所设计的一次PCR引物为MTA110-MseI-1-F/R、二次PCR引物为MTA110-MseI-2-F/R；针对AX-111501563，所设计的一次PCR引物为MTA94-AluI-1-F/R、二次PCR引物为MTA94-AluI-2-F/R。上述各引物的核苷酸序列具体如下：MTA110-MseI-1-F：AGGGGGTCTGTTTAATTACGGTMTA110-MseI-1-R：CCGCGTTGGGGTTATCTTTACMTA110-MseI-2-F：ggacccaaacaccaccttcaMTA110-MseI-2-R：accgcgttggggttatctttMTA94-AluI-1-F：TACCCCAATTCCATCTCGCCMTA94-AluI-1-R：ATTAGCCTTCTCTCAGCGCGMTA94-AluI-2-F：tcagtgaacctacgaaaatattAgcMTA94-AluI-2-R：aaagagccccgaagttccag利用上述一次PCR引物MTA110-MseI-1-F/R和二次PCR引物MTA110-MseI-2-F/R对待测小麦的DNA进行一次PCR扩增和二次PCR扩增，二次PCR扩增产物经限制性内切酶酶切后，即可得到对应的分子标记MTA110-MseI。利用上述一次PCR引物MTA94-AluI-1-F/R和二次PCR引物MTA94-AluI-2-F/R对待测小麦的DNA进行一次PCR扩增和二次PCR扩增，二次PCR扩增产物经限制性内切酶酶切后，即可得到对应的分子标记MTA94-AluI。三、自然群体检测选取243份小麦自然群体的DNA，对MTA110-MseI和MTA94-AluI两个分子标记进行检测，分别得到AX-111500116、AX-111501563两个位点的基因型。1、PCR扩增AX-111500116位点利用一次PCR引物MTA110-MseI-1-F和MTA110-MseI-1-R对待测小麦品种的DNA进行一次PCR扩增得到PCR产物A1，PCR扩增体系为10μL，具体有：1μL DNA模板、0.5μL上游引物、0.5μL下游引物、5μL 2×Taq PCR预混试剂、3μL ddH2O；采用普通扩增程序扩增，步骤如下：95℃变性5min；34个循环：95℃变性30s，58℃复性30s，72℃延伸1min；72℃延伸10min；结束扩增，4℃保存。将产物A1稀释1000倍后使用二次PCR引物MTA110-MseI-2-F和MTA110-MseI-2-R进行二次PCR扩增，扩增体系和扩增程序与一次PCR扩增一致，得到产物A2。AX-111501563位点利用一次PCR引物MTA94-AluI-1-F和MTA94-AluI-1-R对待测小麦品种的DNA进行一次PCR扩增得到PCR产物B1，PCR扩增体系为10μL，具体有：1μL DNA模板、0.5μL上游引物、0.5μL下游引物、5μL 2×Taq PCR预混试剂、3μL ddH2O；采用普通扩增程序扩增，步骤如下：95℃变性5min；34个循环：95℃变性30s，58℃复性30s，72℃延伸1min；72℃延伸10min；结束扩增，4℃保存。将产物B1稀释1000倍后使用二次PCR引物MTA94-AluI-2-F和MTA94-AluI-2-R进行二次PCR扩增，扩增体系和扩增程序与一次PCR扩增一致，得到产物B2。2、酶切AX-111500116位点利用限制性内切酶MseI对PCR产物A2进行酶切，酶切体系为10μL，具体有：3μL PCR产物A1、0.2μL限制性内切酶MseI、1μL 10×CutSmart Buffer缓冲液、5.8μL ddH2O；反应条件为37℃孵育3-4h，得到酶切产物A3。AX-111501563位点利用限制性内切酶AluI对PCR产物B2进行酶切，酶切体系为10μL，具体有：3μL PCR产物A1、0.2μL限制性内切酶AluI、1μL 10×CutSmart Buffer缓冲液、5.8μL ddH2O；反应条件为37℃孵育3-4h，得到酶切产物B3。3、聚丙烯酰胺凝胶电泳单独电泳检测将酶切产物A3或B3放在6.0%的非变性聚丙烯酰胺凝胶中进行稳压电泳分离2h。如果酶切产物A3的大小为433bp，那么待测小麦在位点AX-111500116上的基因型为AA，记为A。如果酶切产物A3的大小为372bp和61bp，那么待测小麦在位点AX-111500116上的基因型为BB，记为B。如果酶切产物B3的大小为187bp，那么待测小麦在位点AX-111501563上的基因型为AA，记为A。如果酶切产物B3的大小为168bp和19bp，那么待测小麦在位点AX-111501563上的基因型为BB，记为B。混合电泳检测将酶切产物A3和B3以1:1的体积比混合，然后放在6.0%的非变性聚丙烯酰胺凝胶中进行稳压电泳分离2h。对于AX-111500116位点，如果酶切产物存在433bp的片段且不存在372bp和61bp的片段，那么待测小麦在该位点的基因型为A；如果酶切产物的大小为372bp和61bp，那么待测小麦在该位点的基因型为B。对于AX-111501563位点，如果酶切产物存在187bp的片段且不存在168bp和19bp的片段，那么待测小麦在该位点的基因型为A；如果酶切产物的大小为168bp和19bp，那么待测小麦在该位点的基因型为B。混合电泳检测部分电泳鉴定结果如图1所示，所有鉴定结果见表1。表1
