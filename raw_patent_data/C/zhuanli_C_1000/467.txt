标题title
一种适宜工厂化栽培的金黄色绣球菌菌株及其栽培方法
摘要abst
一种适宜工厂化栽培的金黄色绣球菌菌株及其栽培方法，属于食用菌育种领域，保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏编号为CGMCC No.40890，菌株为JSJ‑Spar8‑1C，分类命名为绣球菌Sparassis sp.，其ITS序列如SEQ ID NO：1所示，该菌株是广叶绣球菌。菌株的特征DNA序列片段如SEQ ID NO：2所示。本发明的菌株菌丝生长快、抗杂菌能力强，原基健壮、出菇率高达85%及以上，生物学转化率高达67.05%及以上，生长发育周期短，颜色为金黄色，商品性状良好，在育种和产业化生产栽培等方面具有很好的应用前景。

1.一种适宜工厂化栽培的金黄色绣球菌菌株，保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏编号为CGMCC No.40890 ，菌株为JSJ-Spar8-1C，分类命名为绣球菌Sparassis sp.，其ITS序列如SEQ ID NO：1所示，通过ITS序列比对，该菌株是广叶绣球菌，隶属于多孔菌目、绣球菌科、绣球菌属。2.如权利要求1所述的一种适宜工厂化栽培的金黄色绣球菌菌株，所述菌株具有区别于其他菌株的特征DNA序列，所述特征DNA序列片段如SEQ ID NO：2所示。3.如权利要求1所述的一种适宜工厂化栽培的金黄色绣球菌菌株，所述菌株的子实体颜色为金黄色，子实体边缘呈波浪状，孢子印浅黄色。4.如权利要求1所述的一种适宜工厂化栽培的金黄色绣球菌菌株，所述菌株在F1母种培养基配方中菌丝体白色，有少量气生菌丝，偶见厚垣孢子；所述F1母种培养基配方为葡萄糖18-20g、玉米粉4-6g、糯米粉4-6g、鱼粉蛋白胨2-4g、磷酸二氢钾3g、硫酸镁1.5g、琼脂16g，再用水定容至1升。5.如权利要求1-4任一项所述金黄色绣球菌菌株的栽培方法，其特征在于，方法如下：菌种制备：采用固体菌种进行培养或者采用液体菌种在液体发酵罐进行培养，液体菌种培养周期为7-9天，固体菌种的培养周期为20-30天；固体菌种培养基配方为松木锯末70-72%，红薯粉8-12%，木薯粉8-12%，玉米粉6-10%，鱼粉蛋白胨1%，磷酸二氢钾1%，含水量64%；液体菌种发酵罐配方为：葡萄糖18-20g/L、玉米粉4-6g/L、糯米粉4-6g/L、鱼粉蛋白胨2-4g/L、磷酸二氢钾3g/L、硫酸镁1.5g/L；制备栽培出菇包，出菇包配方为：松木锯末62-68%，碳氮源复合物25-35%，土壤10-15%；所述碳氮源复合物为红薯粉：大米粉和/或木薯粉和/或玉米粉：蛋白胨质量比为7-10：4-6：1-2的混合物；出菇包含水量为63-65%，pH5-6；栽培及管理：将培养的菌种种植于出菇包，在菌丝培养阶段控制培养温度23-25℃，暗培养40-45天，培养室内二氧化碳浓度≤3000ppm，菌袋内二氧化碳浓度≤3%；在原基诱导阶段控制诱导温度16-19℃，诱导20-25天，光照500-2000Lx，每天照射≥11小时，诱导室内二氧化碳浓度≤2000ppm，袋内二氧化碳浓度≤2%，空气湿度70%-80%；在子实体发育阶段控制温度18-20℃，培养15-20天，光照500-2000Lx，每天照射≥11小时，室内二氧化碳浓度≤1000ppm，袋内侧壁二氧化碳浓度≤1%，空气湿度≥95%；整个生长发育周期为90-98天。
说明书desc
技术领域本发明属于食用菌育种领域，具体涉及一种适宜工厂化栽培的金黄色绣球菌菌株JSJ-Spar8-1C及其工厂化栽培方法。背景技术广叶绣球菌，隶属于担子菌门、伞菌纲、多孔菌目、绣球菌科、绣球菌属。绣球菌是一种名贵的食药用菌，含有非常丰富的β-葡聚糖，是目前所有食用菌中β-葡聚糖含量最高的食用菌，见参考文献 。长期以来研究者们在绣球菌的药理活性方面做了大量的研究工作，表明绣球菌中β-葡聚糖具有良好的抗肿瘤活性、免疫调节、抗炎症反应、促进伤口愈合作用等。20世纪80年代，我国开始对绣球菌进行人工驯化研究，于2010年首次在福建省实现了绣球菌的工业化栽培，成为了继日本和韩国之后的第三个实现绣球菌人工栽培的国家。在这之后的十余年间，许多学者在绣球菌的生物学特性、营养生理、化学成分、特别是药理活性等方面做了许多研究工作，而在绣球菌的遗传育种、新品种驯化栽培、液体深层发酵技术等方面的研究仍然较为薄弱，导致绣球菌工厂化栽培规模、产能、产业发展缓慢。据报道，目前绣球菌的日产能约为10万袋，日产鲜菇25吨产能。根据市场调研，目前实际能够稳定供应绣球菌鲜菇每天不到10吨，市场远远供不应求。而严重制约绣球菌产业发展的最根本原因之一是绣球菌栽培品种同质化严重，栽培品种单一，遗传育种方面的研究薄弱。因此，绣球菌新品种的选育工作迫在眉睫。发明内容本发明的目的在于提供一种适宜工厂化栽培的金黄色绣球菌菌株，该菌株在形态、遗传上区别于目前现有的所有栽培菌株，在产量、周期、抗性等方面均优于现有栽培菌株，为绣球菌产业的发展及遗传育种工作提供丰富的材料，推动绣球菌产业快速发展。本发明采取的技术方案如下：一种适宜工厂化栽培的金黄色绣球菌菌株，该菌株保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏单位地址为北京市朝阳区北辰西路1号院3号，保藏日期为2023年10月7日，保藏编号为CGMCC NO：2所示。所述金黄色绣球菌菌株的子实体颜色为金黄色，子实体边缘呈波浪状，孢子印浅黄色。所述金黄色绣球菌菌株在F1母种培养基配方中菌丝体白色，有少量气生菌丝，菌丝生长快，菌落形态一致性好，菌丝锁状联合明显，偶见厚垣孢子；所述F1母种培养基配方为葡萄糖18-20g、玉米粉4-6g、糯米粉4-6g、鱼粉蛋白胨2-4g、磷酸二氢钾3g、硫酸镁1.5g、琼脂16g，再用水定容至1升。所述金黄色绣球菌菌株的工厂化栽培方法如下：菌种制备：采用固体菌种进行培养或者采用液体菌种在液体发酵罐进行培养，液体菌种培养周期为7-9天，固体菌种培养周期为20-30天；固体菌种培养基配方为松木锯末70-72%，红薯粉8-12%，木薯粉8-12%，玉米粉6-10%，鱼粉蛋白胨1%，磷酸二氢钾1%，含水量64%；液体菌种发酵罐配方为：葡萄糖18-20g/L、玉米粉4-6g/L、糯米粉4-6g/L、鱼粉蛋白胨2-4g/L、磷酸二氢钾3g/L、硫酸镁1.5g/L；制备栽培出菇包，出菇包配方为：包含松木锯末62-68%，碳氮源复合物25-35%，土壤10-15%；所述碳氮源复合物为红薯粉：大米粉和/或木薯粉和/或玉米粉：蛋白胨质量比为7-10：4-6：1-2的混合物；出菇包含水量为63-65%，pH5-6；栽培及管理：将培养的菌种种植于出菇包，在菌丝培养阶段控制培养温度23-25℃，暗培养40-45天，培养室内二氧化碳浓度≤3000ppm，菌袋内二氧化碳浓度≤3%；在原基诱导阶段控制诱导温度16-19℃，诱导20-25天，光照500-2000Lx，每天照射≥11小时，诱导室内二氧化碳浓度≤2000ppm，袋内二氧化碳浓度≤2%，空气湿度70%-80%；在子实体发育阶段控制温度18-20℃，培养15-20天，光照500-2000Lx，每天照射≥11小时，室内二氧化碳浓度≤1000ppm，袋内侧壁二氧化碳浓度≤1%，空气湿度≥95%；整个生长发育周期为90-98天。本发明提供了一种适宜工厂化栽培的金黄色绣球菌菌株，该菌株子实体颜色为金黄色，区别于所有现有人工栽培的白色绣球菌菌株。本发明提供的适宜工厂化栽培的金黄色绣球菌菌株具有区别于其他菌株的特征DNA片段，在遗传上存在差异。本发明提供的适宜工厂化栽培的金黄色绣球菌菌株及配套的栽培基质及工厂化栽培管理方法，使其菌丝生长快、抗杂菌能力强，原基白色且健壮、出菇率高，可达85%及以上，生物学转化率高，可达67.05%及以上，生长发育周期短，仅为90-98天。本发明通过开展野生资源收集与系统驯化选育，选育出适宜工厂化栽培的金黄色绣球菌品种，为绣球菌产业的发展及遗传育种工作提供丰富的材料，可推动绣球菌产业快速发展。附图说明本发明所述适宜工厂化栽培的金黄色绣球菌菌株保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏日期为2023年10月7日，保藏编号为CGMCCNo.40890。图1为本发明菌株的子实体形态特征；图2为对照菌株“闽绣2号”的子实体形态特征；图3为本发明菌株特有的DNA分子片段胶图；图4为栽培基质配方1-6的菌丝生长情况；图5为栽培基质配方7-12的菌丝生长情况；图6为不同配方中菌丝生长速度。具体实施方式为了更清楚地描述本发明的技术方案，下面将结合实施例和附图对本发明的技术方案进行清楚、完整的描述。显然，这些实施例和附图仅是本发明的部分具体实施例和附图，而非全部的实施例和附图。本发明的发明内容及保护范围包括但不限于实施例。基于本发明中的实施例，本领域技术人员在无需做出创造性劳动前提下所获得的所有变例，都属于本发明要求保护的范围。除非特别说明，本发明百分比为质量百分比。实施例一种适宜工厂化栽培的金黄色绣球菌菌株，该菌株保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏编号为CGMCC
