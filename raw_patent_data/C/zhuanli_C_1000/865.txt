标题title
一种表达突变型尿卟啉原脱羧酶的重组菌
摘要abst
本发明公开了一种表达突变型尿卟啉原脱羧酶的重组菌，属于酶工程技术领域。本发明以SEQ ID NO.1所示的尿卟啉原脱羧酶为出发序列，将第131位甘氨酸突变为赖氨酸，获得了活性降低的尿卟啉原脱羧酶突变体，并将该突变体在粘着箭菌中表达，使粘着箭菌发酵VB12效价相比出发菌株提高20.2%。
权利要求书clms
1.一种突变型尿卟啉原脱羧酶，以SEQ ID NO.1所示的尿卟啉原脱羧酶为出发序列，将第131位甘氨酸突变为赖氨酸。2.编码权利要求1所述突变型尿卟啉原脱羧酶的基因。3.表达权利要求1所述突变型尿卟啉原脱羧酶的重组微生物细胞。4.根据权利要求3所述的表达权利要求1所述突变型尿卟啉原脱羧酶的重组微生物细胞，其特征在于，所述微生物细胞为粘着箭菌。5.一种重组粘着箭菌，其特征在于，以粘着箭菌CGMCC No.23306为宿主，表达权利要求1所述的突变型尿卟啉原脱羧酶。6.一种提高粘着箭菌维生素B12生产能力的方法，其特征在于，将粘着箭菌的尿卟啉原脱羧酶第131位甘氨酸突变为赖氨酸；所述尿卟啉原脱羧酶的氨基酸序列如SEQ ID NO.1所示。7.氨基酸序列如SEQ ID NO.5所示的突变型尿卟啉原脱羧酶在提高维生素B12生产能力方面的应用。8.一种生产维生素B12的方法，其特征在于，将权利要求5所述的粘着箭菌在发酵培养基中发酵一段时间。9.根据权利要求8所述的方法，其特征在于，所述发酵是将所述粘着箭菌在含赤砂糖、甜菜碱、磷酸氢二铵、硫酸铵、七水硫酸锌、5,6-DMBI、六水氯化钴、三氯化铁的培养基中，于30~35℃有氧发酵至少144小时。10.权利要求5所述的重组粘着箭菌在生产维生素B12中的应用。
说明书desc
技术领域一种表达突变型尿卟啉原脱羧酶的重组菌，属于酶工程技术领域。背景技术维生素 B12  是人体生长发育的必需营养素，是一类含有钴的咕啉类化合物总称，最早作为“抗恶性贫血因子”于1926年被发现。可以促进红细胞生长发育、叶酸的利用，或者以辅酶的形式参与催化体内的、氨基酸、脂肪酸代谢等多种重要的反应。维生素B12的分子结构极为复杂，其全化学合成需要多达70余个的反应步骤，且成本昂贵，故目前几乎都是通过微生物发酵的方式来生产维生素B12。目前工业化生产中有厌氧和好氧两种途径，最常用的菌株有:厌氧发酵的费氏丙酸杆菌和谢氏丙酸杆菌;好氧发酵的脱氮假单胞菌。申请人筛选的粘着箭菌亦可用于生产维生素 B12。HemE在VB12合成途径中，将促进尿卟啉原III转化成粪卟啉原III，进而转化成血红素。血红素是血红蛋白中的非蛋白质的铁卟啉组分，是血红蛋白与某些氧化还原酶的辅基，参与生物体中氧的传递和氧化还原过程。在VB12代谢通路中，血红素是代谢副产物，生产菌株积累血红素会相应的减少VB12的积累，如果能减少血红素的合成，就能有更多的原料用来合成VB12。但HemE是细菌生长代谢的必要基因，敲除后会影响菌株正常生长，降低HemE的酶活，增加尿卟啉原III的积累量，是增加VB12积累的有效途径。因此，筛选酶活降低的HemE突变体对于提高菌株B12生产具有重要的应用价值。发明内容本发明提供了一种突变型尿卟啉原脱羧酶HemE，以SEQ ID NO.1所示的尿卟啉原脱羧酶为出发序列，进行了第52位、105位、131位、133位中至少一处氨基酸的突变。在一种实施方式中，所述突变是将第52位天冬氨酸突变为天冬酰胺，获得突变体D52N，其氨基酸序列如SEQ ID NO.2所示。在一种实施方式中，所述突变是将第105位精氨酸突变为半胱氨酸，获得突变体R105C，其氨基酸序列如SEQ ID NO.3所示。在一种实施方式中，所述突变是将第131位甘氨酸突变为赖氨酸，获得突变体G131K，其氨基酸序列如SEQ ID NO.4所示。在一种实施方式中，所述突变是将第133位甘氨酸突变为赖氨酸，获得突变体G133K，其氨基酸序列如SEQ ID NO.6所示。本发明还提供了编码所述突变型尿卟啉原脱羧酶HemE的基因。在一种实施方式中，所述基因的核苷酸序列如SEQ ID NO.5所示。本发明还提供了表达所述突变型尿卟啉原脱羧酶HemE的重组微生物细胞。在一种实施方式中，所述微生物细胞为粘着箭菌。在一种实施方式中，所述粘着箭菌是保藏编号为CGMCC No.23306的粘着箭菌，该菌株已公开于公开号为CN114231450A的专利申请文件中。本发明还提供了所述突变型尿卟啉原脱羧酶HemE在提高粘着箭菌维生素B12生产能力中的应用。在一种实施方式中，所述应用是在粘着箭菌中表达所述突变型尿卟啉原脱羧酶。在一种实施方式中，所述突变型尿卟啉原脱羧酶具有SEQ ID NO.4所示的氨基酸序列。本发明还提供了所述突变型尿卟啉原脱羧酶HemE和/或所述重组微生物细胞在生产维生素B12中的应用。在一种实施方式中，所述发酵是在含赤砂糖、甜菜碱、磷酸氢二铵、硫酸铵、七水硫酸锌、5,6-DMBI、六水氯化钴、三氯化铁的培养基中，于30-34℃，230-270rpm条件下发酵至少144小时。在一种实施方式中，所述发酵培养基含有：赤砂糖 50 2g/L。本发明还提供粘着箭菌在生产维生素B12中的应用。有益效果：1、本发明通过将尿卟啉原脱羧酶第131甘氨酸位甘氨酸突变为赖氨酸，获得了尿卟啉原脱羧酶活性下降的突变体，能显著降低尿卟啉原III转向成粪卟啉原III的通量，提高了菌株对VB12的生产能力；2、本发明通过将构建的尿卟啉原脱羧酶突变体替换出发菌株原有的尿卟啉原脱羧酶，构建的重组粘着箭菌在摇瓶水平发酵VB12的效价可达99.6 mg/L，相比出发菌株提高20.2%。附图说明图1为血红素合成途径及HemE催化路径。图2为粘着箭菌HemE与人源尿卟啉原脱羧酶氨基酸比对结果。具体实施方式培养基：种子培养基：甜菜糖蜜30g/L；5,6-DMBI0.03g/L；磷酸氢二铵2g/L；硫酸铵1g/L；七水硫酸锌0.015g/L；六水氯化钴0.004g/L；消泡剂0.8g/L。发酵培养基：赤砂糖 50 处测定生物量用于测量细胞浓度，即为OD。VB12浓度检测：用液相色谱外标法检测发酵液中的VB12浓度。取发酵液5mL于25mL具塞比色管中，加入0.5 mL醋酸和1ml的8%亚硝酸钠，用长针头刮一点消泡剂于比色管下部，将产生的气泡晃入液面以下。轻轻混匀后沸水浴30 min，待样品冷却至室温后，用纯化水定容至比色管25 mL刻度线处，摇匀。用0.45μm的微孔滤膜过滤至棕色液相小瓶中，液相检测。检测条件为：色谱柱C18；流动相：水：乙腈=90:10, 流速：1mL/min；柱温：40℃；进样量：20μL；检测波长：550nm。效价= VB12浓度的检测值×50。血红素浓度的检测：带菌体的发酵液1ml在4℃，12000r/min离心5min，得到的上清直接进样即可检测胞外血红素；得到的沉淀加入1ml 1mol/L NaOH，涡旋混匀，然后超声10min后，4℃，12000r/min离心5min，取上清液进行测定胞内血红素。检测条件为：色谱柱；流动相：A：水，B：甲醇；流速：1mL/min；柱温：35℃；检测波长：400nm；进样量：10μL。实施例1 HemE保守位点确定及基因编辑片段的构建将HemE的核苷酸序列与人源尿卟啉原脱羧酶进行序列比对，先找到碱基一致的点，然后再参照人源尿卟啉原脱羧酶氨基酸突变的结果，把粘着箭菌来源HemE对应的关键的活性位点氨基酸突变掉，降低HemE的活性。结合单碱基编辑技术PAM位置确定，最终共确定了4个突变位点，分别是D52，R105，G131，G133。将分别构造单碱基编辑质粒PCZL-HemE-D52，PCZL-HemE-R105、PCZL-HemE-G131、PCZL-HemE-G133来进行着4个位点的氨基酸突变。以质粒PCZL-CBE-1、PCZL-CBE-2为模板，构建相对应位点的单碱基编辑质粒。表1 400ng，混匀后在冰上放置20-30分钟。将混合物转移到提前预冷的电击杯中，延壁加入，轻轻敲击使混合物落到电转杯的底部，防止有气泡产生。将电转杯外壁的水分擦拭干净，将电击杯放到仪器中，电击一次，电压常数2.5 kV；时间常数4ms。电击结束后迅速取出电转杯，吸取1ml的复苏液到电转杯中，混匀后将混合液完整的转移到离心管中，混匀。离心管在30℃，200rpm的条件下复苏3小时。最后离心浓缩，使用TYC平板涂布，30℃培养直至得到单菌落转化子，一般约需3-5天。单菌落转化子测序验证：使用基因组上的引物对PCYW-HemEG-F，PCYW-HemEG-R对转化子的基因组进行扩增后，经由测序得到正确编辑的菌株，分别命名为PCJZ-HemE-D52N，PCJZ-HemE-R105C， PCJZ-HemE-G131K，PCJZ-HemE-G133K。生产菌株质粒脱除：取测序正确的已编辑菌株，在含有氯霉素的TYC培养基中培养，30℃，200rpm条件下培养48小时。沾取菌液在TYC氯霉素平板涂布，30℃培养3-5天直至得到单菌落转化子。取单菌落转化子一半涂布在含有氯霉素抗性与卡那抗性的TYC双抗平板上，另一半涂布在只含有氯霉素抗性的TYC平板上，30℃培养3天后，如果同一克隆只在氯霉素抗性平板上生长，而未在双抗平板上生长，则说明质粒PCZL-CBE已被脱除。实施例3
