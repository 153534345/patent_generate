标题title
PP2C01基因在调控水稻耐盐性方面的应用
摘要abst
本发明涉及OsPP2C01基因在调控水稻耐盐性方面的应用。我们的研究发现了OsPP2C01基因调控水稻耐盐的功能，通过对OsPP2C01基因进行基因编辑可以显著提高水稻的耐盐性。本发明的提出为培育耐盐水稻新品系提供思路。

说明书desc
技术领域本发明涉及植物基因工程技术领域，具体涉及OsPP2C01基因在调控水稻耐盐性方面的应用。背景技术水稻是世界上最重要的粮食作物之一，种植面积占粮食作物总面积约1/3，全世界近一半人口以水稻为主粮。然而，在水稻生产过程中，非生物胁迫严重影响水稻的生长发育，进而影响水稻产量。盐胁迫是水稻种植过程中主要的非生物胁迫之一，高盐不仅导致离子毒害和高渗胁迫，同时伴随发生次级胁迫。高盐条件会抑制水稻的正常生长发育，严重时甚至引起水稻死亡。因此，挖掘出更多参与调控水稻耐盐性的关键基因，选育出抵抗高盐胁迫的水稻育种材料，对于水稻育种、种植产业的发展具有重要意义。目前研究人员已经研究发现了一些与水稻耐盐性相关的基因，但目前OsPP2C01基因在调控水稻耐盐性方面的应用尚未见报道。发明内容本发明旨在提供OsPP2C01基因在调控水稻耐盐性方面的应用。本发明技术方案如下：OsPP2C01基因在调控水稻耐盐性方面的应用，所述OsPP2C01基因的基因组序列如SEQ ID NO.1所示，CDS序列如SEQ ID NO.2所示，蛋白序列如SEQ ID NO.3所示。优选的，在实际应用中，可通过基因编辑技术如CRISPR/Cas9对OsPP2C01基因进行敲除，获得高耐盐性水稻。进一步优选的，所述应用为：以SEQ ID NO.4所示的序列为靶序列1，以SEQ IDNO.5所示的序列为靶序列2，设计扩增靶序列的引物序列；以pCBC-MT1T2 质粒为模板，进行PCR扩增，得到PCR产物；PCR产物纯化，再使用酶切-连接方式，重组得到CRISPR/Cas9-OsPP2C01表达载体；通过农杆菌介导转化法将测序正确的CRISPR/Cas9-OsPP2C01表达载体转化到水稻中，得到转化苗。优选的，所述引物序列如SEQ ID NO.6和SEQ ID NO.7所示。优选的，所述水稻的品种包括中花11。与现有技术相比，本发明的有益成果在于：我们的研究发现了OsPP2C01基因调控水稻耐盐的功能，通过对OsPP2C01基因进行基因编辑可以显著提高水稻的耐盐性，获得高耐盐性水稻。本发明为快速创制耐盐水稻新品系提供了一种简单有效的技术手段。附图说明图1：OsPP2C01基因结构和敲除靶点信息。图2：OsPP2C01耐盐实验结果。具体实施方式为使本领域技术人员更好的理解本发明技术内容，下面结合具体实施例和附图对本发明做进一步的说明。实施例1、CRISPR/Cas9-OsPP2C01表达载体的构建及OsPP2C01突变体的获得一、构建CRISPR/Cas9-OsPP2C01表达载体从OsPP2C01基因的CDS序列区选取两个20bp的靶点进行CRISPR/Cas9的双靶点敲除。通过PCR的方法将靶点序列连接到CRISPR/Cas9载体上，再利用农杆菌介导转化的方法将CRISPR/Cas9-OsPP2C01的载体转化到水稻品种中花11中，获得OsPP2C01缺失的突变体水稻。1、构建CRISPR/Cas9-OsPP2C01表达载体步骤如下：用于编辑OsPP2C01基因的CRISPR载体表达2个sgRNA：sgRNA1和sgRNA2 ，sgRNA1识别区的编码序列为序列2第148-166位，sgRNA2识别区的编码序列为序列2第667-689位。设计的两个靶序列如下所示：靶点1：CTAATACCACACCCAAGAA靶点2：TCTGTCATTTCCGAGTCTCCAGG根据靶点设计出引物序列，引物序列如下：BsF:AATAATGGTCTCAGGCGCTAATACCACACCCAAGAAGTTTTAGAGCTAGAAATAGCBsR:ATTATTGGTCTCTAAACCCTGGAGACTCGGAAATGACAGACGCTTCTTGGTGCC以稀释100倍的pCBC-MT1T2 质粒为模板，BsF和BsR为引物，进行PCR扩增，得到PCR产物。得到PCR产物后纯化，再使用以下酶切-连接方式，得到CRISPR/Cas9-OsPP2C01表达载体。载体构建方法参照Xing HL, Dong L, Wang ZP, Zhang HY, Han CY, Liu B, WangXC, Chen QJ.
