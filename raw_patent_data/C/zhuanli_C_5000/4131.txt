标题title
一种高产巴豆酸乙酯酿酒酵母菌株及其构建方法与用途
摘要abst
本发明属于生物工程技术领域，涉及工业微生物的育种，尤其是构建一种高产巴豆酸乙酯酿酒酵母菌株的方法及其应用。通过在出发菌株中过表达3‑羟基丁酰辅酶A脱氢酶Hbd、3‑羟基丁酰辅酶A脱水酶Crt以及醇酰基转移酶AAT，以获得1株具有产巴豆酸乙酯能力的酵母菌株Ck‑HCVL，与出发菌不产巴豆酸乙酯相比，改造菌株巴豆酸乙酯产量达到了58.6±6.19mg/L。在过表达Erg10和二拷贝AAT基因后获得菌株Ck‑HC‑DVL‑E，巴豆酸乙酯的产量达到了122.99±6.55mg/L，与Ck‑HCVL菌株相比产量提高了109.9％，取得了预料不到的技术效果，为利用微生物产巴豆酸乙酯提供解决思路。
权利要求书clms
1.一种高产巴豆酸乙酯的酿酒酵母基因工程菌株，其特征在于：所述菌株以酿酒酵母为出发菌株，通过过表达乙酰辅酶A酰基转移酶基因Erg10，过表达3-羟基丁酰辅酶A脱氢酶基因Hbd、3-羟基丁酰辅酶A脱水酶基因Crt，同时过表达醇酰基转移酶基因AAT构建而成；其中，通过在酿酒酵母LPP1位点过表达所述乙酰辅酶A酰基转移酶基因Erg10，同时所述3-羟基丁酰辅酶A脱氢酶编码基因Hbd和3-羟基丁酰辅酶A脱水酶编码基因Crt整合到酿酒酵母YCR011C和YBR128C位点；同时醇酰基转移酶基因AAT通过整合到酿酒酵母YHR142W位点实现过表达。2.如权利要求1所述一种高产巴豆酸乙酯的酿酒酵母基因工程菌株，其特征在于：所述基因AAT为双拷贝表达；其中，AAT的双拷贝是通过整合到酿酒酵母416d位点。3.如权利要求1所述一种高产巴豆酸乙酯的酿酒酵母基因工程菌株，其特征在于：所述乙酰辅酶A酰基转移酶基因Erg10，其核苷酸序列如核苷酸序列表中SEQ ID NO：4所示；所述3-羟基丁酰辅酶A脱氢酶基因Hbd，其核苷酸序列如核苷酸序列表中SEQ ID NO：1所示；所述3-羟基丁酰辅酶A脱水酶基因Crt，其核苷酸序列如核苷酸序列表中SEQ ID NO：2所示；所述醇酰基转移酶基因AAT，其核苷酸序列如核苷酸序列表SEQ ID NO：3所示。4.如权利要求1所述一种高产巴豆酸乙酯的酿酒酵母基因工程菌株，其特征在于：所述出发菌株为酿酒酵母CICC32315。5.权利要求1-4任一所述高产巴豆酸乙酯的酿酒酵母基因工程菌株在发酵、香精香料、药物合成领域中的用途。6.如权利要求5所述高产丁酸乙酯的酿酒酵母基因工程菌株的发酵方法，其特征在于：将酿酒酵母基因工程菌经过两级活化后，将种子液按接种量8-12％接种于发酵培养基，30℃静置发酵84-96h。所述发酵培养基组成为：玉米粉300-320g/L，耐高温α淀粉酶为×104U/L，糖化酶90-100U/L，酸性蛋白酶10-20U/L，营养盐溶液5.5-5.6mL/L，余量为水；所述营养盐溶液组成为：MgSO4140-160g/L，KH2PO470-80g/L，尿素80-85g/L，余量为水。
说明书desc
技术领域：本发明属于生物工程技术领域，涉及工业微生物的育种，特别涉及一种高产巴豆酸乙酯酿酒酵母菌株及其构建方法与用途。背景技术：巴豆酸乙酯呈强烈酸焦香和水果香气，有朗姆酒和醚香。天然品存在于苹果、木瓜、草莓、芒果、朗姆酒、葡萄酒和可可等中，可应用于食用香精配方，主要用于配制果酒类香精。在白酒、葡萄酒及其他饮料酒中，酯类物质是各种微量成分中最重要的一种，酯类物质能提高酒的香气浓郁度和香气品质。这些酯类中，乙酸酯类是含量最多的一种，短中链脂肪酸乙酯是饮料酒中的关键风味成分，是酒类芳香的主要承担体。巴豆酸乙酯在工业上用作化妆品、洗发水、洗涤剂和清洁剂中芳香基的前体，还可用作有机合成中间体、溶剂，在药物合成领域内有着广泛的用途，工业上巴豆酸乙酯主要的生产方式是化学法由巴豆酸与乙醇在浓硫酸存在下酯化而成，目前暂时没有利用微生物发酵产巴豆酸乙酯的研究。在酿酒酵母中，短中链酰基辅酶A的合成主要有三种途径，胞内脂肪酸从头合成途径、外源脂肪酸吸收活化途径以及脂肪酸降解途径，这些生成途径的调控十分复杂，因而产短中链脂肪酸乙酯能力低。酿酒酵母自身几乎不产巴豆酸乙酯，如果在酿酒酵母中构建一条巴豆酰辅酶A合成途径以及引入一种能将酰基辅酶A与乙醇合成相应乙基酯的高效醇酰基转移酶，就能够构建出一株产巴豆酸乙酯的酿酒酵母菌株。发明内容本发明的第一个目的是解决酿酒酵母在酒类生产中不合成巴豆酸乙酯的问题，提供一种产巴豆酸乙酯酿酒酵母菌株的构建方法。具体是在酿酒酵母中构建一条巴豆酰辅酶A生成途径使酿酒酵母产生巴豆酰辅酶A；进而在此基础上向酿酒酵母中引入一种高效的醇酰基转移酶，从而构建完整的巴豆酸乙酯途径，实现巴豆酸乙酯的产生。进一步地，过表达内源乙酰辅酶A酰基转移酶基因增加前提物质，并对巴豆酸乙酯生产途径中的醇酰基转移酶基因进行双拷贝表达，使得巴豆酸乙酯产量获得大幅提升。为了解决上述问题，本发明提供一种产巴豆酸乙酯的酿酒酵母基因工程菌株，所述菌株以酿酒酵母CICC32315为出发菌株，通过异源过表达3-羟基丁酰辅酶A脱氢酶基因Hbd、3-羟基丁酰辅酶A脱水酶基因Crt，异源过表达醇酰基转移酶基因AAT，同时过表达内源乙酰辅酶A酰基转移酶基因Erg10构建而成。优选地，所述基因VLAAT为单拷贝表达或双拷贝表达。优选地，所述3-羟基丁酰辅酶A脱氢酶基因Hbd来源于Clostridium kluyveri DSM 555。更优选地，所述3-羟基丁酰辅酶A脱氢酶基因Hbd，其Gene ID为：5394457，酿酒酵母密码子优化后核苷酸序列如核苷酸序列表中SEQ ID NO：1所示。优选地，所述3-羟基丁酰辅酶A脱水酶基因Crt来源于Clostridium kluyveri DSM 555。更优选地，所述3-羟基丁酰辅酶A脱水酶基因Crt，其Gene ID为：5391750，酿酒酵母密码子优化后核苷酸序列如核苷酸序列表中SEQ ID NO：2所示。优选地，所述醇乙酰基转移酶基因AAT来源于葡萄。更优选地，所述醇乙酰基转移酶基因AAT，其GenBank为：KX963771.1，酿酒酵母密码子优化后核苷酸序列如核苷酸序列表中SEQ ID NO：3所示。优选地，所述乙酰辅酶A酰基转移酶Erg10基因来源于酿酒酵母。更优选地，所述乙酰辅酶A酰基转移酶基因Erg10，其Gene ID为：856079，核苷酸序列如核苷酸序列表中SEQ ID NO：4所示。优选地，所述出发酵母菌株为酿酒酵母CICC32315；所述异源过表达3-羟基丁酰辅酶A脱氢酶基因Hbd，目的是由乙酰乙酰辅酶A合成3-羟基丁酰辅酶A。所述异源过表达3-羟基丁酰辅酶A脱水酶基因Crt，目的是由3-羟基丁酰辅酶A合成巴豆酰辅酶A。所述异源过表达醇酰基转移酶基因AAT，目的是为了引入醇酰基转移酶，从而构建完整的巴豆酸乙酯途径，实现巴豆酸乙酯的产生。所述过表达乙酰辅酶A酰基转移酶基因Erg10，目的是由乙酰辅酶A合成乙酰乙酰辅酶A。所述对途径中醇酰基转移酶基因AAT为双拷贝表达，是由于在巴豆酰辅酶A合成途径中醇酰基转移酶催化巴豆酸乙酯合成的关键最后一步，也有可能会限制巴豆酸乙酯的产生，故而对醇酰基转移酶基因进行双拷贝，从而显著提高酿酒酵母的巴豆酸乙酯的产量。所述对内源基因Erg10基因过表达，是由于在巴豆酰辅酶A合成途径中Erg10催化前体物质乙酰乙酰辅酶A合成，也可能会限制巴豆酸乙酯的产生，故而对Erg10基因进行过表达，从而显著提高酿酒酵母的巴豆酸乙酯的产量。本发明的第二个目的是提供利用Crispr基因编辑技术构建上述产巴豆酸乙酯的酿酒酵母基因工程菌株的方法，首先3-羟基丁酰辅酶A脱氢酶和3-羟基丁酰辅酶A脱水酶在酿酒酵母中YCR011C、YBR128C位点进行异源表达，构建具有巴豆酰辅酶A生成途径的酵母菌株Ck-HC；其次，将葡萄醇酰基转移酶VLAAT在上述菌株YHR142W位点进行异源整合表达，得到产巴豆酸乙酯的酵母菌株Ck-HCVL；第三，在Ck-HCVL的LPP1位点过表达乙酰辅酶A酰基转移酶基因Erg10，目的是由乙酰辅酶A合成乙酰乙酰辅酶A，得到菌株Ck-HCVL-E；最后，在Ck-HCVL-E菌株416d位点进行双拷贝VLAAT的表达，获得高产巴豆酸乙酯的菌株Ck-HC-DVL-E。优选地，所述基因VLAAT为双拷贝表达。更优选地，所述葡萄醇酰基转移酶基因VLAAT的双拷贝异源表达是通过整合到416d位点实现的。所述基因YCR011C、YBR128C、YHR142W、LPP1和416d，均来源于酿酒酵母。优选地，高产巴豆酸乙酯的酿酒酵母基因工程菌株的构建方法，如下步骤：以酿酒酵母菌株的单倍体作为出发菌株，以基因YCR011C、YBR128C为整合位点，将基因YCR011C、YBR128C的上游同源臂011U500/128U500、TDH3p-Hbd-TDH1t/CCW12p-Crt-SSA1t片段、基因YCR011C、YBR128C的下游同源臂011D500/128D500，分别按顺序经融合PCR连接后和Cas9-011C-128C质粒经醋酸锂转化插入到整合位点上获得重组菌株Ck-HC；同时以基因YHR142W为整合位点，将基因YHR142W的上游同源臂142U500、PGK1p-VLAAT-ENO2t片段以及基因YHR142W的下游同源臂142D500按顺序融合连接与Cas9-142W质粒经醋酸锂转化插入到整合位点上获得重组菌株Ck-HCVL；优选地，所述步骤中，酿酒酵母菌株的单倍体为α型单倍体。更优选的，对所述构建方法获得的重组菌株，进行后续实验：以Ck-HCVL作为出发菌株，以LPP1为整合位点，将基因LPP1的上游同源臂LPP1U500、TEF1p-Erg10-ADH1t片段、基因LPP1的下游同源臂LPP1D500按顺序融合PCR连接与Cas9-LPP1质粒经醋酸锂转化插入到整合位点上获得重组菌株Ck-HCVL-E；以Ck-HCVL-E作为出发菌株，以416d为整合位点，将基因416d的上游同源臂416U500、PGK1p-VLAAT-ENO2t片段、基因416d的下游同源臂416D500按顺序经融合PCR连接与Cas9-416d质粒经醋酸锂转化插入到整合位点上获得重组菌株Ck-HC-DVL-E；本发明的第三个目的是提供所述高产巴豆酸乙酯的酿酒酵母基因工程菌株的发酵方法。优选地，所述酿酒酵母基因工程菌株的发酵步骤如下：酿酒酵母基因工程菌经过两级活化后，将种子液按接种量8-12％接种于发酵培养基，28-30℃静置发酵80-86h。所述在发酵后期，每隔12h称重1次，当两次失重小于0.5g，即确定发酵结束。优选的，所述玉米浓醪发酵培养基组成为：玉米粉300-320g/L，水180mL，耐高温α-淀粉酶×104U/L，糖化酶90-100U/L，酸性蛋白酶10-20U/L；营养盐5.5-5.6ml/L，余量为水。所述营养盐组成为：MgSO4140-160g/L、KH2PO470-80g/L、尿素80-85g/L，余量为水。优选地，所述酿酒酵母的两级活化条件为：首先将酿酒酵母基因工程菌接入一级种子培养基中，28-30℃静置培养24h，得到一级种子液，将所述一级种子液按8-10％接种量接种于二级种子培养基中，28-30℃静置培养至对数期的后期，即15-18h，得到二级种子液。更优选地，所述一级种子培养基组成为：玉米粉80-85g/L，耐高温α淀粉酶添加量约为×104U/L，糖化酶约为30-35U/L，余量为水，糖度为8°BX。更优选地，所述二级种子培养基组成为：玉米粉约120-130g/L，耐高温α淀粉酶添加量约为×104U/L，糖化酶约为45-55U/L，余量为水，糖度为12°BX。有益效果：1、本发明所述技术内容为在酿酒酵母中产巴豆酸乙酯提供了新途径，通过引入外源巴豆酰辅酶A合成途径以及引入一种能将酰基辅酶A与乙醇合成相应乙基酯的高效醇酰基转移酶，构建出一株高产巴豆酸乙酯的酿酒酵母菌株。相较于不能产巴豆酸乙酯的野生型酿酒酵母菌，该菌株的巴豆酸乙酯产量达到了122.99±6.55mg/L，为酿酒酵母产巴豆酸乙酯奠定了理论基础，具有广阔的应用前景。2、本发明提供的产巴豆酸乙酯的酿酒酵母菌株是在保持优良发酵性能的前提下实现的，为在酿造酒中产巴豆酸乙酯提供解决思路，微生物发酵产酯减少了副产物和能源消耗，也使巴豆酸乙酯不止局限于化学法生产。附图说明图1为在酿酒酵母构建的巴豆酸乙酯合成途径代谢图；图2为重组质粒Yep352-TT-Hbd、Yep352-CS-Crt、Yep352-PE-VLAAT、Yep352-TA-Erg10的构建流程示意图；图3为构建重组质粒Yep352-TT-Hbd、Yep352-CS-Crt、Yep352-PE-VLAAT、Yep352-TA-Erg10的验证电泳图；图中泳道1为Yep352-TT质粒；泳道2为Yep352-TT-CkHbd；泳道3为CkHbd基因片段；泳道4为10000bp
