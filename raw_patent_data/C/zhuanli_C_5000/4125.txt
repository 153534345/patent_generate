标题title
死亡素和斑点叉尾鮰β-防御素基因重组毕赤酵母的构建方法及应用
摘要abst
本发明公开了死亡素和斑点叉尾鮰β‑防御素基因重组毕赤酵母的构建及抗菌活性研究。试验将死亡素和防御素经酵母偏好密码子优化后构建共表达TD基因至载体pPICZαA上，PCR、双酶切和测序筛选后经Sac I线性化，电转化至酵母表达菌株GS115中，Zeocin抗性、PCR筛选获得重组高抗阳性转化工程菌株GS115‑pPICZαA‑TD，用0.5%甲醇诱导目的蛋白的表达，96 h后收集培养基上清冻干浓缩并纯化，进行体外抑菌试验、溶血性试验及细胞毒性试验。结果表明，本试验成功获得重组毕赤酵母且表达出广谱高效抑菌、安全性高的重组抗菌蛋白，为临床预防和治疗细菌性疾病提供治疗思路和实验依据。

1.一种共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl重组毕赤酵母表达菌株，命名为：GS115-pPICZαA-TD。2.权利要求1所述共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl的重组毕赤酵母表达菌株的构建方法，其特征在于包括以下步骤：1）根据死亡素thanatin蛋白序列，经毕赤酵母偏好性密码子进行优化得到thanatin基因序列SEQ ID NO.7；根据斑点叉尾鮰β-防御素基因，经毕赤酵母偏好性密码子进行优化得到defbl基因序列SEQ ID NO.8；经T2A连接的thanatin和defbl基因序列得到共表达TD基因序列SEQ ID NO.9；2）所述的thanatin基因、defbl基因和TD基因的PCR扩增包括：以thanatin-F SEQ IDNO.1和thanatin-R SEQ ID NO.2为引物，扩增thanatin基因；以defbl-F SEQ ID NO.3和defbl-R SEQ ID NO.4为引物扩增defbl基因；以TD-F SEQ ID NO.5和TD-R SEQ ID NO.6为引物扩增TD基因；3）将获得的目的基因克隆至pPICZαA载体，经Sac I线性化后电转化至毕赤酵母GS115感受态细胞内，构建了包含上述基因的重组毕赤酵母表达菌株GS115-pPICZαA-thanatin、GS115-pPICZαA-defbl、GS115-pPICZαA-TD。3.权利要求2所述的共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl的重组毕赤酵母表达菌株的构建方法制备重组毕赤酵母，该重组毕赤酵母能够分泌同时抗革兰氏阴性和阳性菌的抗菌蛋白。4.权利要求2所述的共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl的重组毕赤酵母所表达的抗菌蛋白在预防及治疗细菌性疾病方面的应用：本试验表达的外源抗菌蛋白对革兰氏阴性细菌包括大肠杆菌K88、奇变形杆菌、肺炎克雷伯菌ATCC700603、多杀性巴氏杆菌及革兰氏阳性细菌包括表皮葡萄球菌ATCC 12228、金黄色葡萄球菌CMCC26001、停乳链球菌、无乳链球菌均表现出良好的抑制作用，抑制率均在80%以上，广谱抗菌，抑菌活性显著优于单独thanatin、defbl蛋白；且不具备溶血性、细胞毒性，表现出良好的安全性。
说明书desc
技术领域本发明涉及构建表达菌株的方法，具体说，是一种共表达死亡素thanatin和斑点叉尾鮰β-防御素基因defbl重组毕赤酵母的构建方法及抗菌蛋白活性研究。背景技术抗菌肽广泛存在于各种生物中，是生物天然免疫系统中重要的免疫活性分子，具有分子量小、稳定性好、广谱抗菌、抗肿瘤、抗病毒的特点，同时也可抑制某些真菌和寄生虫，表现出对耐抗生素菌株很强的抑制性，耐酸、耐碱、耐高温、易降解、不易使细菌等产生耐药性，在医药、食品、畜牧行业上有着良好的应用前景，“抗菌肽替抗”已成为一种可行方案。死亡素thanatin是从刺肩蝽血液淋巴中分离出来的有21个氨基酸的小分子抗菌肽，1996年就被报道其具有广谱抗菌作用，生理浓度下就具有广泛抑菌作用，同时抑制某些原虫和真菌，但对金黄色葡萄球菌抗性较低；斑点叉尾鮰的β-防御素是其免疫系统中的一个重要因子，已有研究表明其水溶液能抑制多种细菌生长，反向平行的二硫键结构使其结构稳定，不易被蛋白酶等水解，可发挥长效抑菌作用。目前，随着相关分子生物学技术的发展，利用细菌、真菌、细胞等外源表达出高抗菌活性、不易产生耐药性且安全性高的抗菌肽类药物来代替常规抗生素的使用，已成研究热点，本研究将死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl用T2A连接，构建重组毕赤酵母，以期通过单次诱导即可同时表达两种抗菌肽，既实现了协同抗菌作用，又可共同进行纯化操作，简便操作环节，为预防和治疗细菌性疾病、研发新型广谱抑菌药物提供理论和试验依据。发明内容为实现上述目的，本发明公开了如下的技术内容：一种共表达死亡素thanatin基因和和斑点叉尾鮰β-防御素基因defbl的重组毕赤酵母菌株，命名为：GS115-pPICZαA-TD。本发明进一步公开了所述共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl重组毕赤酵母的构建方法，其特征在于，包括以下步骤：1）根据死亡素thanatin蛋白序列，经毕赤酵母偏好性密码子进行优化得到thanatin基因序列SEQ ID NO.7；根据斑点叉尾鮰β-防御素基因，经毕赤酵母偏好性密码子进行优化得到defbl基因序列SEQ ID NO.8；经T2A连接的thanatin和defbl基因序列得到共表达TD基因序列SEQ ID NO.9；2）所述的thanatin基因、defbl基因和TD基因的PCR扩增包括：以thanatin-F SEQID NO.1和thanatin-R SEQ ID NO.2为引物，扩增thanatin基因；以defbl-F SEQ ID NO.3和defbl-R SEQ ID NO.4为引物扩增defbl基因；以TD-F SEQ ID NO.5和TD-R SEQ ID NO.6为引物扩增TD基因；3）将获得的目的基因克隆至pPICZαA载体，经Sac I线性化后电转化至毕赤酵母GS115感受态细胞内，构建了包含上述基因的重组毕赤酵母表达菌株GS115-pPICZαA-thanatin、GS115-pPICZαA-defbl、GS115-pPICZαA-TD。本发明公开的一种共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl重组毕赤酵母的构建方法制备重组毕赤酵母，该重组毕赤酵母能够分泌同时抗革兰氏阴性和阳性菌的抗菌蛋白。本发明同时也公开了一种共表达死亡素thanatin基因和斑点叉尾鮰β-防御素基因defbl重组毕赤酵母的构建方法制备的重组毕赤酵母所表达的外源抗菌蛋白在预防及治疗细菌性疾病方面的应用。试验结果显示：本试验共表达的外源抗菌蛋白对革兰氏阴性细菌包括大肠杆菌K88、奇变形杆菌、肺炎克雷伯菌ATCC700603、多杀性巴氏杆菌及革兰氏阳性细菌包括表皮葡萄球菌ATCC 12228、金黄色葡萄球菌CMCC26001、停乳链球菌、无乳链球菌均表现出良好的抑制作用，广谱抗菌，且相较于单独thanatin、defbl蛋白，TD组蛋白在抑制肺炎克雷伯菌ATCC700603、金黄色葡萄球菌ATCC 12228、无乳链球菌上具有明显优势，表现在抑菌率及抑菌圈直径上，同时，表达蛋白不具备溶血性、细胞毒性，安全性良好。死亡素thanatin，源自刺肩蝽，基因全长63 bp，编码21个氨基酸，序列为GSKKPVPIIYCNRRTGKCQRM，APD分析显示，该肽带有6个净正电荷，疏水性为28%，具有广谱的抗革兰氏阴性细菌、革兰氏阳性细菌以及抗真菌、抗病毒活性，经预测显示该肽结构稳定，生理浓度下就能广泛抑制革兰氏阴性菌、革兰氏阳性菌、某些真菌和原虫，但其对金黄色葡萄球菌抗性较低。斑点叉尾鮰的β-防御素基因，基因全长204 bp，编码68个氨基酸，蛋白序列为MKPHCTLLLVLLVILALHSKGNEAVSFPWSCAALS GVCRQGACLPSELYFGPLGCGKGSLCCVSYFL，反相平行的片层结构使其不易被蛋白酶等水解，可以长效发挥广谱抗菌作用，2020年曾在王莎莎等人的试验中得到表达，且具有很好的抑菌活性。天然抗菌肽资源有限，获取困难，目前无法实现大量获得，尚未得到广泛应用，现期望通过基因工程技术探索一条低成本大量获得抑菌效果好的抗菌活性蛋白道路。本试验将thanatin、defbl和共表达TD基因重组到真核表达载体pPICZαA 结论3.1 以pMD-thanatin、pMD-defbl和pMD-TD为模板，通过质粒提取、双酶切、产物的胶回收、与酵母真核表达载体pPICZαA Vector用T4 DNA连接酶连接，转化E.coliDH5α感受态细胞，获得重组质粒，经过菌液PCR、双酶切鉴定和测序分析，成功构建含目的基因的重组表达载体并分别命名为pPICZαA-thanatin、pPICZαA-defbl、pPICZαA-TD。3.2 将测序正确的重组质粒pPICZαA-thanatin、pPICZαA-defbl、pPICZαA-TD进行Sac I酶切线性化后电转化至毕赤酵母GS115感受态细胞，经Zeocin抗性筛选及PCR验证，成功获得重组毕赤酵母工程菌株并分别命名为GS115-pPICZαA-thanatin、GS115-pPICZαA-defbl、GS115-pPICZαA-TD。3.3 利用0.5%甲醇诱导目的蛋白的表达，收集表达产物进行Western blotting检测，结果显示，GS115-pPICZαA-TD表达出大小为2.3 kd、7.8 kd的两蛋白，GS115-pPICZαA-thanatin表达出大小为2.3 kd的蛋白，GS115-pPICZαA-defbl表达出大小为7.8kd的蛋白。3.4 通过计算各重组蛋白抑菌率及抑菌圈直径对重组蛋白进行抑菌效果分析，结果显示，本试验所获重组蛋白对试验所选用的八种指示菌株均有抑制效果，TD蛋白显示广谱高抗菌特性，且相较thanatin、defbl蛋白，共表达TD蛋白具有显著优势，抑菌效果更强，MIC值为25 μg/mL。3.5 检测重组蛋白溶血性及细胞毒性，结果表明，本试验所获thanatin、defbl及TD重组蛋白在较高浓度时均不存在红细胞溶血作用，不产生细胞毒性作用，表明本试验所获重组蛋白安全性良好。本发明的创新之处在于，首次将死亡素thanatin和斑点叉尾鮰β-防御素基因defbl及其共表达TD基因在毕赤酵母GS115菌株中进行重组表达，将thanatin生理浓度下的广谱抑菌但对金黄色葡萄球菌抗性较低及defbl广谱长效抗菌联系起来，结合毕赤酵母表达系统高效表达外源基因的优势，达到通过一次诱导即可同时获得两种抗菌肽蛋白协同抗菌，且一同纯化简便操作步骤的目的，终验证本试验构建的重组毕赤酵母表达的重组抗菌肽蛋白对受试菌株均表现出良好的抑菌活性，共表达TD组具有明显抗菌优势且重组抗菌肽蛋白安全性良好，本试验为后期对该抗菌蛋白的研究及简便、低成本、获得大量抗菌蛋白“替抗”思路、预防和治疗细菌性疾病提供提供理论和试验基础。以上所述的实施例仅用于说明本发明的技术思想及特点，其目的在于使本领域内的技术人员能够理解本发明的内容并据以实施，不能仅以本实施例来限定本发明的专利范围，即凡本发明所揭示的精神所作的同等变化或修饰，仍落在本发明的专利范围内。
