标题title
美国山核桃CilGLR3.6-4和CilGLR3.6-5基因及其应用
摘要abst
本发明公开了美国山核桃CilGLR3.6‑4和CilGLR3.6‑5基因及其应用，属于基因工程技术领域。所述CilGLR3.6‑4基因的CDS区核苷酸序列如SEQ ID NO.1所示，所述CilGLR3.6‑5基因的CDS区核苷酸序列如SEQ IDNO.2所示。本发明首次公开了CilGLR3.6‑4基因和CilGLR3.6‑5基因在调控美国山核桃对真菌病害抗病性方面的用途。基于美国山核桃抗病植株在真菌侵染条件下上述基因表达显著上调的特性，本发明利用CilGLR3.6‑4基因或CilGLR3.6‑5基因筛选美国山核桃抗真菌病害品种，为筛选抗病良种提供了一种简单易操作且判断结果准确的方法。

说明书desc
技术领域本发明涉及基因工程技术领域，具体涉及美国山核桃CilGLR3.6-4基因和CilGLR3.6-5基因及其应用。背景技术美国山核桃K.Koch)是胡桃科山核桃属植物中最具有经济意义的物种，又名碧根果。美国山核桃的坚果果仁富含健康因子如不饱和脂肪酸、抗氧化多酚类物质和维生素等，这些成分都是人体组织代谢的重要物质之一。美国山核桃还是重要的油料树种之一，其油脂含量高达70％以上。随着人民生活水平不断提高，美国山核桃果仁受到许多人喜爱，我国也逐渐重视美国山核桃引种和栽培。近年来，随着美国山核桃种植规模不断扩大，其病虫害也有所上升，制约了干果产量和产业发展，对大面积建立经济林造成潜在的重大危害。危害美国山核桃的病害相继发生，其中美国山核桃疮痂病、黑斑病、白粉病、炭疽病危害最为严重。因此探究其科学合理有效的防治方法，是目前美国山核桃产业的当务之急。目前，主要种植区对于病害的控制很大程度上依赖于重复且昂贵的杀真菌剂，其导致杀真菌剂对病原菌的抗药性等问题。种植抗病品种能够有效防止美国山核桃病害，减少对环境的负面影响。“Elliot”、“Kanza”是种植广泛的抗病品种，其出仁率高，果仁营养价值高；“Adam’s 5”、“Amling”、“Excel”、“Gafford”、“Headquarters”、“Lakota”对美国山核桃病害也有较高抗性。因此，培育抗病品种、改善林地卫生情况、加强林地管理等也是控制美国山核桃病害的重要措施。发掘抗病种质与相关基因，明确其调控机制是培育抗病品种的基础和和关键。通过同时识别和选择多个基因位点的全基因组数据库资源，对于加速美国山核桃研究和育种方面具有巨大潜力。GLR是配体-门控的阳离子通道，存在于细菌、动物和植物中。植物中谷氨酸受体样基因首次在拟南芥中被鉴定。现有研究表明，植物中的GLRs参与了多种重要的生物过程，如光合作用、调节C/N平衡、植物器官发育、响应非生物胁迫、植物-病原体相互作用和钙介导的信号转导等。在美国山核桃中还未有关于GLR基因的研究，因此，探究其调控生物胁迫响应机制，对于选育抗病品种具有重要意义。发明内容本发明的目的在于提供一种从美国山核桃中克隆得到的具有抗真菌病害作用的基因，为选育抗病品种提供理论依据和相关基因。为实现上述目的，本发明采用如下技术方案：本发明通过对36个美国山核桃品种的基因组学遗传分析，筛选出与美国山核桃疮痂病抗性相关的候选基因，进一步的，通过检测几丁质模拟真菌侵染下抗病品种和易感品种中候选基因的表达水平，发现抗病品种中两个拟南芥AtGLR3.6/GRIP的同源基因Cil_15G_00018V2和Cil_15G_00019V2显著上调表达，具体的，与未处理对比，抗病品种在几丁质处理0.5h和3h时CilGLR3.6-4和CilGLR3.6-5均有显著差异，CilGLR3.6-4的P值为0.003和0.006，CilGLR3.6-5的P值均为0.001。而在易感品种中这两个基因表达变化不明显，甚至是下调表达。上述结果表明这两个基因与抗病性密切相关。CilGLR3.6-4基因由6个外显子和5个内含子组成，其CDS区全长为2589bp，核苷酸序列如SEQ ID NO.1所示。该基因编码一个含862个氨基酸残基的蛋白序列。CilGLR3.6-5基因由20个外显子和19个内含子组成，其CDS区全长为4428bp，核苷酸序列如SEQ ID NO.2所示。该基因编码一个含1475个氨基酸残基的蛋白序列。本发明研究表明，在真菌侵染条件下，美国山核桃抗真菌病害植株中的CilGLR3.6-4基因和CilGLR3.6-5基因表达显著上调，该基因的表达上调提高了美国山核桃对真菌病害的抗病性，有望应用到真菌病害的防治当中。本发明提供了CilGLR3.6-4基因或CilGLR3.6-5基因在调控美国山核桃抗真菌病害中的应用。进一步的，所述真菌病害为疮痂病、炭疽病或枯萎病。基于美国山核桃抗病植株在真菌侵染条件下其CilGLR3.6-4基因或CilGLR3.6-5基因表达显著上调的特性，本发明提供了CilGLR3.6-4基因或CilGLR3.6-5基因在筛选美国山核桃抗真菌病害品种中的应用。进一步的，所述应用包括以下步骤：将几丁质溶液喷洒于待检美国山核桃植株叶片的背面，处理150-200min；分别收集待检植株在几丁质处理前和处理后的至少三个生物重复的叶片，提取RNA，逆转录成cDNA，以此为模板进行CilGLR3.6-4基因或CilGLR3.6-5基因的荧光实时定量PCR分析；比较几丁质处理前后CilGLR3.6-4基因或CilGLR3.6-5基因的相对表达量，如经几丁质处理后其表达量具有统计学意义的显著上调，则判断该待检美国山核桃植株为抗真菌病害品种。步骤中，利用几丁质处理的方法来模拟真菌侵染，有效避免因局部真菌侵染试验造成果园大面积发生真菌病害的风险。进一步的，所述几丁质溶液的浓度为80-200μg/mL，利用喷壶喷洒使得液滴均匀覆盖叶片的背面。作为优选，所述几丁质溶液的浓度为100μg/mL，处理时间为180min。进一步的，选择苗期的美国山核桃植株进行几丁质处理筛选抗病植株。美国山核桃为多年生树种，选择苗期植株筛选可缩短育种时间，尽早筛出抗病品种。步骤中，利用荧光实时定量PCR技术对待检样品中的CilGLR3.6-4基因或CilGLR3.6-5基因的转录表达水平进行分析。进一步的，采用18S rRNA作为内参对目标基因表达值进行矫正处理。进一步的，荧光实时定量PCR采用的引物为：CilGLR3.6-4基因：上游引物：5’-GAGTTACACTGCAAGTCTGA-3’，下游引物：5’-ATTCTTCTGCCGAGTTGAG-3’；CilGLR3.6-5基因：上游引物：5’-AGTCTGACCTCAATCCTTAC-3’，下游引物：5’-CATATCCTTCTTCCGAGTTG-3’；内参基因18S rRNA：上游引物：5’-ACATCTTACCACGATACATAAC-3’，下游引物：5’-AACTTGCGTTCAAAGACTC-3’。步骤中，根据待检叶片中CilGLR3.6-4基因或CilGLR3.6-5基因的表达情况来判断该待检美国山核桃植株是否为抗真菌病害品种。其判断标准为：以自然生长条件下的植株作为对照，若几丁质处理后待检叶片中CilGLR3.6-4基因或CilGLR3.6-5基因的表达量具有统计学意义的显著上调，则判断该植株为抗真菌病害品种。所述具有统计学意义是指根据显著性检验方法所得到的P值＜0.05为差异显著，P值＜0.01为差异极显著。本发明具备的有益效果：本发明首次公开了CilGLR3.6-4基因和CilGLR3.6-5基因在调控美国山核桃对真菌病害抗病性方面的用途。进一步的，基于美国山核桃抗病植株在真菌侵染条件下其CilGLR3.6-4基因或CilGLR3.6-5基因表达显著上调的特性，本发明利用CilGLR3.6-4基因或CilGLR3.6-5基因筛选美国山核桃抗真菌病害品种，为筛选抗病良种提供了一种简单易操作且判断结果准确的方法。附图说明图1为NJ树和亚基因组的种群结构，以及每个加入品种的抗痂等级。颜色比例尺的分数为1分表示该节点的树结构与它的最佳对应节点的树结构相同。K＝4，最佳的子结构。PSRG，美国山核桃抗疮痂病等级。中间的红条的长度显示抗病等级。图2为美国山核桃疮痂病相关品种中最高的5％π和Fst值的图。箭头表示与美国山核桃疮痂病抗性有关的关键候选基因的位点。图3为染色体上选定的美国山核桃疮痂病抗性区域的位置。图4为含有五个推定的GRIP编码基因的候选区域的LD热图，SNPs之间的成对LD表示为D'值，其中红色表示数值为1，黄色表示0。Rosa-red表示基因的CDS区域，绿色表示多态性启动子和CD区域的SNP位点。图5为美国山核桃真菌病原菌形态特征，其中为PDA上W-6正面；为PDA上P-6正面；为PDA上Scab正面。图6为W-6、P-6和Scab侵染抗病品种“Excell”和易感品种“Pawnee”的健康叶片，其中为3d时叶片发病症状。为5d时叶片发病症状。图7为采样方法和几丁质处理用于qPCR分析的示意图。图8为几丁质处理后美国山核桃品种和叶片中涉及几丁质信号通路的CilGLRs关键候选基因的表达情况，其中为CilGLR3.6-4基因，为CilGLR3.6-5基因；橙色字母表示易感品种“Pawnee”，绿色字母表示抗病品种“Excell”，小写字母不同表示差异显著；星号表示易感和抗病品种之间差异显著。图9为几丁质信号通路中关键候选基因作用机制示意图。具体实施方式下面结合具体实施例对本发明做进一步说明。以下实施例仅用于说明本发明，不用来限制本发明的适用范围。在不背离本发明精神和本质的情况下，对本发明方法、步骤或条件所做的修改或替换，均属于本发明的范围。下述实施例中所使用的试验方法如无特殊说明，均为常规方法；所使用的材料、试剂等，如无特殊说明，为可从商业途径得到的试剂和材料。实施例一、美国山核桃疮痂病相关的种群系统发育建立和遗传分析利用种群基因组学鉴定美国山核桃疮痂病相关的候选基因组区域和基因。我们通过纳米孔测序和Hi-C技术构建了美国山核桃品种“Pawnee”的高质量染色体水平的基因组。建立NJ系统发育树来显示36个品种之间的每个亚基因组变异的系统发育关系。如图1所示，两个NJ树的拓扑结构明显形成了每个亚基因组的两个主要支系，但不同主要支系的品种在亚基因组A和亚基因组B之间有差异。亚基因组A或亚基因组B主要支系下的栽培品种的系统发育关系与它们的遗传关系密切相关，但与它们的抗病性没有明显关联。两种系统发育树的内部结构比较显示，亚基因组A和亚基因组B的NJ树之间的所有叶节点和部分内节点的对应性最好为1，根节点的对应性相对较低分值为0.5。在亚基因组水平上对36个栽培品种进行种群结构调查发现，K＝4是数据集的最佳簇数。为了便于亚基因组之间的比较，我们根据可视化结构的颜色将四个K编号指定为K1到K4，并记录了亚基因组和亚基因组B中每个品种的祖先类型。根据结果表明，我们发现10个品种起源于单一祖先，6个品种起源于2-4个祖先，所有这16个品种在亚基因组A和B中具有相同的祖先类型。对于其余20个栽培品种，它们在亚基因组之间的祖先和祖先类型上都不同。这些结果可能反映了复杂的驯化历史和自然和人类选择、种间和种内杂交以及品种间混合的频繁基因流动。由于R种群的抗病性相关区域受到环境的选择，因此与S种群相比，其具有多态性较低特点，同时具有πS/πR的π比和Fst值列在前5％的染色体区域，其确定为与美国山核桃疮痂病抗性相关的选定区域。分析发现共有47个候选区域，涉及185个推定的蛋白编码基因，其中141个位
