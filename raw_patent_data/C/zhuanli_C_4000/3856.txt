标题title
一种水稻温敏雄性不育基因及其应用
摘要abst
本发明涉及一种水稻温敏雄性不育基因及其应用，属于分子育种和基因工程领域。本发明利用CRISPR/Cas9对BIL1启动子和外显子进行了定点敲除获得温敏雄性不育系。bil1突变材料在高温下呈现花粉败育率增高，结实率降低，小穗空壳直立的表型。在低温下花粉败育率有所恢复，说明该基因是温敏雄性不育基因；利用该基因可创制水稻温敏雄性不育系，丰富不育系资源，服务于两系杂交育种。
权利要求书clms
1.一种温敏雄性不育基因，其特征在于：所述温敏雄性不育基因为水稻BIL1基因，具有如下任意一种特征：a、编码氨基酸序列如SEQ ID NO：02所示的蛋白质；b、编码由SEQ ID NO：02经替换、缺失或插入一个或多个氨基酸残基所形成的具有与a同等功能的蛋白质。2.根据权利要求1所述的温敏雄性不育基因，其特征在于：核苷酸序列如SEQ ID NO：01所示。3.一种重组表达载体，含有权利要求2所述的温敏雄性不育基因。4.一种转化体，其特征在于：所述转化体是将权利要求3所述的重组表达载体转化宿主细胞所得。5.权利要求1所述的温敏雄性不育基因、权利要求3所述的重组表达载体或权利要求4所述的转化体在以下任意一种中的应用：水稻花粉育性调控；水稻温敏雄性不育系培育；水稻杂种优势利用育种；水稻花粉育性机制研究。6.一种水稻温敏雄性不育系的创制方法，其特征在于：所述方法的操作如下：利用基因编辑技术对水稻BIL1基因进行定点突变，使其编码的如SEQ ID NO：02所示的蛋白质的表达受阻或活性丧失，水稻表型显示为高温雄性不育、低温育性恢复。7.根据权利要求6所述的方法，其特征在于：所述的水稻为粳稻。8.根据权利要求7所述的方法，其特征在于：所述的粳稻为中花11。
说明书desc
技术领域本发明属于分子育种和基因工程领域，具体涉及一种水稻温敏雄性不育基因BIL1及其在植物花粉育性调控方面的应用。背景技术杂种优势是指具有一定遗传差异的两个个体进行杂交后，所产生的后代在生活力、生长发育速度、抗逆性以及形态大小等方面明显优于杂交双亲的现象。水稻利用杂种优势的途径主要是通过杂交水稻育种，其方式主要有两系法和三系法。三系法需配套保持系，不育系和恢复系。而两系法则不需要保持系，两系法中的不育系的雄性可育与否受周围的环境影响，在特定的环境条件下可以作为保持系。利用两系法减少了三系配套所必须的种植面积，降低了生产成本，同时其不育性受细胞核控制，从而避免了细胞质不育对杂种优势的负效应，以及因细胞质单一化而导致某种毁灭性病虫害爆发的潜在威胁。此外，两系杂交稻比三系杂交稻具有5-10％的增产潜力。因此，温敏雄性不育系的发掘和利用对两系杂交稻的发展具有重要意义。环境敏感型不育系其本质是一些环境敏感基因突变后导致植物对环境更加敏感。目前所鉴定的主要环境敏感雄性不育系有：光敏不育系，湿敏不育系和温敏不育系。其中，温敏不育系是当前两系杂交法中主要应用的不育系。温敏不育材料往往产生于自然突变，出现频率低。因此，用传统手段培育温敏不育系的周期长、效率低，且不可控。发明内容针对现有技术中温敏不育系培育中存在的周期长、效率低，且不可控等问题，本发明利用中花11作为代表性水稻品种，基于CRISPR/Cas9技术编辑BIL1基因，可快速精准获得温敏不育系。本发明的第一方面，是提供一种温敏雄性不育基因，所述温敏雄性不育基因为水稻BIL1基因，具有如下任意一种特征：a、编码氨基酸序列如SEQ NO：01所示。本发明的第二方面，是提供一种重组表达载体，含有所述的温敏雄性不育基因，即水稻BIL1基因。本发明的第三方面，是提供一种转化体，所述转化体是将上述的重组表达载体转化宿主细胞所得。本发明的第四方面，是提供上述温敏雄性不育基因、重组表达载体或转化体在以下任意一种中的应用：水稻花粉育性调控；水稻温敏雄性不育系培育；水稻杂种优势利用育种；水稻花粉育性机制研究。本发明的第五方面，是提供一种水稻温敏雄性不育系的创制方法，所述方法的操作如下：利用基因编辑技术对水稻BIL1基因进行定点突变，使其编码的如SEQ ID NO：02所示的蛋白质的表达受阻或活性丧失，水稻表型显示为高温雄性不育、低温育性恢复。所述的水稻优选为粳稻。所述的粳稻优选为中花11。与现有技术相比，本发明具有以下优点及有益效果：本发明首次发现温敏基因BIL1能够调控花粉育性，可为两系法的新型不育系创制提供靶点。本发明利用PCR的方法从水稻中克隆出BIL1。本发明通过基因编辑手段，创制了中花11为背景的bil1温敏不育系，其具有结实率较低，花粉败育率增高，谷粒空壳等雄性不育系表型，可为温敏不育系提供新种质资源。本发明提供了一种水稻育性相关蛋白及其编码基因，影响植物的花药发育过程中绒毡层的降解，从而影响了花粉成熟，导致水稻不育；低温条件下，该材料能适度恢复育性缺陷，为水稻两系法提供了重要的不育系资源。附图说明图1为BIL1基因的克隆和突变位点示意图。CACCCACCCACCAAGCGCGT为BIL1启动子上的靶点，GGGTTCGTCCGCAAGGTGTA为BIL1外显子上的靶点，PAM代表crRNA互补序列邻近基序，箭头所指为突变位点，bil1-1和bil1-2表示敲除BIL1基因获得的两个纯合等位变异材料，两者均由于碱基的缺失或插入导致BIL1基因的翻译提前终止。图2为WT和bil1突变体的表型分析。图3为bil1相关田间农艺性状以及花粉败育率、结实率量化分析。图4为不同环境温度下WT和bil1的败育率分析，a为高温下的WT与bil1的花粉粒碘染对比；b为低温下的WT和bil1突变体的花粉粒碘染对比；c为24℃和30℃下WT与bil1碘染败育率分析量化图；标尺长度：200μm。图5为bil1与WT在不同花粉发育时期的花药半薄切片分析。具体实施方式BIL1定位于第五号染色体，利用CRISPR/Cas9对BIL1启动子和外显子进行了定点敲除获得温敏雄性不育材料bil1。正反交遗传分析表明，其温敏不育性是受一对隐性核基因所控制，且不育表型与bil1共分离。bil1在高温下呈现花粉败育率增高，结实率降低，小穗空壳且直立的表型。在低温下，bil1花粉败育率有所恢复，表现为可育。综上，我们的研究结果说明bil1材料的不育性受温度控制，属于温敏不育系。所述的BIL1基因核苷酸序列如序列表中SEQ
