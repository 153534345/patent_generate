标题title
一种利用IbMYB1-4紫色基因位点创制甘薯彩色种质资源的方法
摘要abst
本发明涉及一种利用IbMYB1‑4紫色基因位点创制甘薯彩色种质资源的方法，所述IbMYB1‑4基因包括启动子和编码区；其中，所述启动子的核苷酸序列如SEQ ID No.1所示；所述编码区的核苷酸序列如SEQ ID No.2所示。启动子序列的差异导致IbMYB1‑4基因主要在茎外周皮及叶片等部位表达，并改变了甘薯对应部位花青素和类胡萝卜素的积累量，这种花青素与类胡萝卜素的共积累的甘薯茎叶呈现紫黑色。优异基因资源的发掘与利用是我国实现种业振兴的基础，本发明提供了一个优异紫色基因位点，为甘薯营养品质改良和彩色种质创新提供了关键基因资源。
权利要求书clms
1.一种IbMYB1-4基因，其特征在于，所述IbMYB1-4基因包括启动子和编码区；其中，所述启动子的核苷酸序列如SEQ ID No.1所示；所述编码区的核苷酸序列如SEQ ID No.2所示。2.根据权利要求1所述IbMYB1-4基因，其特征在于，所述IbMYB1-4基因还包括3UTR区，所述3UTR区的核苷酸序列如SEQ ID No.3所示。3.含有权利要求1或2任一项所述IbMYB1-4基因的重组载体或重组菌。4.一种检测权利要求1或2任一项所述IbMYB1-4基因的特异性分子标记，其特征在于，所述特异性分子标记包括正向引物，所述正向引物选自SEQ ID No.4-SEQ ID No.8核苷酸序列的至少一种。5.一种检测IbMYB1基因全长的3UTR反向引物，其特征在于，所述反向引物选自SEQ IDNo.9-SEQ ID No.12核苷酸序列的至少一种。6.一种植物育种的方法，其特征在于，包括如下步骤：将携带权利要求1或2任一项所述IbMYB1-4基因的植物作为亲本，与其他待改良植物进行定向杂交，从杂交后代中筛选获得符合育种目标的茎叶为彩色的植物种质资源。7.根据权利要求6所述植物育种的方法，其特征在于，所述待改良植物不携带IbMYB1-4基因。8.权利要求1-2任一项所述IbMYB1-4基因在植物彩色种质创制中的应用。9.根据权利要求8所述IbMYB1-4基因在植物彩色种质创制中的应用，其特征在于，通过IbMYB1-4基因序列调控植物茎叶花色苷积累，实现植物彩色种质创制。
说明书desc
技术领域本发明属于农作物遗传育种领域，具体涉及一种利用IbMYB1-4紫色基因位点创制甘薯彩色种质资源的方法。背景技术甘薯Lam.)是我国重要的粮食、饲料及工业原料作物，年产量超过5000万吨，产值近1万亿元，生产与消费总量均稳居全球首位。近年来随着国民生活水平提高，优质专用型甘薯的消费已超过我国甘薯总消费的一半。紫色甘薯因在自然生长条件下特异性积累花色苷而呈紫色，兼有粮食、医药、保健、观赏等多重用途，已为产业发展做出突出贡献。紫色甘薯可分为薯块紫色的紫心甘薯与茎叶紫色的彩色甘薯，日本与美国最先开始了紫色种质资源的系统性发掘工作，分别主导了紫心甘薯与茎叶紫色甘薯的利用。我国对紫色种质资源以引进和再利用为主，虽然培育紫色甘薯品种已超过60份，但是核心基因资源发掘与利用不足，种质资源的单一利用不仅加剧了品种同质化发展的趋势，还存在知识产权利用的风险。花色苷积累是植物呈现紫色的主要原因，人们对其已有较深入的了解，花色苷合成相关酶基因的组织特异性表达受转录因子的时空调控。这种严格的时空调控作用以MBW转录复合体为核心，包括MYB、bHLH与WDR三类TFs，这些TFs可单独或共同激活酶基因的表达来调控花色苷的特异性积累，其中MYB类TFs对花色苷合成调控的组织特异性最强。甘薯花色苷合成的主要酶基因多数已被预测或克隆，此处不再赘述。紫色基因IbMYB1最先被日本科学家克隆，在紫心甘薯中编码一个R2R3-MYB类转录激活因子。IbMYB1可与IbbHLH2转录因子结合形成转录复合体，靶向花色苷合成酶基因IbANS启动子而调控基因表达。已有研究发现在薯块或茎中外源特异性表达IbMYB1基因，可诱导甘薯同时积累花色苷和类胡萝卜素而呈紫黑色。由于甘薯栽培种是高度杂合的六倍体作物，针对其花色苷积累调控的研究虽然较多，但紫色种质资源遗传多态性形成的分子基础并不完全清楚，这严重限制了紫色基因发掘利用工作的开展，基因资源的匮乏已然成为我国紫色甘薯种质创新的瓶颈。IbMYB1基因对甘薯花色苷积累具有关键调控作用，是目前唯一被公认的紫色基因。研究发现甘薯紫色种质资源中应存在多个IbMYB1基因位点，但是目前被鉴定并广泛利用的只有IbMYB1-2位点。因此，亟需在现有报道的基础上，提供新的基因位点，开辟更为有效的甘薯品质改良技术方案。发明内容为解决现有技术的不足，本发明提供了一种利用IbMYB1-4紫色基因位点创制甘薯彩色种质资源的方法。本发明的一个目的在于，提供一种IbMYB1-4基因，所述IbMYB1-4基因包括启动子和编码区；其中，所述启动子的核苷酸序列如SEQ ID No.1所示；所述编码区的核苷酸序列如SEQ ID No.2所示。进一步地，所述IbMYB1-4基因还包括3UTR区，所述3UTR区的核苷酸序列如SEQ IDNo.3所示。本发明的另一个目的在于，提供上述IbMYB1-4基因的重组载体或重组菌。本发明的另一个目的在于，提供一种检测上述IbMYB1-4基因的特异性分子标记，所述特异性分子标记包括正向引物，所述正向引物选自SEQ ID No.12核苷酸序列的至少一种本发明的另一个目的在于，提供一种植物育种的方法，包括如下步骤：将携带权利要求1或2任一项所述IbMYB1-4基因的植物作为亲本，与其他待改良植物进行定向杂交，从杂交后代中筛选获得符合育种目标的茎叶为彩色的植物种质资源。进一步地，所述待改良植物不携带IbMYB1-4基因。本发明的另一个目的在于，提供上述IbMYB1-4基因在植物彩色种质创制中的应用。进一步地，通过IbMYB1-4基因序列调控植物茎叶花色苷积累，实现植物彩色种质创制。本发明具有以下有益效果：1.本发明通过FPNI-PCR同源克隆的方法从甘薯紫色种质资源中克隆到编码IbMYB1转录因子的新拷贝IbMYB1-4，该基因序列的编码区与已知IbMYB1基因序列一致，而启动子序列完全区别于已报道的IbMYB1-1、IbMYB1-2基因及发明人此前克隆到的IbMYB1-3基因。启动子序列的差异导致IbMYB1-4基因主要在茎外周皮及叶片等部位表达，并改变了甘薯对应部位花青素和类胡萝卜素的积累量，这种花青素与类胡萝卜素的共积累的甘薯茎叶呈现紫黑色。本发明的启动子序列，提供了IbMYB1基因的一个新拷贝，该启动子能够有效提高编码区的表达水平，二者的结合促进了翻译形成特定蛋白质，提升了花色苷的含量，进一步提高了植物变色效果。2.本发明通过构建“IbMYB1-4-pCAMBIA1300”的表达载体转入拟南芥，可导致拟南芥莲座叶、茎等部位相对于野生型植株呈现明显的紫色，有效实现了植物彩色种质创制。3.甘薯遗传群体中茎叶的紫色性状与IbMYB1-4基因的分子标记紧密连锁，利用IbMYB1-4基因在甘薯六倍体基因组中的基因剂量效应，可创制紫黑色茎叶的彩色甘薯。此外利用携带该基因的“黑叶”作父本，与一类叶绿体突变的“金叶甘薯”杂交，可获得茎叶为浅红色至深红色的子代，为红色系列的彩色甘薯种质创制提供了有效方案。优异基因资源的发掘与利用是我国实现种业振兴的基础，本发明提供了一个优异紫色基因位点，为甘薯营养品质改良和彩色种质创新提供了关键基因资源。附图说明图1示出了“紫湘薯20”与“湘薯20”的主要表型差异；其中，图1为“紫湘薯20”，从上到下对应茎顶端、叶片、茎、薯块表型图；图1为“湘薯20”，从上到下对应茎顶端、叶片、茎、薯块表型图。图2示出了“紫湘薯20”与“湘薯20”的色素含量检测图；其中，图2为花青素含量检测图；图2为类胡萝卜素含量检测图；图2为叶绿素含量检测图；图2为成熟叶片中的色素含量检测图；图2为茎外周皮中的色素含量检测图；图2为薯皮中的色素含量检测图。图3示出了不同组织部位花色苷总含量的检测结果图。图4示出了花色苷合成途径的主要通路。图5示出了合成通路相关基因的表达，图例中的基因表达值为转录组TPM值以10为底取的对数值，图中“+”与“-”分别表示基因表达量上调或下调超过2倍。图6示出了“紫湘薯20”与“湘薯20”的RT-qPCR检测结果图；其中，图6为IbMYB1检测结果图；图6为IbbHLH2检测结果图；图6为IbANS检测结果图。图7示出了“黑叶”及其杂交子代的主要表型；其中，左上角为“黑叶”，其余均为其杂交子代。图8示出了紫色种质资源基因组IbMYB1基因各拷贝的基因型特异性分子标记检测结果。图9示出了qPCR荧光绝对定量法检测结果图。图10示出了IbMYB1-4基因在拟南芥中表达后生长45d的植株主要表型；其中，左1为相同生长条件下的野生型col表型，左2和左3为不同转基因株系的F1代材料表型。图11示出了拟南芥转基因F1代植株中IbMYB1-4基因的鉴定结果图。图12示出了采用IbMYB1-4基因创制茎叶红色的彩色甘薯表型图。图13示出了“黑叶”、“金叶甘薯”及其杂交子代中的色素含量。具体实施方式为了更清楚地说明本发明的技术方案，列举如下实施例。实施例中所出现的原料、反应和后处理手段，除非特别声明，均为国内外同一领域常见原料，以及本领域技术人员所熟知的技术手段。本发明中的词语“优选的”、“优选地”、“更优选的”等是指，在某些情况下可提供某些有益效果的本发明实施方案。然而，在相同的情况下或其他情况下，其他实施方案也可能是优选的。此外，对一个或多个优选实施方案的表述并不暗示其他实施方案不可用，也并非旨在将其他实施方案排除在本发明的范围之外。应当理解，除了在任何操作实例中，或者以其他方式指出的情况下，表示例如说明书和权利要求中使用的成分的量的所有数字应被理解为在所有情况下被术语“约”修饰。因此，除非相反指出，否则在以下说明书和所附权利要求中阐述的数值参数是根据本发明所要获得的期望性能而变化的近似值。IbMYB1-4基因的发现过程及表征：“湘薯20”茎叶主要颜色为绿色，在组培的过程发明人发现一个芽变的单株，茎变为紫色，叶脉的紫色加深，薯皮由浅红色
