标题title
利用马克斯克鲁维酵母制备可溶性铁蛋白纳米颗粒的方法
摘要abst
本发明属于生物技术领域，具体为用马克斯克鲁维酵母制备可溶性铁蛋白纳米颗粒的方法。本发明首先提供高效重组表达铁蛋白的马克斯克鲁维酵母工程菌株，是将密码子优化的人重链铁蛋白基因FTH1或幽门螺旋杆菌铁蛋白基因FTNA或幽门螺旋杆菌铁蛋白基因截短体FtnA的表达载体转化马克斯克鲁维酵母所构建获得，分别记为：KM‑Fth1、KM‑Fth1‑14Δ、KM‑FtnA和KM‑FtnA‑3Δ；进一步将构建的马克斯克鲁维酵母工程菌株经发酵培养得到可溶性铁蛋白纳米颗粒。本发明提供的铁蛋白纳米颗粒的制造方法，在5L发酵罐中的产量达到11g/L。
权利要求书clms
1.一种高效重组表达铁蛋白的马克斯克鲁维酵母工程菌株，其特征在于，由如下步骤构建得到：密码子优化人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA；通过PCR扩增获得人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA的截短体基因，记为FTH1-14Δ和FTNA-3Δ；分别将上述铁蛋白全长基因与截短体基因插入到马克斯克鲁维酵母表达载体pUKDN115，构建获得重组载体，记为：pUKDN115-Fth1、pUKDN115-Fth1-14Δ、pUKDN115-FtnA，和pUKDN115-FtnA-3Δ；重组载体转化马克斯克鲁维酵母FIM-1URA3Δ菌株，获得的重组表达铁蛋白的马克斯克鲁维酵母菌株，记为：KM-Fth1、KM-Fth1-14Δ、KM-FtnA和KM-FtnA-3Δ；其中：所述人重链铁蛋白基因FTH1，其核苷酸序列为SEQ ID No.1所示，氨基酸序列为SEQIDNo.2所示；所述人重链铁蛋白截短体基因FTH1-14Δ，其核苷酸序列为SEQ ID No.3所示，氨基酸序列为SEQ ID No.4所示；所述幽门螺杆菌铁蛋白基因FTNA，其核苷酸序列为SEQ ID No.5所示，氨基酸序列为SEQ ID No.6所示；所述幽门螺杆菌铁蛋白截短体基因FTNA-3Δ，其核苷酸序列为SEQ ID No.7所示，氨基酸序列为SEQ ID No.8所示；所述马克斯克鲁维酵母FIM-1URA3Δ菌株是以保藏于中国普通微生物菌种保藏管理中心的保藏编号为CGMCC No.10621的马克斯克鲁维酵母FIM-1作为出发菌株，通过分子生物学技术方法，敲除尿嘧啶合成基因URA3作为筛选标记构建获得。2.一种制备可溶性铁蛋白纳米颗粒的方法，其特征在于，权利要求1所述的马克斯克鲁维酵母工程菌株，经发酵培养得到可溶性铁蛋白纳米颗粒，具体步骤为：将所述的马克斯克鲁维酵母工程菌株，接种于酵母培养基培养发酵；收集发酵培养的酵母细胞；酵母细胞破碎回收获得铁蛋白纳米颗粒。3.根据权利要求2所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，所述酵母培养基包含硫酸铵、葡萄糖、硫酸镁、维生素成分的培养基为原料，控制葡萄糖流加量进行高密度发酵培养。4.根据权利要求2所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，所述酵母培养基是以SM培养基为基础改良获得的SMO基础培养基，所述SMO基础培养基组成为：葡萄糖60g/L，2SO4 18g/L，MgSO4·7H2O 0.5g/L，KH2PO4 3g/L，维生素1mL/L，柠檬酸/K2HPO4缓冲液200mM，pH 5.5。5.根据权利要求2所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，在5L发酵罐中通过补料培养基分批发酵，控制发酵温度为30-35℃，用氨水自动控制pH为5.5—6.5，无菌过滤空气以3L/min供应氧气，溶氧与搅拌速率串级联速；所用补料培养基中，控制葡萄糖补料量获得高产量铁蛋白纳米颗粒，葡萄糖的补料量范围在1000g-1500g。6.根据权利要求2所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，所述回收获得铁蛋白纳米颗粒，是以添加选自Triton X-100、NP-40和/或Tween 20的表面活性剂的缓冲液破碎酵母细胞，获得可溶性铁蛋白纳米颗粒；其中表面活性剂添加量为0.25％-2％。7.根据权利要求5所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，Triton X-100的添加量为1％，NP-40的添加量为NP-401％，Tween 20的添加量为2％。8.根据权利要求1-7之一所述的制备可溶性铁蛋白纳米颗粒的方法，其特征在于，铁蛋白纳米颗粒的粒径为10-14nm。
说明书desc
技术领域本发明属于生物技术领域，具体涉及一种利用马克斯克鲁维酵母制备可溶性铁蛋白纳米颗粒的方法。背景技术铁蛋白纳米颗粒天然存在于生物体内，包括动物、细菌和植物。不同物种来源的铁蛋白氨基酸序列存在很大的差异，同源度仅为20％。铁蛋白是由24个铁蛋白亚基组装成的纳米颗粒，其外径约为12nm，内径约为8nm，内腔可容纳高达4500个铁原子。因此，铁蛋白纳米颗粒不仅在生物体的铁稳态中发挥作用，也广泛应用于生物医学和临床研究。铁蛋白纳米颗粒作为搭载舱，用于搭载生物活性物质提高稳定性和生物利用度，搭载药物靶向到肿瘤细胞，搭载显像剂进行活体成像等等，生物相容性好。铁蛋白纳米颗粒可以利用基因重组表达技术进行制备。目前重组表达的铁蛋白主要有人重链铁蛋白、人轻链铁蛋白、幽门螺杆菌铁蛋白等。用于重组表达铁蛋白的宿主分别是大肠杆菌、毕赤酵母、多形汉逊酵母、以及CHO细胞。人重链铁蛋白在大肠杆菌中的重组表达量有7.26g/L，毕赤酵母和多形汉逊酵母中的重组表达量分别25.7mg/L和1.9g/L)。人轻链铁蛋白在大肠杆菌中的重组表达量是10mg/L。使用杆状病毒表达系统在昆虫细胞中重组表达幽门螺杆菌铁蛋白，表达量是11.25mg/L。由此可见，铁蛋白的重组表达水平大多数在mg级别。虽然，原核生物大肠杆菌重组表达铁蛋白的产量可达7.26g/L，但大肠杆菌生长过程中产生内毒素的天然缺陷，也是重组表达铁蛋白纳米颗粒在生物医药应用中的风险因素。铁蛋白纳米颗粒的高产菌株是工业生物技术产业的核心技术，重组表达的活性蛋白的高产和安全有效，直接代表了生物产业的技术水平。马克斯克鲁维酵母FIM1是食品安全级的宿主菌，利用马克斯克鲁维酵母高效制备铁蛋白纳米颗粒的研究，至今没有见到相关报道。发明内容本发明的目的组提供一种高效制备可溶性铁蛋白纳米颗粒的方法，铁蛋白纳米颗粒的产量达到11g/L。本发明提供的制备可溶性铁蛋白纳米颗粒的方法，是利用重组表达人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA的马克斯克鲁维酵母工程菌株，经发酵培养得到可溶性铁蛋白纳米颗粒。因此，本发明首先提供一种高效重组表达铁蛋白的马克斯克鲁维酵母工程菌株。所述的高效重组表达铁蛋白的马克斯克鲁维酵母工程菌株，由如下步骤构建得到：密码子优化人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA；通过PCR扩增获得人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA的截短体基因，记为FTH1-14Δ和FTNA-3Δ；分别将上述铁蛋白全长基因与截短体基因插入到马克斯克鲁维酵母表达载体pUKDN115，构建获得重组载体，记为：pUKDN115-Fth1、pUKDN115-Fth1-14Δ、pUKDN115-FtnA，和pUKDN115-FtnA-3Δ；重组载体转化马克斯克鲁维酵母FIM-1URA3Δ菌株，获得的重组表达铁蛋白的马克斯克鲁维酵母菌株，记为：KM-Fth1、KM-Fth1-14Δ、KM-FtnA和KM-FtnA-3Δ。进一步，对重组表达铁蛋白的马克斯克鲁维酵母菌株进行发酵培养，获得可溶性铁蛋白纳米颗粒。具体地，本发明提供的制备可溶性铁蛋白纳米颗粒的方法，是利用重组表达人重链铁蛋白基因FTH1或幽门螺杆菌铁蛋白基因FTNA的马克斯克鲁维酵母工程菌株，经发酵培养得到可溶性铁蛋白纳米颗粒，具体步骤为：将所述的马克斯克鲁维酵母工程菌株，接种于酵母培养基培养发酵；收集发酵培养的酵母细胞；酵母细胞破碎回收获得铁蛋白纳米颗粒。其中，所述酵母培养基包含葡萄糖、硫酸铵、硫酸镁、维生素等成分培养基为原料，控制葡萄糖流加量进行高密度发酵培养。所述酵母培养基为以SM培养基为基础改良获得的SMO基础培养基，所述SMO基础培养基组成为：葡萄糖60g/L，2SO4 18g/L，MgSO4·7H2O 0.5g/L，KH2PO4 3g/L，维生素1mL/L，柠檬酸/K2HPO4缓冲液200mM，pH 5.5。所述回收获得铁蛋白纳米颗粒，是以添加选自Triton X-100、NP-40和或Tween 20的表面活性剂的缓冲液破碎酵母细胞，获得可溶性铁蛋白纳米颗粒，优选地，表面活性剂添加范围0.25％-2％。本发明还包括所述的马克斯克鲁维酵母菌株，以及可溶性铁蛋白纳米颗粒制备方法在生产铁蛋白、铁蛋白纳米颗粒及其衍生物中的应用。本发明中：所述的人重链铁蛋白基因FTH1具有SEQ ID No.1所示核苷酸序列和SEQ ID No.2所示氨基酸序列。所述的人重链铁蛋白截短体基因FTH1-14Δ具有SEQ ID No.3所示核苷酸序列和SEQ ID No.4所示氨基酸序列。所述的幽门螺杆菌铁蛋白基因FTNA具有SEQ ID No.5所示核苷酸序列和SEQ IDNo.6所示氨基酸序列。所述的幽门螺杆菌铁蛋白截短体基因FTNA-3Δ具有SEQ ID No.7所示核苷酸序列和SEQ ID No.8所示氨基酸序列。所述的马克斯克鲁维酵母FIM-1URA3Δ菌株是以马克斯克鲁维酵母FIM-1，通过常规分子生物学技术方法，敲除尿嘧啶合成基因URA3作为筛选标记构建获得。所述铁蛋白纳米颗粒，电镜下观察形成球形纳米颗粒，直径为10-14nm，主要为12nm。为提高铁蛋白纳米颗粒可溶产量，本发明在摇瓶中通过三因素三水平正交试验优化发酵培养基中的三个因素葡萄糖、硫酸铵、硫酸镁，获得铁蛋白高产的SMO培养基，既可以提高铁蛋白Fth1的可溶表达量，也可以提高FtnA-3Δ的可溶表达量。其中培养基中的葡萄糖含量对铁蛋白表达量的影响最大。进一步地，本发明将马克斯克鲁维酵母工程菌株在5L发酵罐中，通过补料分批发酵，控制发酵温度为30-35℃，用氨水自动控制pH为5.5—6.5，无菌过滤空气以3L/min供应氧气，溶氧与搅拌速率串级联速。所用补料培养基中，控制葡萄糖补料量获得高产量铁蛋白纳米颗粒，葡萄糖的补料量范围在1000g-1500g；优选地，补料量1300g葡萄糖，铁蛋白纳米颗粒的产量达到11g/L。在实施方式中，表达人重链铁蛋白FTH1全长基因和截短体基因FTH1-14Δ，都获得高表达量可溶性铁蛋白纳米颗粒。表达幽门螺杆菌铁蛋白截短体基因FTNA-3Δ和全长基因FTNA，也都获得高表达量可溶性铁蛋白纳米颗粒。进一步地，本发明还提供提高铁蛋白纳米颗粒可溶性回收量的方法，在细胞裂解液中添加不同的表面活性剂，表面活性剂可以是Triton X-100、NP-40、Tween 20中的一种，添加的量可以是0.25％-2％。在一个优选的实施方式中，添加1％Triton X-100、1％NP-40或2％Tween 20，可溶铁蛋白纳米颗粒的回收率达≥80％。本发明的特点和优势：本发明首次对编码人重链铁蛋白Fth1和幽门螺杆菌铁蛋白FtnA的基因经密码子优化和基因截短表达，得到高效重组表达铁蛋白的马克斯克鲁维酵母工程菌株。用该马克斯克鲁维酵母工程菌株，经发酵培养得到可溶性铁蛋白纳米颗粒，可大大提高铁蛋白纳米颗粒的产量，达到11g/L，远高于报道的在大肠杆菌中的重组表达量7.26g/L，提高了50％以上。另外，与现有技术相比，本发明方法在发酵周期、制备得率等方面都具有明显的优势。附图说明图1为马克斯克鲁维酵母重组表达人重链铁蛋白Fth1及其截短体。其中，SDS-PAGE分析Fth1和Fth1-14Δ在K.marxianus中的重组表达。红色箭头指示表达的铁蛋白条带。M:PageRuler预染蛋白Marker；T：细胞裂解物；S：细胞裂解液的上清；P：细胞裂解液的沉淀；Control：含空载体的K.marxianus菌株。Fth1和Fth1-14Δ的可溶性分析。通过GenoSens软件测量可溶和不可溶的蛋白条带强度。透射电镜分析细胞裂解液上清中的Fth1和Fth1-14Δ铁蛋白纳米颗粒。Bar，100nm。t检验分析统计学差异，*P＜0.05；**P＜0.01；***P＜0.001。图2为马克斯克鲁维酵母重组表达幽门螺旋杆菌铁蛋白及其截短体。其中，SDS-PAGE分析FtnA和FtnA-3Δ在K.marxianus中的重组表达。红色箭头指示表达的铁蛋白条带。M:PageRuler预染蛋白Marker；T：细胞裂解物；S：细胞裂解液的上清；P：细胞裂解液的沉淀；Control：含空载体的K.marxianus菌株。FtnA和FtnA-3Δ的可溶性分析。透射电镜分析细胞裂解液上清中的FtnA和FtnA-3Δ铁蛋白纳米颗粒。Bar，100nm。*P＜0.05；**P＜0.01；***P＜0.001。图3为正交设计优化马克斯克鲁维酵母工程菌株表达铁蛋白纳米颗粒的条件。其中，正交试验分析三种不同因素三个水平对Fth1可溶表达量的影响。优化前后的发酵培养基对Fth1可溶表达量的影响。优化前后的发酵培养基对FtnA-3Δ可溶表达量的影响。SM是优化前的培养基，SMO是优化后的培养基。图4为优化铁蛋白纳米颗粒的可溶回收条件。其中，细胞裂解液中添加不同浓度的Triton
