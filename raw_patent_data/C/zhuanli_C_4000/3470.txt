标题title
与甜瓜种子重量基因共分离的SNP分子标记及其应用
摘要abst
本发明提供了一种与甜瓜种子重量基因共分离的SNP分子标记，位于甜瓜基因组DHL92 v3.6.1第6号染色体的MELO3C014002基因的启动子核苷酸序列的第35765030至35765356位如SEQ ID NO.1所示，SEQ ID NO.1的第257位为C/T。本发明还提供了一种上述SNP分子标记的检测引物。本发明还提供一种含上述SNP分子标记或检测引物的试剂盒。本发明又提供了一种上述SNP分子标记、检测引物或试剂盒在甜瓜种子重量分子标记辅助育种中的应用，能提高选择的准确性、加快新品种培育的进程。本发明还提供了一种利用上述SNP分子标记、检测引物或试剂盒筛选甜瓜种子重量的方法。
权利要求书clms
1.一种与甜瓜种子重量基因共分离的SNP分子标记，其特征在于：它位于甜瓜基因组DHL92 v3.6.1第6号染色体的MELO3C014002基因的启动子的核苷酸序列的第35765030至35765356位，如SEQ ID NO .1所示，SNP位点位于序列SEQ ID NO .1的第257位且碱基为C/T。2.根据权利要求1所述的SNP分子标记，其特征在于：所述SNP位点上，碱基为C时为重甜瓜种子；碱基为T时为轻甜瓜种子。3. 一种与甜瓜种子重量基因共分离的SNP分子标记的检测引物，其特征在于：包括一对检测引物对，正向引物的核苷酸序列如SEQ ID NO.2所示，反向引物的核苷酸序列如SEQID NO.3所示。4.一种含权利要求1或2所述的SNP分子标记或权利要求2所述的检测引物的试剂盒。5.权利要求1或2所述的SNP分子标记、权利要求3所述的检测引物、权利要求4所述的试剂盒在甜瓜种子重量分子标记辅助育种中的应用。6.权利要求1或2所述的SNP分子标记、权利要求3所述的检测引物、权利要求4所述的试剂盒在鉴定筛选甜瓜种子重量中的应用。7.一种筛选甜瓜种子重量的方法，包括以下步骤：提取待测甜瓜样本的基因组DNA；以步骤提取的基因组DNA作为模板，利用权利要求1或2所述的SNP分子标记、权利要求3所述的检测引物、权利要求4所述的试剂盒进行PCR扩增；纯化PCR扩增产物并进行基因测序，检测基因型；将检测到的基因型，与权利要求1或2所述的SNP分子标记进行比对，若基因型为C，则待测甜瓜种子为重甜瓜种子；若基因型为T，则待测甜瓜种子为轻甜瓜种子。
说明书desc
技术领域本发明属于分子生物技术领域，具体涉及一种与甜瓜种子重量基因共分离的SNP分子标记及其应用。背景技术甜瓜是主要的葫芦科蔬菜作物之一，栽培面积广泛，具有重要的经济价值。中国是甜瓜生产大国，甜瓜在果蔬生产中占有重要地位。种子是植物生命的起点和终点，是植物生长发育的重要决定因素。在许多以种子为主要生产器官的作物的驯化和育种过程中，种子重量从理论上反映了种胚所存的物质能量，具有重要的生态指示意义。种子愈重；其活力愈高、出苗迅速整齐、出苗高峰明显；幼苗生长占优势、成苗素质好，因此；在农业生产中，为保证一播全苗和培育壮苗应优选粒大饱满的种子。目前我国的甜瓜种子生产已基本实现了杂种化，使甜瓜具有较强的杂种优势，因此加大甜瓜种子的生产培育可以推定我国甜瓜产业的进一步发展，所以从分子水平深入研究甜瓜种子的发育机制可以加速甜瓜种子的育种进程。相对于拟南芥、水稻和小麦等作物对种子重量的研究，瓜类种子重量基因的QTL定位与克隆研究目前还处于初级阶段，主要集中于对控制种子重量基因进行初步定位与遗传分析方面。根据目前对甜瓜种子重量的研究结果显示，现有技术将甜瓜种子重量定位在12条染色体上，没有一个统一的定位，有关甜瓜种子重量调控基因的精细定位、克隆及其遗传结构研究还未见报道，也未能找到与甜瓜种子重量紧密连锁的分子标记。因此，现有的甜瓜种子重量基因的定位结果难以用于甜瓜的分子育种实践。发明内容有鉴于此，本发明确有必要提供一种与甜瓜种子重量基因共分离的SNP分子标记及其应用，以解决上述问题。具体地，本发明提供一种与甜瓜种子重量基因共分离的SNP分子标记，其中，SNP分子标记位于甜瓜基因组DHL92 v3.6.1第6号染色体的MELO3C014002基因的启动子核苷酸序列的第35765030至35765356位如SEQ ID NO.1所示，SNP位点位于如SEQ ID NO.1所示序列的第257位，且所述SNP位点碱基为C/T。基于上述，所述SNP位点上，碱基为C时为重甜瓜种子；碱基为T时为轻甜瓜种子。其中，所述重甜瓜种子的千粒重大于17.97g，所述轻甜瓜种子千粒重小于17.97g。本发明还提供一种与甜瓜种子重量基因共分离的SNP分子标记的检测引物，包括一对检测引物对，正向引物的核苷酸序列如SEQ ID NO.2所示，反向引物的核苷酸序列如SEQ ID NO.3所示。本发明还提供一种包含上述检测引物的试剂盒。本发明还提供一种含上述SNP分子标记或检测引物或试剂盒在甜瓜种子重量分子标记辅助育种中的应用。本发明还提供一种含上述SNP分子标记或检测引物或试剂盒在鉴定筛选甜瓜种子重量中的应用。本发明还提供一种筛选甜瓜种子重量的方法，所述方法包含应用所述的SNP分子标记组合、所述的特异性引物或所述试剂盒的步骤。所述筛选甜瓜种子重量的方法，包括以下步骤：提取待测甜瓜样本的基因组DNA；以步骤提取的基因组DNA作为模板，利用所述的特异性引物或所述的试剂盒进行PCR扩增；纯化PCR扩增产物并进行基因测序，检测基因型；将检测到的基因型，与所述的SNP分子标记组合进行比对，如果基因型为C，则待测甜瓜种子为重甜瓜种子；如果基因型为T，则待测甜瓜种子为轻甜瓜种子。因此，本发明提供的上述SNP分子标记的核苷酸序列如SEQ ID NO.1所示，SNP位点位于SEQ ID NO.1所示序列的第257位，该SNP位点为C/T，重甜瓜种子的碱基为C，轻甜瓜种子的碱基为T，所以，利用本发明提供的上述SNP分子标记能够准确区别不同类型甜瓜材料的种子重量，具有很好地实用性和代表性，以此作为甜瓜育种过程中种子重量性状的辅助选择标记，能提高选择的准确性、加快品种培育的进程。附图说明图1基于高密度遗传图谱的甜瓜种子重量QTL定位图。图2基于BSA分池技术的甜瓜种子重量QTL定位图。图3为Chr6_35765286位点不同基因型样本种子大小相关性状的箱形图。图4为鉴定甜瓜种子重量的测序基因分型图。其中的序列表说明如下：SEQ NO.1所示。本实施例提供的分子标记CmSWT与甜瓜种子千粒重紧密连锁，双亲序列分别位于甜瓜基因组DHL92 v3.6.1第6号染色体的MELO3C014002基因的启动子2Kb区域内的第35765030位核苷酸至第35765356位核苷酸，大小为327bp，如SEQ ID NO.1所示。通过对双亲序列比对，明确双亲上存在SNP突变，MELO3C014002基因启动子区域内的35765286位点上，即分子标记CmSWT的核苷酸序列如SEQ ID NO.1的第257位为SNP位点，该位点的碱基发生C/T突变。具体地，在所述SNP位点，重甜瓜种子的碱基为C，千粒重大于17.97g；轻甜瓜种子的碱基为T，千粒重小于17.97g。本实施例提供的分子标记CmSWT的获取：利用小种子甜瓜品系581和大种子甜瓜品系906为亲本获得F2分离群体，通过双亲全基因组重测序结合F2代群体ddRAD-seq技术构建高密度遗传图谱，通过2018年F2群体、2019年春秋F2:3群体3个季节的种子千粒重性状调查，在甜瓜基因组第1、2、3、6、7、10、11号染色体上定位到与种子重量相关的QTL位点如图1所示，由于定位区间较多，无法开展精细定位工作，因而利用扩大的F2群体结合BSA分池技术，对极端个体混合基因池进行重测序，其中，所述极端个体混合基因池包括大种子池和小种子池，将控制种子重量基因的初步定位在甜瓜基因组第6号染色体上，如图2所示。两种定位方法结合后将控制甜瓜种子重量基因的候选区间定位至6号染色体35741213～35835025区段内。在区段内存在12个候选基因，通过对双亲序列比对及自然群体验证，发现在MELO3C014002基因启动子区域内发现一个SNP位点，该位点位于甜瓜基因组6号染色体35765286位点上，且碱基有C/T突变。根据该突变点开发分子标记，命名为CmSWT，上游引物CmSWTL，下游引物CmSWTR。实施例二自然群体法区分验证甜瓜种子重量利用表1所示的180份甜瓜种质材料重测序数据对Chr6_35765286位点进行自然群体验证，结果见表1。表1中的“种子重量”为甜瓜种子千粒重，其是利用万分之一天平称量每30粒种子的总质量，最终折算成千粒重，单位用g表示。表1植物材料来源及相区分的SNP位点/＞将表1中的180份种子的平均值作为划分种子千粒重轻重的标准，大于平均值的为重种子，小于平均值的为轻种子，经计算，180份种子的千粒重平均值为17.97g。确定180份甜瓜种子千粒重大小的划分标准为：重甜瓜种子、千粒重＞17.97g；轻甜瓜种子、千粒重＜17.97g。通过重测序数据获得上述180份甜瓜种质材料所有SNP位点的基因分型，并使用Graphpad Prism软件绘制位SNP点不同基因型样本种子大小相关性状的箱形图，如图3所示。结合表1和图3可知：千粒重＞17.97g的重甜瓜种子在上述SNP位点表现为碱基为C；千粒重＜17.97g的轻甜瓜种子在上述SNP位点表现为碱基为T。证明了本发明实施例提供的SNP位点可准确区分甜瓜种子重量。实施例三检测引物及试剂盒本实施例提供一种与甜瓜种子重量基因共分离的SNP分子标记的检测引物，包括一对检测引物对，正向引物的核苷酸序列如SEQ ID NO.2所示，即上游引物CmSWTL；反向引物的核苷酸序列如SEQ ID NO.3所示，即下游引物CmSWTR。本实施例还提供一种包含上述检测引物的试剂盒。实施例四鉴定甜瓜种子重量的方法本实施提供一种利用实施例一提供的分子标记CmSWT或实施例二提供的特异性检测引物或试剂盒鉴定甜瓜种子重量的方法，具体方法如下：步骤1、利用改良过的CTAB法提取苗期待测甜瓜DNA从表1所示的甜瓜种质材料随机挑选20种种子材料在相同的条件下进行种植，待植株三叶一心时作为检测样品，取幼叶于2mL离心管中，用冰袋带回，使用真空冷冻抽干机进行抽干，抽干后用研磨仪磨碎至甜瓜叶粉末状；其中，本步骤中的20种种子材料由随机挑选的10种种子千粒重小于17.97g的甜瓜材料和10种种子千粒重大于17.97g的甜瓜材料组成；该20种种子的编号分别为：CB22、CB28、CB29、CB34、CB36、CB37、CB38、CB39、CB46、CB48、CH4、CH5、CH9、CH11、CH17、CH29、CH30、CH39、CH45、CH53。向离心管中加入1000μL已经在65℃水浴锅中预热1h的2％CTAB，震荡，使甜瓜叶粉末于液体充分接触，放入65℃水浴锅内1h，期间每10min混匀一次；水浴后晾至室温，于4℃采用13000r/min的转速离心15min；离心后，将上清液吸出，移入新的2mL离心管中，加入900μL的氯仿/异戊醇溶液，充分摇匀后，于4℃采用13000r/min的转速离心15min；再重复2次此步骤；吸出600～700μL上清液并置于1.5mL离心管中，加入700μL 4℃预冷的无水乙醇，充分摇匀，4℃沉淀1h；取出离心管，于4℃采用13000r/min的转速离心15min，离心后吸出上清液，保留沉淀；用75％的乙醇清洗2～3次，室温晾干，加入蒸馏水补足50μL，获得待测甜瓜DNA样品，该待测甜瓜DNA样品可在-80℃冰箱永久保存；本实施例中获得24份待测甜瓜DNA样品。步骤2、分别以上述步骤1得到的待测甜瓜DNA样品为模板，用实施例二提供的检测引物或试剂盒进行PCR扩增，获得PCR扩增产物PCR扩增的反应体系为10μL反应体系，由下列成分组成：DNA模板1μL、2×Es TaqMasterMix4μL、正向引物2pM 1μL、反向引物2pM 1μL、蒸馏水3μL，其中的Es Taq MasterMix包括Es Taq DNA Polymerase、Mg2+和dNTPs；PCR反应条件：94℃预变性5min，94℃变性30s，55℃退火30s，72℃延伸30s，共25个循环，再72℃延伸7min，4℃保温保存。步骤3、利用1.5％的琼脂糖凝胶电泳对步骤2所得的各个PCR扩增产物进行检测，确认PCR扩增产物是否出现目标条带，对出现目标条带的PCR扩增产物进行纯化，纯化的PCR扩增产物由擎科生物科技进行Sanger测序。使用NovoSNP软件检查测序图文件，并通过提取或比较峰信号来检测基因型。所选24种甜瓜种质材料对应的6号染色体中的第35765273～35765298位点的核苷酸序列如图4所示。从图4中可以看出：经核苷酸序列比对发现，所选的20种甜瓜品种在对应的6号染色体35765286位点为SNP位点，即SEQ ID NO.1所示的核苷酸序列的第257位为SNP位点；且选择的10种种子千粒重小于17.97g的甜瓜材料在SNP位点的碱基为C，选择的10种种子千粒重大于17.97g的甜瓜材料在SNP位点的碱基为T；该检测结果与表1所示甜瓜种子重量性状结果一致，从而证明了本发明实施例提供的方法可以准确鉴定甜瓜种子重量。因此，利用本实施例一提供的分子标记CmSWT或实施例二提供的特异性检测引物或试剂盒通过PCR产物与测序，可以在甜瓜苗期将甜瓜种子重量区分开，实现在苗期判断甜瓜种子大小，快速筛选所需要的个体，加快甜瓜的选育过程。最后应当说明的是：以上实施例仅用以说明本发明的技术方案而非对其限制；尽管参照较佳实施例对本发明进行了详细的说明，所属领域的普通技术人员应当理解：依然可以对本发明的具体实施方式进行修改或者对部分技术特征进行等同替换；而不脱离本发明技术方案的精神，其均应涵盖在本发明请求保护的技术方案范围当中。
