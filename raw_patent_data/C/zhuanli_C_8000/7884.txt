标题title
一种大肠杆菌BW25113菌株yjgW基因敲除突变株及其应用
摘要abst
本发明涉及基因工程技术领域，具体涉及一种大肠杆菌BW25113菌株yjgW基因敲除突变株及其应用。本发明通过敲除大肠杆菌BW25113中yjgW基因后获得基因敲除突变株K12‑yjgW，以秀丽隐杆线虫作为模式生物，研究了该突变株K12‑yjgW对线虫聚集行为的影响。本发明通过与喂食野生型大肠杆菌相比，发现喂食基因敲除大肠杆菌的线虫聚集行为显著提高，为制备治疗神经系统疾病药物提供新的思路和手段。
权利要求书clms
1.一种大肠杆菌BW25113菌株yjgW基因敲除突变株，其特征在于，所述突变株的保藏编号为CGMCC No.23332。2.根据权利要求1所述的一种大肠杆菌BW25113菌株yjgW基因敲除突变株，其特征在于，所述yjgW基因序列如SEQ ID NO.1所示。3.根据权利要求2所述的一种大肠杆菌BW25113菌株yjgW基因敲除突变株，其特征在于，所述yjgW基因的扩增引物如SEQ ID NO.2-3所示。4.如权利要求1所述的一种大肠杆菌BW25113菌株yjgW基因敲除突变株在制备治疗自闭症药物中的应用。5.根据权利要求4所述的应用，其特征在于，所述大肠杆菌BW25113菌株yjgW基因敲除突变株制备的药物用于提高秀丽隐杆线虫的聚集行为。6.根据权利要求4所述的应用，其特征在于，所述药物的有效成分为所述大肠杆菌BW25113菌株yjgW基因敲除突变株的菌液。7.根据权利要求6所述的应用，其特征在于，所述菌液的浓度为0.8-1.2×109CFU/ml。8.根据权利要求7所述的应用，其特征在于，所述菌液的制备方法如下：将所述的大肠杆菌BW25113菌株yjgW基因敲除突变株在含有25μmol/L的硫酸卡那霉素的LB液体培养基中过夜培养14-16h，测得菌液OD600为0.6时，将菌液6000rpm/min离心5min，倒掉上清，加入M9缓冲液洗涤菌体1-2次，调整菌液浓度0.8-1.2×109CFU/ml，获得菌液。
说明书desc
技术领域本发明涉及基因工程技术领域，具体涉及一种大肠杆菌BW25113菌株yjgW基因敲除突变株及其应用。背景技术肠道共生菌与人类协同进化，其遗传变异可以影响宿主的生存能力。传统研究认为，许多神经系统疾病的核心症状是由遗传突变影响了大脑的发育及功能而引起的。现在研究发现，肠道微生物群作为另一个重要的变量，也会影响特定行为的发生。目前我们对微生物区系的了解大多局限于分类学和元基因组学方面，而单个微生物基因的功能及其调控宿主特定行为发生的分子机制仍然不清楚。基于此研究背景，我们旨在研究单一微生物基因功能对宿主社会行为的影响。因此我们构建了高通量筛选平台，选择秀丽隐杆线虫作为模式生物，研究大肠杆菌及其突变菌株与秀丽隐杆线虫社会行为间的关系。秀丽隐杆线虫因繁殖周期短、价格低廉、遗传易处理性、与人类基因同源性、神经系统简单、微生物群明确、可以在单菌种环境下培养，其肠道细菌是完全可控的，在标准实验条件下以大肠杆菌为食，同时，已有研究表明其社会行为与神经系统功能有关。所以，是作为研究肠道细菌与神经系统疾病间关系的良好模式生物。在自然条件下，秀丽隐杆线虫表现出“聚集”和“不聚集”状态。已有的研究将“聚集”定义为社会行为，而“不聚集”定义为孤独行为。这种孤独行为与自闭症患者的“孤僻、不与人交流”的临床症状很相似。发明内容为解决上述技术问题，本发明提供了一种大肠杆菌BW25113菌株yjgW基因基因敲除突变株，所述突变株能够使得线虫聚集行为显著提高，能够用于制备治疗自闭症药物。本发明的第一个目的是提供一种大肠杆菌BW25113菌株yjgW基因敲除突变株，所述突变株的保藏编号为CGMCC No.23332。进一步地，所述yjgW基因序列如SEQ ID NO.1所示。进一步地，所述yjgW基因的扩增引物如SEQ ID NO.2-3所示。本发明的第二个目的是提供大肠杆菌BW25113菌株yjgW基因敲除突变株在制备治疗自闭症药物中的应用。进一步地，所述大肠杆菌BW25113菌株yjgW基因敲除突变株制备的药物用于提高秀丽隐杆线虫的聚集行为。进一步地，所述药物的有效成分为所述大肠杆菌BW25113菌株yjgW基因敲除突变株的菌液。进一步地，所述菌液的浓度为0.8-1.2×109CFU/ml。进一步地，所述菌液的制备方法如下：将所述的大肠杆菌BW25113菌株yjgW基因敲除突变株在含有25μmol/L的硫酸卡那霉素的LB液体培养基中过夜培养14-16h，测得菌液OD600为0.6时，将菌液6000rpm/min离心5min，倒掉上清，加入M9缓冲液洗涤菌体1-2次，调整菌液浓度0.8-1.2×109CFU/ml，获得菌液。与现有技术相比，本发明具有如下有益效果：1、本发明中通过敲除大肠杆菌BW25113中yjgW基因后获得基因敲除突变株K12-yjgW，将该突变株喂食秀丽隐杆线虫，发现秀丽隐杆线虫的聚集行为显著提高，因此，突变株K12-yjgW为制备治疗神经系统疾病药物提供新的思路和手段。2、鉴于秀丽隐杆线虫拥有60-80％的人类同源基因和至少42％的人类疾病相关基因，因此突变株K12-yjgW有望用于制备治疗自闭症药物。生物材料保藏信息说明K12-yjgW，已于2021年09月01日保藏于中国微生物菌种保藏管理委员会普通微生物中心，保藏编号为：CGMCC coliMG1655细胞，通过同源重组替换目标区域。随后转化pCP20质粒，表达翻转酶重组酶基因，促进FRT位点自身的同源重组，实现大肠杆菌中糖原分子结构相关基因的敲除或是外源yjgW基因的敲入。pKD46和pCP20均为温敏型质粒，通过改变环境温度可以实现质粒的清除。最后，通过对替换位点进行PCR扩增及片段测序，确定所构建的突变体的真实性和准确性，PCR扩增该片段的上下游引物如SEQ ID NO.2-3所示，产物长度为157bp，结果如图2所示，证明大肠杆菌BW25113菌株yjgW基因敲除成功，突变株命名为K12-yjgW。该大肠杆菌突变株K12-yjgW有以下特征：所述突变株K12-yjgW属于肠杆菌科、埃希菌属，为革兰氏阴性菌；兼性厌氧，呈短杆状，有菌毛，无芽胞；在液体培养基中呈浑浊生长，营养要求不高，在含25ug/ml的硫酸卡娜霉素的普通琼脂平板上37摄氏度培养24小时后，形成直径2～3mm圆形突起灰白色S型菌落；能发酵葡萄糖等多种糖类，产酸产气。实施例2：K12-yjgW的应用1、将实施例1构建的突变株K12-yjgW及野生型BW25113，在含有25umol/L的硫酸卡那霉素的LB液体培养基中过夜培养14-16h，测得菌液浓度OD600值为0.6时，将其6000rpm/min离心5min，倒掉上清，加入M9缓冲液洗涤菌体2次，调整菌液K12-yjgW浓度为2.0×109CFU/ml、1.0×109CFU/ml、2×108CFU/ml、1×108CFU/ml。2.将喂食实验室标准大肠杆菌OP50的产卵期的雌雄同体线虫进行同步化处理得到虫卵，虫卵在含少量M9缓冲液的无菌试管中过夜培养10-12。将L1期的无菌秀丽隐杆线虫分别点在含有野生型BW25113菌液以及不同浓度K12-yjgW菌液的食物盘中，每种食物盘分别点300条左右的秀丽隐杆线虫，直到L4期，再用琼脂块切割的办法，分别转移到含对应突变菌株和野生型菌株的食物盘中，每种3个食物盘，每个食物盘保证线虫的数目在80-120条左右，培养10h，在秀丽隐杆线虫到young adult期用CDC成相系统进行拍照计数。3.将两条线虫身体接触＞50％、两条线虫相交的情况计为聚集，线虫身体不接触或者接触面积小于50％的情况计为不聚集。我们分别统计每个食物盘中线虫的总数和线虫聚集的总数。用公式计算平均聚集程度：结果如图3-4所示，K12-yjgW菌液浓度为1.0×109CFU/ml时秀丽隐杆线虫的聚集程度以及发育状态最好，在此浓度下，相比于喂食野生型BW25113菌液，聚集程度显著提高。已有研究报道，秀丽隐杆线虫的聚集行为与口部的菌膜有关，我们用扫描电镜对分别喂食野生型大肠杆菌和敲除yjgW基因的突变菌株线虫进行拍摄，发现口部菌膜无明显差异，因此可以进一步说明突变菌株改变线虫聚集行为很大程度上是通过影响其神经系统功能而导致的。综上，K12-yjgW能够提高分子生物学和发育生物学研究领域的模式生物秀丽隐杆线虫的聚集行为，该实验结果为神经系统疾病的治疗提供新的研究思路。尽管已描述了本发明的优选实施例，但本领域内的技术人员一旦得知了基本创造性概念，则可对这些实施例作出另外的变更和修改。所以，所附权利要求意欲解释为包括优选实施例以及落入本发明范围的所有变更和修改。显然，本领域的技术人员可以对本发明进行各种改动和变型而不脱离本发明的精神和范围。这样，倘若本发明的这些修改和变型属于本发明权利要求及其等同技术的范围之内，则本发明也意图包含这些改动和变型在内。
