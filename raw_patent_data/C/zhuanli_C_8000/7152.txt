标题title
一种半滑舌鳎CHST12蛋白及其在抗菌中的应用
摘要abst
本发明提供一种具有抗菌功能的半滑舌鳎Chst12蛋白及其应用，所提供的CHST12蛋白，其氨基酸序列为SEQ ID NO:1。本发明再一个方面还提供所述的半滑舌鳎CHST12蛋白在制备抗菌制品中的应用；所述的抗菌制品，优选为抗哈维氏弧菌或迟缓爱德华氏菌的制品。本发明获得半滑舌鳎chst12基因，并发现半滑舌鳎chst12基因具有抗菌功能，对哈维氏弧菌和迟缓爱德华氏菌均有抗菌效应。本发明不仅为进一步了解半滑舌鳎CHST12蛋白抗菌功能奠定基础，还为抗菌药物及杀菌剂的研制提供了基因资源。
权利要求书clms
1.一种Chst12蛋白，其特征在于，所述的Chst12蛋白包含有：1)氨基酸序列为SEQ ID NO:1的蛋白；2)在1)的氨基酸序列上经过取代、缺失、添加一个或数个氨基，由1)衍生的蛋白。2.一种基因，其特征在于，所述的基因编码权利要求1所述的Chst12蛋白。3.如权利要求2所述的基因，其特征在于，所述的基因的核苷酸序列为SEQ ID NO:2。4.权利要求1所述的Chst12蛋白在制备抗菌制品中的应用。5.如权利要求4所述的应用，其特征在于，所述的抗菌制品为抗哈维氏弧菌或迟缓爱德华氏菌的制品。6.一种抗菌制品，其特征在于，所述的抗菌制品中包含有药理有效浓度的权利要求1所述的Chst12蛋白。7.如权利要求6所述的抗菌制品，其特征在于，所述的抗菌制品为抗哈维氏弧菌或迟缓爱德华氏菌的制品。8.权利要求1所述的Chst12蛋白的编码基因在筛选具有抗病原菌潜力的半滑舌鳎个体中的应用。9.一种筛选具有抗病原菌潜力的半滑舌鳎个体的方法，其特征在于，所述的方法是通过检测权利要求1所述的Chst12蛋白的含量来筛选个体。10.如权利要求9所述的方法，其特征在于，所述的方法是通过检测编码权利要求1所述的Chst12蛋白的基因的表达量来进行筛选。
说明书desc
技术领域本发明属于海洋生物技术领域，具体涉及一种半滑舌鳎CHST12蛋白及其在抗菌中的应用。背景技术半滑舌鳎是一种暖温型底栖鱼类，自然分布于我国沿海地区。由于肉质鲜美，营养丰富，深受消费者喜爱，是我国重要海水养殖经济鱼类。然而，随着半滑舌鳎养殖业的快速发展，高密度集约化养殖模式的广泛应用，以及种质退化、缺乏抗病良种等问题，致使养殖过程中病害频发，严重阻碍了半滑舌鳎养殖产业的健康可持续发展。半滑舌鳎养殖病害中，细菌性疾病尤为严重，特别是以哈维氏弧菌 为主要病原的弧菌病频繁发生，造成了巨大的经济损失。目前主要采用抗生素等药物防治弧菌病，但药物滥用不仅影响水产品质量，还会造成食品安全和环境污染等问题，给消费者带来潜在的危害。抗菌蛋白是一类具有抗菌功能的蛋白分子，在生物体内广泛存。抗菌蛋白能够抵御病原菌入侵，抑制病原菌生殖繁殖，提高机体免疫抗病能力，维护生命活动的正常运行。因此，寻找和开发能够替代传统抗生素、具有抗菌功能的蛋白分子尤为重要，已成为制备抗菌药物的一个有效途径，对于鱼类病害防控具有广阔的应用前景。硫酸软骨素磺基转移酶12属于碳水化合物磺基转移酶 家族的成员。该蛋白可催化硫酸盐转移到软骨素和脱硫的硫酸皮肤素的N-乙酰半乳糖胺的第4位残基，在主要蛋白聚糖的合成中起重要作用。CHST12在人类胶质母细胞瘤、癌症方面研究较多，研究表明，chst12基因在胶质母细胞瘤、胰腺癌、肝细胞癌、卵巢癌中表达上调，有利于肿瘤细胞增殖和迁移。因此，CHST12可能会是一种新的癌症生物标志物。但有关CHST12蛋白在抗菌作用方面的研究，迄今国际上均未见报道。因此，探究CHST12蛋白的抗菌功能是本领域亟待解决的技术难题。发明内容本发明的目的是提供一种具有抗菌功能的半滑舌鳎CHST12蛋白及其应用，从而能够解决了上述背景技术中的问题。本发明首先提供一种半滑舌鳎来源的CHST12蛋白，包含有：1)氨基酸序列为SEQ
