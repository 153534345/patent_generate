标题title
一种根瘤特异表达的启动子LjNAD1及其在根瘤特异性表达中的应用
摘要abst
本发明属于植物生物技术和基因工程领域，具体涉及一种根瘤特异表达的启动子LjNAD1及其在根瘤特异性表达中的应用。本发明核苷酸序列如SEQ ID NO.1所示的启动子LjNAD1是根瘤特异启动子，并且截短后的LjNAD1启动子在根瘤中也有活性，LjNAD1启动子及其不同截短片段均被根瘤菌诱导表达，表达水平高，并且在根瘤中特异表达，可以启动报告基因在根瘤中表达，为根瘤特异表达提供了新的有效调控元件。
权利要求书clms
1.一种根瘤特异表达的启动子LjNAD1，其特征在于，所述启动子LjNAD1的核苷酸序列如SEQ ID NO.1所示，或者为由SEQ ID NO.1所示的核苷酸序列经取代、缺失或添加至少一个核苷酸而衍生的且具有相同功能的核苷酸序列。2.根据权利要求1所述的启动子LjNAD1，其特征在于，所述启动子LjNAD1的核苷酸序列如SEQ ID NO.2、SEQ ID NO.3、SEQ ID NO.4、SEQ ID NO.5、SEQ ID NO.6、SEQ ID NO.7或SEQ ID NO.8所示。3.权利要求1或2所述启动子LjNAD1或含有权利要求1或2所述启动子LjNAD1的元件在植物根瘤特异性表达中的应用。4.根据权利要求3所述的应用，其特征在于，所述启动子LjNAD1或含有启动子LjNAD1的元件驱动外源基因在根瘤中特异性表达。5.根据权利要求3所述的应用，其特征在于，所述元件包括表达盒、表达载体或细胞。6.根据权利要求3所述的应用，其特征在于，所述植物包括豆科植物。7.根据权利要求6所述的应用，其特征在于，所述豆科植物包括百脉根、苜蓿、大豆、花生、菜豆、豇豆和鹰嘴豆。8.一种驱动外源基因在根瘤中特异性表达的方法，其特征在于，包括如下步骤：将权利要求1或2所述启动子LjNAD1或含有权利要求1或2所述启动子LjNAD1的元件和目的外源基因导入植物基因组，得到转基因植物。9.根据权利要求8所述的方法，其特征在于，所述将启动子LjNAD1或含有启动子LjNAD1的元件和目的外源基因导入植物基因组的方式包括：将所述启动子LjNAD1和目的外源基因导入载体，将得到的重组载体导入植物基因组。
