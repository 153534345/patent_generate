标题title
新真菌硫酮还原酶变体及使用其生产L-赖氨酸的方法
摘要abst
本公开涉及一种新真菌硫酮还原酶变体、包含所述变体的谷氨酸棒状杆菌菌株以及使用所述菌株生产L‑赖氨酸的方法。
权利要求书clms
1.一种真菌硫酮还原酶变体，其由SEQ ID NO:1表示的氨基酸序列组成，其中，作为对应于SEQ ID NO:3的第358位的氨基酸的脯氨酸被丝氨酸取代。2.一种多核苷酸，其编码根据权利要求1所述的变体。3.一种谷氨酸棒状杆菌菌株，其包含根据权利要求1所述的变体或编码所述变体的多核苷酸。4.根据权利要求3所述的菌株，其与包含SEQ ID NO:3的多肽或编码所述多肽的多核苷酸的谷氨酸棒状杆菌相比，L-赖氨酸生产能力增加。5.一种生产L-赖氨酸的方法，所述方法包括：在培养基中培养包含根据权利要求1所述的变体或编码所述变体的多核苷酸的谷氨酸棒状杆菌菌株的步骤。
