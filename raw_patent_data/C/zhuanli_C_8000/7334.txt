标题title
水稻株高调控基因OsSP3、及其突变基因和应用
摘要abst
本发明公开了一种水稻株高调控基因OsSP3、其突变基因及其在水稻株高改良中的应用。所述水稻株高调控基因OsSP3，其核苷酸序列如SEQ ID NO：1所示；其突变基因的核苷酸序列如SEQ ID NO：3和SEQ ID NO：4所示，该水稻株高突变基因可导致水稻植株株高变矮。可将其应用于水稻株型改良工作中，特别是针对高秆水稻品种，能够有效降低其株高，并优化其株型，对进一步提升水稻产量具有重要的意义。
权利要求书clms
1.水稻株高调控基因OsSP3，其核苷酸序列如SEQ ID NO：1所示。2.蛋白质，由权利要求1所述的水稻株高调控基因OsSP3编码，其氨基酸序列如SEQ IDNO：2所示。3.权利要求1所述水稻株高调控基因OsSP3的水稻株高突变基因，其核苷酸序列如SEQID NO：3和SEQ ID NO：4所示。4.蛋白质，由权利要求3所述的水稻株高突变基因编码，其氨基酸序列分别如SEQ IDNO：5和SEQ ID NO：6所示。5.一种载体，含有权利要求1所述的水稻株高调控基因OsSP3或权利要求3所述的水稻株高突变基因。6.权利要求1所述的水稻株高调控基因OsSP3、权利要求3所述的水稻株高突变基因或权利要求5所述的载体在水稻育种中的应用。7.权利要求3所述的水稻株高突变基因或权利要求5所述的载体在改良高秆水稻株高中的应用。8.根据权利要求7所述的应用，其特征在于，包括将所述的水稻株高突变基因或载转入所需改良的高杆水稻品种中。9.一种改良高秆水稻株高的方法，其特征在于，包括将权利要求3所述的水稻株高突变基因或权利要求5所述的载体转入所需改良的高杆水稻品种中。
说明书desc
技术领域本发明属于水稻遗传育种的领域，具体涉及水稻株高调控基因OsSP3、及其突变基因和在水稻株型改良中的应用。技术背景株型是水稻重要的农艺性状，一般会影响植株生长的诸多农艺性状，特别是对产量有着重要影响作用。收集、创制和鉴定株高相关突变体并克隆分析相关基因，将其用于新品种株型的改良，是创制新株型、提高产量的有效途径。通过挖掘与鉴定株型性状相关基因，为构建水稻理想株型，提供新的种质与遗传资源。发明内容发明目的：为解决上述技术问题，本发明提供了一种水稻株高调控基因OsSP3、及其突变基因和在水稻株型改良中的应用。技术方案：本发明提供了一种分离的水稻株高调控基因OsSP3，其核苷酸序列如SEQ ID NO：1所示。蛋白质，由所述的水稻株高调控基因OsSP3编码，其氨基酸序列如SEQ ID NO：2所示。本发明还提供了分离的水稻株高突变基因，即所述水稻株高调控基因OsSP3的突变基因，其在SEQ ID NO.1所示序列中取代、插入或缺失一个或多个核苷酸，导致其不编码SEQ ID：2所示的氨基酸序列。所述水稻株高突变基因，其核苷酸序列如SEQ ID NO：3和SEQ ID NO：4所示。蛋白质，由所述的水稻株高突变基因编码，其氨基酸序列分别如SEQ ID NO：5和SEQ ID NO：6所示。本发明还提供了一种载体，含有所述的水稻株高调控基因OsSP3或水稻株高突变基因。本发明还提供了所述的水稻株高调控基因OsSP3、所述的水稻株高突变基因或所述的载体在水稻育种中的应用。进一步的，所述的水稻株高突变基因或所述的载体在改良高秆水稻株高中的应用。利用所述水稻株高突变基因，能够优化水稻株型，改良其株高，有效降低高秆水稻品种的株高性状。具体的，包括将所述的水稻株高突变基因或载转入所需改良的高杆水稻品种中。本发明还提供了一种改良高秆水稻株高的方法，包括将所述的水稻株高突变基因或所述的载体转入所需改良的高杆水稻品种中。本发明鉴定了控制水稻株高的基因OsSP3，研究发现其突变体OsSP3-1有1个碱基发生缺失，导致移码突变，其核苷酸序列如SEQ ID NO：3所示，其编码的氨基酸序列如SEQID NO：5所示；突变体OsSP3-2有2个碱基发生插入，导致移码突变，其核苷酸序列如SEQ IDNO：4所示，其编码的氨基酸序列如SEQ ID NO：6所示。有益效果：本发明提供的水稻突变体OsSP3-1和OsSP3-2均表现为株高变矮，为水稻提供了两个矮秆的种质资源，为下一步矮化育种提供新的基因资源，有效降低高秆水稻品种的株高性状，丰富水稻遗传的多样性，在优化水稻株型方面具有广阔的应用前景。附图说明图1为突变体与野生型的株型比较；bar＝10cm；图2为OsSP3基因的定位图；图3OsSP3的组织表达分析。左野生型浙粳99取样结构图；右OsSP3在各组织中的表达水平；R，根；S，茎；L，叶；YP，幼穗；LS，叶鞘；图4OsSP3的亚细胞定位，上为空载对照，下为35S::OsSP3:GFP表达载体，GFP，绿色荧光蛋白；Nucleus，细胞核；Bright field，明场；Merged，荧光合并图像；具体实施方式实施例1：OsSP3突变体的分离和遗传分析OsSP3-1和OsSP3-2两个矮秆突变体，是常规晚粳稻品种浙粳99经甲基磺酸乙酯化学诱变而来，经田间连续种植多代，突变性状稳定遗传。与野生型株型相比，OsSP3-1和OsSP3-2株高显著降低，且分蘖角度明显增大。突变体OsSP3-1株高下降了44.1％，突变体OsSP3-2株高下降了34.7％。OsSP3-1作为母本与野生型浙粳99获得的F2遗传分离群体中，正常株型的植株为358株，矮秆植株96株。OsSP3-2作为母本与野生型浙粳99杂交获得的F2植株中，正常株型植株是312株，矮秆植株为92株。卡方检验显示，该分离符合3：1的分离比。因此，OsSP3-1和OsSP3-2突变体的表型受单个隐性核基因控制。实施例2：OsSP3基因的图位克隆为了定位克隆调控两个突变体株高的调控基因，我们构建了遗传定位群体。将两个突变体分别与籼稻品种杂交，即突变体OsSP3-1与籼稻品种明恢86杂交，突变体OsSP3-2与籼稻品种MP239杂交，形成遗传定位群体用于基因定位。经过选取突变体OsSP3-1与籼稻品种明恢86杂交F2群体中24株突变表型非常显著的单株进行混池筛选和单株验证，将该基因暂定于第2染色体M2-6与M2-8的区间之内，根据日本晴和93-11序列的差异，我们在M2-6和M2-8两个区间之间开发了新的分子标记，往两个标记中间不断缩小区间，最终将该矮秆基因定位于标记M2-6与BS-14的1.9Mb区间内。OsSP3-2与籼稻品种MP239构建的F2群体中选取1048株具有突变表型的个体，再从中选取突变表型非常显著的24株样品进行混池筛选和单株验证，发现OsSP3-2定位于M2-6和M2-8之间，根据日本晴和9311序列的差异，我们在M2-6和M2-8区间范围之间开发了新的分子标记，将OsSP3-2最终定位于M2-6和S15的1.4Mb区间内。将突变体OsSP3-1与突变体OsSP3-2的定位结果进行比对发现，OsSP3-2与OsSP3-1两个突变群体的定位区间重合。由于两个突变体的表型相似，且定位区间重和，据此推测，控制两个突变体表型的为同一个基因。随后，我们分别对两个突变体进行测序分析。发现OsSP3-1在基因LOC_Os02g25230外显子的第54个碱基处发生了碱基G的缺失，而OsSP3-2在基因LOC_Os02g25230外显子的第1049个碱基处发生了碱基AC的插入。由于两个相似表型的突变体均在该基因处发生了碱基突变，因此我们可以确定LOC_Os02g25230为我们株高调控基因的目的基因，并将该基因命名为OsSP3。OsSP3编码一个表达蛋白，具有多效性，与水稻株型的多个性状相关。两个突变体的碱基突变均会导致该基因翻译的移码突变，使翻译过程提前终止。。表1
