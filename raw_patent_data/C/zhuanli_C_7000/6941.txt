标题title
纳米抗体及其制备方法与应用
摘要abst
本发明公开了一种纳米抗体及其制备方法与应用，涉及纳米抗体技术领域。具体的，所述纳米抗体的氨基酸核苷酸序列如SEQIDNo.1～2所示。此外，本发明还公开了一种核酸分子、一种载体、一种药物组合物和一种生物制剂。实施本发明，可以辅助对于CXCL16异常高表达的炎症性疾病进行治疗，更可以针对CXCL16高表达的肿瘤进行干预治疗。
权利要求书clms
1.一种纳米抗体，其特征在于，其氨基酸核苷酸序列如SEQ ID No.1～2所示。2.一种核酸分子，其特征在于，其氨基酸核苷酸序列如SEQ ID No.1～2所示。3.一种载体，其特征在于，其含有如权利要求2所述的核酸分子。4.一种生物制剂，其特征在于，其由如权利要求1所述的纳米抗体经序列改造而得。5.一种药物组合物，其特征在于，其包括如权利要求1所述的纳米抗体。6.如权利要求1所述的纳米抗体在制备治疗炎性疾病的药物中的应用。7.如权利要求1所述的纳米抗体在制备治疗炎性肿瘤的药物中的应用。8.如权利要求1所述的纳米抗体在制备治疗冠状病毒的药物中的应用。9.如权利要求1所述的纳米抗体的制备方法，其特征在于，包括：制备CXCL16-HSA重组融合蛋白；采用CXCL16-HSA重组融合蛋白作为抗原免疫羊驼，利用免疫后羊驼的外周血淋巴细胞构建噬菌体文库；采用CXCL16-HSA重组融合蛋白作为抗原对所述噬菌体文库进行淘选，获得纳米抗体。10.如权利要求9所述的制备方法，其特征在于，所述噬菌体文库的构建方法包括：采用CXCL16-HSA重组融合蛋白作为抗原免疫羊驼，取四免或五免后的外周血淋巴细胞，分离并提取总RNA，逆转录为cDNA；以所述cDNA为模板，扩增得到VHH目的片段；将所述VHH目的片段连接到载体pcomb3xss中；将步骤得到连接产物与电穿孔感受态细胞共同电转化，即得到噬菌体文库。11.如权利要求9所述的制备方法，其特征在于，所述CXCL16-HSA重组融合蛋白的制备方法包括：构建CXCL16-HSA载体，其氨基酸核苷酸序列如SEQ ID NO.3所示；将所述载体通过转化抽提得到质粒；将所述质粒转入293F细胞，进行抗原表达，并纯化，得到CXCL16-HSA重组融合蛋白。
说明书desc
技术领域本发明涉及纳米抗体技术领域，尤其涉及一种纳米抗体的制备方法与应用。背景技术在驼科的动物体内会产生一种独特的抗体：缺失轻链的重链抗体。通常把源于这种抗体的可变区片段称为纳米抗体。纳米抗体具有分子量小、性质稳定、体内穿透性好、高亲和力和高特异性的特点。CXC趋化因子配体16是一种CXC可溶性趋化因子、粘附分子。它包含跨膜区和粘蛋白样结构，以膜结合型和分泌型两种形式存在，主要表达于抗原提呈细胞表面。CXCL16调节炎症、组织损伤和纤维化,有重要的生物学功能。发明内容本发明所要解决的技术问题在于，提供一种纳米抗体。本发明还要解决的技术问题在于，提供一种核酸分子。本发明还要解决的技术问题在于，提供一种载体。本发明还要解决的技术问题在于，提供一种药物组合物。本发明还要解决的技术问题在于，提供一种生物制剂。本发明还要解决的技术问题在于，提供一种上述的纳米抗体的应用。本发明还要解决的技术问题在于，提供一种上述的纳米抗体的制备方法。为了解决上述技术问题，本发明提供了一种纳米抗体，其氨基酸核苷酸序列如SEQID No.1～2所示。需要说明的是，本发明中纳米抗体的氨基酸核苷酸序列并不限于SEQ ID No.1～2所示的序列，其还可以是在SEQ ID No.1～2所示的序列上经过一个或多个氨基酸残基的替换和/或删除得到的，具有与SEQ ID No.1～2所示序列相同生物活性的衍生序列。相应的，本发明还公开了一种核酸分子，其氨基酸核苷酸序列如SEQ ID No.1～2所示。相应的，本发明还公开了一种载体，其含有上述的核酸分子。需要说明的是，该载体可为克隆载体，也可为表达载体，但不限于此。本发明还公开了一种生物制剂，其由上述的纳米抗体经序列改造而得。具体的，所述改造为单点或多点突变，但不限于此。相应的，本发明还公开了一种药物组合物，其包括上述的纳米抗体，还包括其他药学上可接受的载体。相应的，本发明还公开了上述的纳米抗体在制备治疗炎症性疾病中的应用。具体的，人CXCL16基因定位于染色体17p13上，编码254个氨基酸。全长的CXCL16是跨膜型CXCL16趋化因子，解整合素样金属蛋白酶10和解整合素样金属蛋白酶17对其进行蛋白水解后形成可溶性CXCL16趋化因子，又称分泌型CXCL16。CXCR6是CXCL16的唯一受体，也称孤儿受体BONZO，也是HIV的辅助受体，其基因位于第3号染色体上，由7段跨膜区序列及胞外的氨基端和胞内的羧基端组成。对CXCL16/CXCR6在人类疾病中的报道早先集中在免疫及炎症相关疾病如肝炎、免疫缺陷综合征、动脉粥样硬化等；CXCL16通过诱导和活化T细胞、巨噬细胞、炎症组织中的粒细胞、NKT细胞上的CXCR6受体促进疾病进展。因此，本发明中的纳米抗体也可应用在炎症性疾病的治疗中。相应的，本发明还公开了上述的纳米抗体在制备治疗炎症性肿瘤的药物中的应用。经研究，CXCL16/CXCR6高表达于乳腺癌、肺癌、食管癌、头颈部鳞癌、膀胱癌、结直肠癌、甲状腺癌乳腺癌、胰腺癌、前列腺癌等，CXCL16/CXCR6能促进宫颈癌、膀胱癌、卵巢癌、神经母细胞瘤和恶性黑色素瘤的侵袭转移，并参与神经胶质瘤的恶性转化。此外，研究表明：胃粘膜相关淋巴瘤向DLBCL转化过程中CXCR7、CXCR6受体表达上调。CXCL16无论是跨膜型还是分泌型的均在霍奇金淋巴瘤组织和细胞中高表达。另外CXCL16/CXCR6在NHL多种类型中高表达。通过检测临床病人的预后和分期等指标，发现CXCL16/CXCR6表达与DLBCL临床病理特征不良预后呈正相关，提示TM-CXCL16不仅可以作为一种辅助诊断标记，还可能在DLBCL的治疗方面有着极为重要的作用。在体外实验中，CXCL16外源性重组蛋白单独作用时，对DLBCL的增殖等不产生显著影响，但可以促进趋化和迁移作用。综上，CXCL16与多数炎症性肿瘤的不良预后相关，与肿瘤的恶性级别和迁徙转移相关。研究也提示它可能促进了一些炎症性肿瘤的恶性转化和转移。因此，可以针对其作为靶点，对于高表达该因子的恶性肿瘤进行干预治疗。相应的，本发明还公开了上述的纳米抗体在制备治疗冠状病毒的药物中的应用。此外，经研究，CXCL16还能与SARS-冠状病毒的N蛋白结合，这也提示本发明中的纳米抗体存在治疗冠状病毒的可能。相应的，本发明还公开了上述的纳米抗体的制备方法，其包括：制备CXCL16-HSA重组融合蛋白；采用CXCL16-HSA重组融合蛋白作为抗原免疫羊驼，利用免疫后羊驼的外周血淋巴细胞构建噬菌体文库；采用CXCL16-HSA重组融合蛋白作为抗原对所述噬菌体文库进行淘选，获得纳米抗体。优选的，所述噬菌体文库的构建方法包括：采用CXCL16-HSA重组融合蛋白作为抗原免疫羊驼，取四免或五免后的外周血淋巴细胞，分离并提取总RNA，逆转录为cDNA；以所述cDNA为模板，扩增得到VHH目的片段；将所述VHH目的片段连接到pcomb3xss中；将步骤得到连接产物与电穿孔感受态细胞共同电转化，即得到噬菌体文库。优选的，所述CXCL16-HSA重组融合蛋白的制备方法包括：构建CXCL16-HSA载体，其氨基酸核苷酸序列如SEQ
