标题title
水稻籽粒大小和灌浆的调控基因GFD2及其应用
摘要abst
本发明属于水稻育种技术领域，具体涉及一种水稻籽粒大小和灌浆的调控基因GFD2及其应用。本发明首次公开了一种水稻籽粒大小和灌浆的调控基因GFD2，其编码的蛋白质的氨基酸序列如SEQ ID No.2所示，其编码区域的核苷酸序列如SEQ ID No.1所示。该调控基因GFD2除通过影响周期蛋白表达来调控籽粒宽度外，还具备通过影响蔗糖合酶活性来调控灌浆速率的功能，在水稻分子育种中有着广泛的应用前景。

说明书desc
技术领域本发明属于水稻育种技术领域，具体涉及一种水稻籽粒大小和灌浆的调控基因GFD2及其应用。背景技术水稻是世界上最重要的主食作物之一，是世界一半以上人口的主要碳水化合物和营养来源。提高水稻产量是防止全球粮食危机的关键，而千粒重是水稻产量的关键因素之一。水稻籽粒由胚和胚乳组成，水稻的粒重由籽粒库的大小和胚乳细胞的生理生化活性共同决定。而淀粉合成是一个涉及二磷酸腺苷葡萄糖焦磷酸酶、淀粉合成酶、淀粉分支酶以及淀粉去支酶等一系列生物合成酶的复杂过程。一般来说，籽粒越大的水稻产量越高，但米质变差，特别是稻米的外观品质，比如垩白率增加。因此，高产和理想的籽粒形状是水稻育种计划的一个重要目标。晶莹透明没有垩白且细长的粒型，在世界范围内都具有高度的商业价值。如何同时调控粒型和品质是关键因素。目前的研究中已经克隆和鉴定了一部分控制籽粒形态的基因和主要数量性状位点，有相当一部分调控籽粒大小和产量的同时，会对籽粒外观产生反作用。这些克隆的基因为研究籽粒大小和粒型调节提供了相当大的参考。FUWA编码一个包含NHL结构域的表达蛋白，通过限制细胞周期蛋白进程调控籽粒发育。GW2可以泛素化WG1，并调控WG1的蛋白稳定性。WG1编码一个CC类型谷氧还蛋白OsGRX8，通过促进细胞分裂调控粒宽。qSW5/GW5编码一个参与泛素-蛋白酶体途径的多聚泛素结合蛋白，功能缺失后激活颖花外壳细胞的分裂，从而增加颖花外壳的宽度，最终谷壳的宽度、粒重以及产量都得到了增加。SPLs是植物特有的一个转录因子家族，参与多种发育过程和生理活动的调控。研究发现低活性的粳稻OsSPL12能促进粒宽而高活性的籼稻OsSPL12则抑制粒宽。粒宽基因GW5受籼稻OsSPL12的特异调控。OsSPL12与GW5共进化并协同决定水稻粒型。BG1是生长素的原初响应基因，参与调节生长素的转运，是生长素响应和转运的正调控因子，调控细胞分裂和细胞伸长，是植物特异的控制器官大小的调节因子。qGL3编码蛋白磷酸酶OsPPKL1，该酶可能参与水稻中油菜素内酯信号转导进而调控籽粒。NF-YC12协调多种途径来调控水稻胚乳发育和种子贮藏物质的积累，可以与NF-YB1互作后在糊粉层中协同调节蔗糖转运蛋白基因OsSUT1。NF-YC12和bHLH144，能够维持NF-YB1的稳定性，NF-YB1直接结合到颗粒结合型淀粉合成酶基因Wx 启动子的“G-box”域，从而调控稻米品质。此外，OsNF-YB1可能是一个细胞周期调控因子，在维持胚乳细胞的增殖中发挥作用。OsNF-YB1可以直接结合到OsYUC11和OsYUC9启动子以激活其表达，通过调控生长素合成来影响籽粒灌浆。OsNF-YB9能与水稻胚乳表达的蔗糖合酶蛋白激酶SPK相互作用，在水稻种子发育中起关键作用。TGW6编码IAA-葡萄糖水解酶，不仅直接控制胚乳的长度而且还间接参与到源到库的碳水化合物的运输。在库器官中，日本晴的tgw6等位基因能通过控制IAA供应影响合体到细胞化阶段的转变，从而限制了细胞数目和籽粒长度，而Kasalath中tgw6等位基因功能丧失会通过对源器官的多效影响增加籽粒重量从而使水稻增产。RGB1是异三聚体G蛋白的一个亚基，在调节水稻籽粒大小和粒重方面起着重要作用。抑制RGB1表达还导致籽粒中生长素含量降低，这与籽粒灌浆阶段OsNF-YB1和OsYUC11的较低表达有关。GS3是一个控制籽粒大小的主效QTL，它在调节籽粒和器官大小中发挥负调节子的功能。同时还能和DEP1与MADS直接相互作用，作为辅助因子发挥作用。调控籽粒大小和形状，同时提高水稻的产量和稻米品质。GW2编码一个环型E3泛素连接酶，负调节细胞的分裂，从而增加了粒宽和粒重，加快灌浆速率。GL7编码拟南芥LONGIFOLIA蛋白的同源蛋白，可以调控水稻粒长并改善稻米外观品质。FLO2通过影响胚乳中储藏性物质的积累在籽粒大小和淀粉品质中发挥关键的调节作用。FLO18编码一个线粒体定位的P型PPR蛋白，flo18突变导致线粒体呼吸电子传递链中复合物I的组装减少，线粒体超微结构异常，同时ATP合成受阻，胚乳中淀粉颗粒变小，同时伴随糊粉层发育异常。FLO19编码一个丙酮酸脱氢酶复合体E1-α亚基，参与造粉体发育所需的半乳糖脂的生物合成。该基因突变导致造粉体发育异常、淀粉合成受阻，严重影响水稻产量和品质。OsPK3影响水稻中丙酮酸激酶活性，调控蔗糖从叶片到籽粒的运输，进而影响籽粒灌浆。淀粉合成的相关酶的改变也会直接影响到稻米品质，例如AGPase亚基的突变会产生皱褶的胚乳，显著降低淀粉含量。GBSSI功能缺失突变体，其胚乳直链淀粉完全缺失。对OsSSIIIa进行基因敲除，发现其形成了白色的粉质胚乳且改变了淀粉的颗粒形态、结构和凝胶化温度。总的来说，颖壳可以限制籽粒的大小，成熟籽粒的大小和重量主要由颖壳的形状和大小以及颖果的灌浆程度决定，两者差异很大，受不同遗传和环境因素的控制。虽然已经在水稻中克隆了部分重要的粒型调控基因，但它们的调控机制和调控网络仍不清楚。因此，对水稻籽粒的调控基因进行研究具有重要的意义，有助于进一步理解稻米粒形的形成机制，有助于开发出高产量高品质的水稻。发明内容有鉴于此，本发明的目的在于提供一种水稻籽粒大小和灌浆的调控基因GFD2及应用。为了实现上述目的，本发明的技术方案如下：本发明提供了一种水稻籽粒大小和灌浆的调控基因GFD2，其编码的蛋白质的氨基酸序列如SEQ 。对授粉后不同时期的精米粒进行半薄切片观察，相较于野生型来说，gfd2的淀粉颗粒始终处于较为松散的状态，在DAF6时期，gfd2和野生型淀粉颗粒均比较松散；虽然随着DAF时期的增加，淀粉的密度逐渐增大，但即使在野生型几乎看不到淀粉颗粒空隙的DAF15时期，gfd2的淀粉颗粒依然有清晰可见的空隙。这进一步暗示gfd2可能在灌浆期存在某种缺陷，导致其淀粉颗粒排列松散，灌浆速率可能有所下降。测定籽粒淀粉在尿素溶液中的溶解度，将籽粒研磨成粉末后与不同浓度尿素溶液混合后，发现gfd2淀粉较难糊化，在6-9M浓度尿素中，gfd2糊化特性与野生型相比有显著差异。总之，gfd2胚乳中淀粉的理化性质与野生型有显著差异。二、GFD2的图位克隆和基因测序以及互补系验证GFD2的图位克隆和基因测序对其进行遗传分析，利用缙恢10号与突变体gfd2作为亲本构建F2群体用于基因定位，该突变体在F2代的表型分离符合3:1的比例，表明突变体受隐性单基因控制，在F2群体中分别选取正常株和突变株构建正常基因池和突变基因池，选用400对均匀分布于水稻全基因组的SSR引物对缙恢10号和突变体gfd2进行多态性分析，结果表明位于第2号染色体长臂上的两个引物RM3685和RM5460在基因池存在多态性，进一步利用F2群体中的隐性单株验证，发现两个标记与突变位点连锁，因此初步将目的基因定位在RM3685和RM5460之间。在初步定位区间内，进一步设计引物，最终将GFD2定位在RM13924和F26-4之间，物理距离为99.2kb。根据国家水稻数据中心 提供的基因注释信息，该区间共有19个开放阅读框。表S1引物序列表对其进行测序分析，在突变体中，LOC_Os02g49770的第4个外显子有一个单核苷酸 的突变，核苷酸序列如SEQ ID No.1所示。该突变体基因含有一个碱基替换，导致形成一个过早的终止密码子，进而翻译出一个缺乏C端且NHL结构域受损的包含172 个氨基酸的截断的蛋白质产物，其氨基酸序列如SEQ ID No.2所示。互补系验证将野生型的LOC_Os02g49770基因的7.7kb基因组片段转化至突变体中构建互补系，完全恢复了gfd2突变体的表型，可以证实LOC_Os02g49770的突变是造成突变体 gfd2表型的原因。表S2农艺性状统计表三、GFD2调控机制研究GFD2表达模式为确认GFD2的表达模式，采用实时荧光定量PCR分析，结果表明GFD2在所有水稻组织中都有广泛表达。包括根、茎、叶、鞘、穗，其中在根和穗的表达较高，且随着小穗的发育表达量逐渐降低。转基因株系在发育中的小穗、抽穗期籽粒颖壳、雄蕊和子房中检测到GUS活性。为确定GFD2的亚细胞定位，将GFD2的开放阅读框融合到GFP中，在CaMV35S启动子的驱动下，将质粒转化到水稻原生质体中。GFD2融合蛋白与顺式高尔基体标记物 GmMan1-mCherry特异共定位，这表明GFD2位于顺式高尔基体上。进一步分析其时空表达，通过原位RNA杂交发现，GFD2在小穗一次分生组织、二次分生组织、小花中均有强烈表达，这与QPCR的结果一致。籽粒大小调控机制GFD2编码一个包含NHL结构域的蛋白，在动物中，含NHL结构域的蛋白高度保守，并已被证明可调节细胞增殖和发育。而在植物中的研究却几乎没有。FUWA作为一个同样编码NHL结构域的蛋白，在早前的研究中被证实可以通过限制CYC家族周期蛋白来调控细胞增殖，且在水稻和单子叶植株中高度保守。然而，在 FUWA功能受到影响后，虽然粒宽同样得到增加，但是其米质却并未受到影响。虽然突变体二次枝梗数有显著降低，但由于千粒重的极显著上升，最终的单株产量并没有显著差异 。构建在水稻中包含NHL结构域的家族进化树发现，FUWA与GFD2的亲缘关系较远，这可能暗示其功能差异的原因。考虑到突变体gfd2的颖壳细胞分裂明显增强，对细胞周期蛋白表达水平进行Q-PCR分析，结果表明gfd2中的周期蛋白表达水平有极显著上升，且随着花序发育大部分周期蛋白如 cdc2Os-1,CDKB1,CDKB2,CycA1；1,CycA2；1,CycA3；1,CycB2；CycD3；1；CycT1；1；OsMCM2转录水平随着小穗发育，上升幅度会逐步增大，这与FUWA的研究结果相似。暗示 GFD2可能同样通过影响周期蛋白表达水平来调控籽粒颖壳宽度。不同的是，GFD2的千粒重没有显著差异。灌浆调控机制通过分析发现在宽度增加的颖壳中，籽粒无法饱满的填充整个颖壳。因此尽管拥有更大的籽粒，却并没有提高千粒重。这表明GFD2除了同样能影响周期蛋白的作用外，还具备调控其他途径的功能。该功能最终导致植株在灌浆期出现缺陷，使得灌浆速率下降。为了探究gfd2的灌浆缺陷，分析灌浆期包含颖壳和去除颖壳的籽粒干重，结果表明，在包含颖壳的情况下，由于颖壳宽度的增加，尽管在部分灌浆时期gfd2粒重低于野生型，其余时期与野生型无显著差异。而去除颖壳后，gfd2在全灌浆期籽粒干重始终是低于野生型的。这证实了相较野生型而言gfd2的灌浆速率降低，导致干物质的积累量不足。同时，在灌浆期gfd2烘干后的籽粒相较野生型种皮泛白严重，即使在DAF12仍未完全转绿，这暗示gfd2种皮同样存在发育迟缓。检测灌浆期和成熟后籽粒的淀粉、蔗糖和葡萄糖含量，结果表明gfd2籽粒淀粉在灌浆期和成熟期均低于野生型，而蔗糖含量则始终高于野生型，而葡萄糖含量相较野生型在灌浆期降低而在成熟后升高。葡糖糖在灌浆期的极显著下降可能预示着由葡糖糖转化而成的ADPG严重不足。这些证据表明gfd2的灌浆存在缺陷导致灌浆速率下降且淀粉合成不足。剥开灌浆期籽粒颖壳发现，野生型在DAF6时已经形成饱满的籽粒。而gfd2在DAF6时颖壳中仍有大量空隙未填充，直至DAF12才填充满颖壳。同时，在DAF9之后籽粒表面已经可以观察到凹陷皱缩的现象。测定灌浆期淀粉合成途径的相关酶活指标，相较野生型而言，GBSS并无显著差异，且AGPase 有极显著上升，SS-II活性的显著下降。这表明在灌浆速率降低的gfd2可能正在尝试通过提高AGPase活性来挽救淀粉合成的缺陷。而由于SS-II活性的显著下降，使得淀粉合成的底物在gfd2中严重不足，最终导致了灌浆速率下降。以上所述仅为本发明的较佳实施例而已，并不用以限制本发明，凡在本发明的精神和原则之内，所有的任何修改、等同替换、改进等，均应包含在本发明的保护范围之内。
