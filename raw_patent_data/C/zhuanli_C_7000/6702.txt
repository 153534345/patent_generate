标题title
大丽轮枝菌VdNRPS2基因抗病原菌靶基因片段及其干扰载体和应用
摘要abst
本发明公开了大丽轮枝菌VdNRPS2基因抗病原菌靶基因片段及其干扰载体和应用。本发明提供了大丽轮枝菌VdNRPS2基因抗病原菌靶基因片段，其核苷酸序列分别为SEQ ID No.1、SEQ ID No.3或SEQ ID No.5所示。本发明进一步提供了采用所述靶基因片段构建的Gateway干扰载体。本发明提供的大丽轮枝菌VdNRPS2基因抗病原菌靶基因片段以及采用该靶基因片段所构建的Gateway干扰载体能应用于提高作物对大丽轮枝菌所导致病害的抗性以及培育抗大丽轮枝菌的转基因植物新品种等方面。

说明书desc
技术领域本发明涉及大丽轮枝菌抗病原菌靶基因片段以及含有所述靶基因片段的RNA干扰载体，尤其涉及大丽轮枝菌VdNRPS2基因抗病原菌靶基因片段以及采用所述抗病原菌靶基因片段构建的RNA干扰载体及其在提高作物或蔬菜抗病性中的应用，属于大丽轮枝菌抗病原菌靶基因片段及抗病性应用领域。背景技术棉花黄萎病被称为“棉花癌症”，在很多国家使棉花平均减产10-35%，严重危害棉花生产，造成了巨大的经济损失。其致病菌为大丽轮枝菌，它具有很强的致病性，在整个棉花生长阶段内，它都能够进行侵染，导致棉花出现叶片萎蔫、变黄等现象；发病严重时，整个棉花叶片枯焦破碎，最终死亡。同时，它的寄主植物范围较为广泛，能够侵染的植物种类达六百多种，包括一年生草本植物、多年生草本植物和木本植物，其中不乏十字花科、茄科、菊科和蔷薇科等多种具有重要经济价值的农作物、苗木和花卉等，并且可侵染的寄主的范围还在不断扩大。由于大丽轮枝菌是土传植物病原真菌，防治困难，目前生产上还没有具有效果较好的防治药剂。丝状真菌可以通过非核糖体途径合成一系列低分子量的具有药用价值的多肽类次级代谢产物，这种结构复杂、种类繁多的多肽类化合物统称为非核糖体肽。 目前发现的非核糖体肽中超过80%来自于真菌和放线菌,其次是粘细菌和蓝藻等单细胞原核生物。基因组序列分析结果表明,古生菌、后生动物和甲藻也有合成非核糖体肽的潜力。它们具有抗菌、抗病毒和抗癌等多种生物活性，目前已经得到了广泛的应用。NRP主要是由非核糖体肽合成酶催化形成，第一个被鉴定的NRPS是L‐氨基己二酰‐L‐半胱氨酸‐D‐缬氨酸合酶，它催化β‐内酰胺类抗生素生物合成的第一步反应，将3个氨基酸缩合成一个三肽化合物进而形成青霉素等β‐内酰胺抗生素。NRPS具有多种生理功能，可作为抗生素抑制同自身进行营养物质竞争的竞争者；也可作为毒素，在病原菌侵入寄主并定殖的过程中扮演着重要的角色；还有一些可以作为铁载体参与微生物生长，还可作为储存含氮物质的场所或作为信号因子，调节微生物的生长、繁殖以及分化等。有些NRPS还与其他酶系如聚酮类或萜类合成酶系共同完成杂合分子的合成。典型的NRPS由多个模块组成，这些模块按特定的空间顺序排列,每个模块由多个功能结构域或催化单元组成,结构多样的非核糖体肽在这些模块及其结构域上有序地进行合成。近期玉米小斑病菌全基因组测序数据显示，玉米小斑病菌中编码NRPS的基因有12个，其中，NPS6作为玉米小斑病中的典型的毒力决定因子，不仅参与其致病还参与对H2O2的耐受性，是玉米病原菌的毒力决定因素，与H2O2的耐受性有关。水稻病原菌宫部旋孢腔菌，小麦赤霉病病原菌禾谷镰刀菌和拟南芥病原菌中与NPS6同源基因的缺失也会导致它们相对应的毒力下降和对H2O2的耐受性变差。通过对大丽轮枝菌的基因组序列分析发现共存在6个NRPS基因，对不同侵染时期的基因表达量进行分析发现VdNRPS2在侵染初期显著上调表达，可能与大丽轮枝菌的致病力密切相关。相应的，从VdNRPS2基因中筛选得到抗病靶基因片段并构建相应的干扰载体对于提高棉花的抗病性具有重要的应用前景。发明内容本发明的目的之一是提供大丽轮枝菌VdNRPS2基因抗病靶基因片段；本发明的目的之二是提供含有所述大丽轮枝菌VdNRPS2基因抗病靶基因片段的RNA干扰载体；本发明的目的之三是将所述大丽轮枝菌VdNRPS2基因抗病靶基因片段以及含有该靶基因片段的RNA干扰载体应用于植物抗病或构建获得抗病的转基因植物新品种。为实现上述目的，本发明所采取的主要技术方案包括：本发明首先公开了能够提高植物抗病原菌的大丽轮枝菌VdNRPS2靶基因片段，其核苷酸序列分别为SEQ ID No.1、SEQ ID No.3或SEQ ID No.5所示；优选的，所述靶基因片段的核苷酸序列为SEQ ID No.1所示。本发明还公开了含有所述的大丽轮枝菌VdNRPS2靶基因片段的RNA干扰载体以及含有所述RNA干扰载体的宿主细胞。此外，由SEQ ID No.1、SEQ ID No.3或SEQ ID No.5所示的靶基因片段所转录的dsRNA也包含在本发明的保护范围之内。本发明所述大丽轮枝菌VdNRPS2靶基因片段能够应用于提高植物对大丽轮枝菌所导致的疾病的抗病性，包括以下步骤：构建含有所述大丽轮枝菌VdNRPS2靶基因片段的RNA干扰载体；将所构建的RNA干扰载体转化到植物或植物细胞中；筛选获得对大丽轮枝菌抗病性提高的转基因植物。优选的，一种构建所述RNA干扰载体的方法，包括：通过BP反应，将所述的大丽轮枝菌VdNRPS2靶基因片段连接至pDONR207中，再通过LR反应，将其构建至pK7GWIWG2,0中得到Gateway干扰载体。本发明所述RNA干扰载体能够应用于提高植物对大丽轮枝菌所导致的疾病的抗病性，包括以下步骤：将所述RNA干扰载体转化到植物或植物细胞中；筛选获得对大丽轮枝菌抗病性提高的转基因植物。本发明中所述的大丽轮枝菌所导致的病害优选为棉花黄萎病。本发明进一步公开了一种培育抗大丽轮枝菌的转基因植物新品种的方法，包括以下步骤：构建含有所述大丽轮枝菌VdNRPS2靶基因片段的RNA干扰载体；将所构建的RNA干扰载体转化到植物或植物细胞中；筛选获得对大丽轮枝菌抗病性提高的转基因植物新品种。所述转化的方案以及将所述核苷酸引入植物的方案可适用于转化的植物或植物细胞的类型而变化。将所述核苷酸引入植物细胞的合适方法包括：显微注射、电穿孔、农杆菌介导的转化和直接基因转移等。本发明所述植物为大丽轮枝菌的寄主植物，优选为农作物或蔬菜，包括：烟草、棉花、番茄、马铃薯、甜瓜、西瓜、黄瓜或花生中的任意一种。本发明采用寄主诱导的基因沉默技术，以高致病力的大丽轮枝菌株V991为实验材料，构建多个针对大丽轮枝菌靶标基因非核糖体肽合成酶2的烟草脆裂病毒干扰质粒。通过农杆菌注射的方法转化本氏烟草，并进行大丽轮枝菌接种。将其中可以明显降低植物病情指数的靶标片段构建Gateway干扰载体，获得稳定遗传的转基因植物。通过病情指数以及分子生物学手段检测真菌生物量以及靶标基因的转录水平，筛选到效果最佳的干扰片段。本发明筛选得到的大丽轮枝菌VdNRPS2靶基因片段以及应用该靶基因片段所构建的RNA干扰载体能够应用于提高植物对大丽轮枝菌所导致疾病的抗病性以及培育抗大丽轮枝菌的转基因植物新品种。本发明整体技术方案详述本发明根据大丽轮枝菌VdNRPS2编码序列信息，设计引物，扩增获得针对靶标基因的5个不同区段，这5个靶标基因分别为VdNRPS2-1、VdNRPS2-2、VdNRPS2-3、VdNRPS2-4和VdNRPS2-5，其核苷酸序列分别为SEQ ID No.1、SEQ ID No.2、SEQ ID No.3、SEQ ID No.4和SEQ ID No.5所示。通过BamH I和EcoR I将克隆获得的靶标片段进行酶切，然后将其构建到TRV2载体中，成为VIGS系列RNAi载体。通过菌液扩增和DNA测序进行验证后，发现序列与靶标片段序列完全一致。然后，将验证正确的阳性质料转化至农杆菌GV3101，用于本氏烟草的注射。从注射后的第7天起，本氏烟草的新生嫩芽开始出现白化现象。待到第10天，新生的叶片全部为白色，并且这种叶片白化的现象可以持续45天。这表明，接种后的第7天，本氏烟草体内VIGS载体已经产生出大量dsRNA，并发挥着干扰作用。因此，选择了从注射VIGS系列载体后的第7天，进行106 cfu/mL孢子悬液的蘸根法接种。对接菌后的10 dpi、11 dpi和12 dpi进行本氏烟草的病情指数统计。结果显示，与空载相比，注射真菌靶标基因片段的本氏烟草，病情指数均有所下降；随着天数的增加，病情指数逐渐上升。其中，注射VdNRPS2-1、VdNRPS2-3、VdNRPS2-5三组靶标基因片段的烟草中，病情指数一直保持在较低水平，初步说明这三段靶标片段的引入可以有效降低植物的病情指数。本发明通过VIGS筛选的方法获得了可以提高植物对病原菌抗性的靶标区段。为了进一步验证VdNRPS2与病原菌致病性之间的关系以及干扰后能显著降低病原菌致病力的区段，获得稳定遗传的转基因本氏烟草，本实验设计了针对靶标基因的引物，引物两端带有BP位点，通过扩增获得了目的基因片段。通过BP反应和LR反应，将克隆获得的3个VdNRPS2靶标片段分别连接到Gateway干扰载体pK7GWIWG2,0上构建得到含有真菌靶标基因的植物转化载体。为了获得能够稳定遗传针对病原菌靶标基因VdNRPS2的dsRNA，通过农杆菌介导及组织培养的方法，将构建好的Gateway干扰载体转化至本氏烟草，最终获得转基因烟草。对获得的含有dsVdNRPS2的阳性转基因烟草，进行大丽轮枝菌接种，然后从接种后10 dpi、11dpi和12 dpi进行病情指数分析。根据实验结果看出，转基因烟草对病原菌的抗性明显提高，病情指数下降约50-85%。提取转基因烟草根部DNA，利用qRT-PCR进行真菌生物量分析。转基因阳性烟草的真菌生物量明显降低，约为野生型的20-40%。病情指数统计以及真菌生物量分析，可以明显观察到转基因阳性烟草对病原菌具有更强的抗性。为了进一步验证植物病情指数降低与靶标基因表达之间的关系，通过对植物根部病原菌靶标基因的表达量分析，观察到与野生型植物相比，转基因植物体内靶标基因表达量下降了约55-80%。同时，RNAi-VdNRPS2转基因烟草的抗病性明显优于野生型烟草。并且根据三个材料的病情指数、真菌量和靶标基因表达量检测，表明VdNRPS2基因的区段1作为靶标片段设计dsRNA,可以达到最佳的干扰效果，从而可以有效降低病原菌的致病力。本发明所涉及到的术语定义除非另外定义，否则本文所用的所有技术及科学术语都具有与本发明所属领域的普通技术人员通常所了解相同的含义。术语“多核苷酸”或“核苷酸”意指单股或双股形式的脱氧核糖核苷酸、脱氧核糖核苷、核糖核苷或核糖核苷酸及其聚合物。除非特定限制，否则所述术语涵盖含有天然核苷酸的已知类似物的核酸，所述类似物具有类似于参考核酸的结合特性并以类似于天然产生的核苷酸的方式进行代谢。除非另外特定限制，否则所述术语也意指寡核苷酸类似物，其包括PNA、在反义技术中所用的DNA类似物。除非另外指定，否则特定核酸序列也隐含地涵盖其保守修饰的变异体简并密码子取代）和互补序列以及明确指定的序列。特定而言，可通过产生其中一个或一个以上所选密码子的第3位经混合碱基和/或脱氧肌苷残基取代的序列来实现简并密码子取代。术语“重组宿主细胞”或“宿主细胞”意指包含本发明核苷酸的细胞，而不管使用何种方法进行插入以产生重组宿主细胞。宿主细胞可为原核细胞或真核细胞。术语“RNA干扰”意指通过外源或内源性的双链RNA在细胞内诱导同源序列的基因表达沉默的现象。附图说明图1为VIGS载体示意图。图2为VIGS不同区段扩增结果。图3为VIGS干扰载体菌液扩增验证结果；M为marker，N为阴性对照，1-4、5-8、9-12、13-16和17-20为针对VdNRPS2基因构建VIGS质粒的菌液扩增结果。图4为病情指数统计结果。图5为RNAi扩增结果；M为marker，1、2、3分别为针对VdNRPS2-1、VdNRPS2-3、VdNRPS2-5的条带。图6为RNAi载体示意图。图7为转基因阳性烟草PCR检测；M：marker，1-4、5-8、9-12为转基因阳性烟草，WT为野生型烟草。图8为转基因烟草病情指数分析结果。图9为植物体内真菌生物量分析结果。图10为植物体内真菌靶标基因表达量分析结果。具体实施方式 kDa富含半胱氨酸区域等其它元件。TRV2包含病毒的衣壳蛋白、多克隆位点等其它元件。多克隆位点的引入，方便了外源基因的插入。根据大丽轮枝菌VdNRPS2编码序列信息，设计引物，扩增获得针对靶标基因的5个不同区段，这5个靶标基因分别为VdNRPS2-1、VdNRPS2-2、VdNRPS2-3、VdNRPS2-4和VdNRPS2-5，其核苷酸序列分别为SEQ ID No.1、SEQ ID No.2、SEQ ID No.3、SEQ ID No.4和SEQ ID No.5所示。通过BamH I和EcoR I将克隆获得的靶标片段进行酶切，然后将其构建到TRV2载体中，成为VIGS系列RNAi载体。通过菌液扩增和DNA测序进行验证后，发现序列与靶标片段序列完成一致。然后，将验证正确的阳性质料转化至农杆菌GV3101，用于本氏烟草的注射。⑵本氏烟草的病情指数分析从注射后的第7天起，本氏烟草的新生嫩芽开始出现白化现象。待到第10天，新生的叶片全部为白色，并且这种叶片白化的现象可以持续45天。这表明，接种后的第7天，本氏烟草体内VIGS载体已经产生出大量dsRNA，并发挥着干扰作用。因此，选择了从注射VIGS系列载体后的第7天，进行106 cfu/mL孢子悬液的蘸根法接种。对接菌后的10 dpi、11 dpi和12 dpi进行本氏烟草的病情指数统计。结果显示，与空载相比，注射真菌靶标基因片段的本氏烟草，病情指数均有所下降；随着天数的增加，病情指数逐渐上升。其中，注射VdNRPS2-1、VdNRPS2-3、VdNRPS2-5三组靶标基因片段的烟草中，病情指数一直保持在较低水平，初步说明这三段靶标片段的引入可以有效降低植物的病情指数。⑶转基因植株的获得通过VIGS筛选的方法，获得了3个可以提高植物对病原菌抗性的靶标区段。为了进一步验证VdNRPS2与病原菌致病性之间的关系以及干扰后能显著降低病原菌致病力的区段，获得稳定遗传的转基因本氏烟草，本实验设计了针对靶标基因的引物，引物两端带有BP位点，通过扩增获得了目的基因片段。从电泳图5中可以发现明亮的目的条带。通过进一步的DNA测序、比对后发现，序列与靶标序列完全一致。通过BP反应和LR反应，将克隆获得的3个VdNRPS2靶标片段分别连接到Gateway干扰载体pK7GWIWG2,0上构建得到含有真菌靶标基因的植物转化载体。为了获得能够稳定遗传针对病原菌靶标基因VdNRPS2的dsRNA，通过农杆菌介导及组织培养的方法，将构建好的Gateway干扰载体转化至本氏烟草，最终获得转基因烟草。⑷转基因烟草的抗病性分析对获得的含有dsVdNRPS2-1、3、5基因的阳性转基因烟草，进行大丽轮枝菌接种。然后从接种后10 dpi、11 dpi和12 dpi进行病情指数分析。从图8可以看出，转基因烟草对病原菌的抗性明显提高，病情指数下降约50-85%。提取转基因烟草根部DNA，利用qRT-PCR进行真菌生物量分析。图9表明，转基因阳性烟草的真菌生物量明显降低，约为野生型的20-40%。病情指数统计以及真菌生物量分析，可以明显观察到转基因阳性烟草对病原菌具有更强的抗性。为了进一步验证植物病情指数降低与靶标基因表达之间的关系，通过对植物根部病原菌靶标基因的表达量分析，可以观察到与野生型植物相比，转基因植物体内靶标基因表达量下降了约55-80%。同时，图片显示RNAi-VdNRPS2转基因烟草的抗病性明显优于野生型烟草。并且根据三个材料的病情指数、真菌量和靶标基因表达量检测，表明VdNRPS2基因的区段1作为靶标片段设计dsRNA，可以达到最佳的干扰效果，从而可以有效降低病原菌的致病力。
