标题title
孢壁蛋白在拌饲投喂虾防控虾肝肠胞虫感染中的应用
摘要abst
本发明公开一种孢壁蛋白在拌饲投喂虾防控虾肝肠胞虫感染中的应用，属于水产养殖病害防控技术领域。本发明利用原核表达生产了一种虾肝肠胞虫的孢壁蛋白SWP12，将该孢壁蛋白按不同浓度拌入饲料投喂对虾后，发现投喂高浓度蛋白饲料的对虾，在停止投喂SWP12蛋白至少10天中，投喂含有虾肝肠胞虫的肝胰腺感染，虾也能100％抵抗该虫的感染，即使在停止投喂SWP12蛋白21天以内，虾肝肠胞虫的感染率仍显著低于不投喂或投喂低浓度SWP12蛋白的对虾，说明投喂高浓度虾肝肠胞虫孢壁蛋白SWP12能显著增强对虾抗虾肝肠胞虫感染的能力。该方法更安全更有效，更具备可操作性。

说明书desc
技术领域本发明属于水产养殖病害防控技术领域，具体涉及一种孢壁蛋白的制备及其在拌饲投喂虾防控虾肝肠胞虫感染中的应用。背景技术虾肝肠胞虫属于微孢子目、肠胞虫科、肠胞虫属，是凡纳滨对虾养殖中常见的细胞内寄生微孢子虫，主要寄生在肝胰腺细胞内，引起的疾病叫虾肝肠胞虫病。患虾肝肠胞虫病的虾主要表现为生长迟缓，其体重仅为同一批次健康虾体重的10％～40％，延长饲养上市的时间，饲料消耗比正常上市的虾更大，但产量更低，不仅多消耗饲料，还减少了一年时间内同一池塘养虾的造数，给养殖户造成严重经济损失。虾肝肠胞虫的危害范围很广，绝大部分养殖凡纳滨对虾的池塘水体和底泥中都能检测到该虫，可以通过带虫体孢子的水流、病虾、污染的饲料等多途径传播，从环境切断传播途径的防控难度极大。所以，迄今为止，没有有效的防控措施，急需从新视角开发新的有效防控技术。发明内容为了克服现有技术的缺点与不足，本发明的目的在于提供一种孢壁蛋白SWP12在拌饲投喂虾防控虾肝肠胞虫感染中的应用。本发明的目的通过下述技术方案实现：本发明提供一种孢壁蛋白SWP12在制备防控虾肝肠胞虫感染的产品中的应用；进一步的，所述的孢壁蛋白SWP12在制备拌饲投喂虾防控虾肝肠胞虫感染的产品中的应用。优选的，所述的虾为凡纳滨对虾，以下简称对虾；所述的孢壁蛋白SWP12的氨基酸序列如SEQ ID NO.1所示；编码孢壁蛋白SWP12的核苷酸序列如SEQ ID NO.2或MN604021.1所示。优选的，孢壁蛋白SWP12在产品中的添加量为60mg/kg以上，进一步为65±5mg/kg。本发明还提供一种孢壁蛋白多克隆抗体，所述多克隆抗体是利用孢壁蛋白SWP12免疫动物制备得到的；具体步骤包括：先将孢壁蛋白SWP12进行原核表达，纯化后测定蛋白浓度，然后进行动物免疫；收集免疫后的动物血清：对动物免疫2次后，采血检测，抗血清针对孢壁蛋白SWP12的效价，待效价大于等于1：51200进行最终采血收集抗血清；抗体纯化。所述的动物为新西兰兔；所述孢壁蛋白多克隆抗体在制备诊断或检测虾肝肠胞虫试剂或试剂盒中的应用。一种诊断或检测虾肝肠胞虫的试剂盒，含有上述孢壁蛋白多克隆抗体。本发明从虾肝肠胞虫基因组克隆获得一个孢壁蛋白编码区，共732bp，编码244个氨基酸，氨基酸序列中含有一个肝素结合基序。该孢壁蛋白SWP12的氨基酸序列如SEQ ID NO.1所示，利用激光共聚焦技术对该蛋白进行亚细胞定位显示SWP12主要分布在孢子壁上。该孢壁蛋白SWP12添加至虾饲料中并投喂健康对虾后，再进行EHP攻毒实验。具体包括以下步骤：S1：孢壁蛋白SWP12的原核表达和纯化原核表达孢壁蛋白SWP12，纯化后测定纯化的重组蛋白浓度。S2：制备多克隆抗体、抗体效价测定和特异性检验用纯化的SWP12按规范的免疫方法对新西兰兔免疫后，采血，通过ELISA测定多克隆抗体效价，Western blotting检测多克隆抗体特异性。S3：将SWP12蛋白拌饲投喂健康对虾一段时间、EHP攻毒对虾、检测EHP感染对虾情况。将健康对虾分为五组分别投喂普通饲料、添加BSA的饲料、添加低浓度SWP12蛋白的饲料、添加中浓度SWP12蛋白的饲料、添加高浓度SWP12蛋白的饲料，每天投喂3次，连喂14天，立即停止投喂SWP12蛋白；接着对各组虾投喂含EHP的病虾肝胰腺进行攻毒，持续投喂7天；然后，各组均改投普通饲料，持续24天，并于改投普通饲料的第3、7、14、21、24天取虾样，提取肝胰腺DNA，用Taqman探针荧光定量PCR技术检测虾样中EHP是否存在。本发明的主要机理是：虾肝肠胞虫孢子壁上的蛋白是该虫感染宿主细胞时首先与宿主细胞接触的分子，孢壁蛋白分子在感染中发挥了重要介导作用，包括识别与附着等作用；同时，这些壁蛋白也是最早引起宿主产生排异反应的分子，由它们引起的宿主免疫反应可以减少甚至消除该虫对宿主细胞的感染。所以，可以生产大量孢壁蛋白添加到饲料中投喂虾，由其激发产生免疫分子，预防虾肝肠胞虫感染虾。这是预防该寄生虫感染的新思路。基于该思路，克隆一种虾肝肠胞虫孢壁蛋白基因，通过原核表达该蛋白，将该蛋白添加到饲料中投喂凡纳滨对虾一段时间，检测该虾抗虾肝肠胞虫感染的效果。本发明相对于现有技术具有如下的优点及效果：本发明利用原核表达生产了一种虾肝肠胞虫的孢壁蛋白SWP12，将该孢壁蛋白按不同浓度拌入饲料投喂对虾后，发现投喂高浓度蛋白饲料的对虾，在停止投喂SWP12蛋白至少10天中，投喂含有虾肝肠胞虫的肝胰腺感染，虾也能100％抵抗该虫的感染，即使在停止投喂SWP12蛋白21天以内，虾肝肠胞虫的感染率仍显著低于不投喂或投喂低浓度SWP12蛋白的对虾，说明投喂高浓度虾肝肠胞虫孢壁蛋白SWP12能显著增强对虾抗虾肝肠胞虫感染的能力。本发明在投喂SWP12蛋白14天后，停喂该蛋白的间隔时间不超过10天，再连续投喂SWP12蛋白14天，如此投喂-停喂-再投喂轮流循环直到对虾上市；或者，通过不间断投喂SWP12蛋白直到对虾上市；均可以保证在此期间内完全预防虾肝肠胞虫感染，与其它消灭水环境中虾肝肠胞虫的防控方法相比，该方法更安全更有效，更具备可操作性。附图说明图1是磁珠亲和纯化的孢壁蛋白SWP12的SDS-PAGE电泳结果；其中，M：标准蛋白质分子量；泳道1：重组工程菌超声破碎后的上清液；泳道2：磁珠纯化蛋白后的冲洗液；泳道3：包涵体溶解后的上清液；泳道4-14：11个不同咪唑浓度洗脱液工作液洗脱的SWP12蛋白样品；泳道15-17：三次完全的咪唑洗脱液洗脱的SWP12蛋白样品。图2是抗孢壁蛋白SWP12的多克隆抗体特异性的Western Imidazole，100mMTris-HCl。并将pH调至8.0。3、孢壁蛋白SWP12多克隆抗体制备及效价测定和特异性检验3.1SWP12多克隆抗体的制备和效价测定取已经饲养一周的新西兰兔耳缘静脉血2mL制作阴性血清，在37℃静置1小时，待血清析出，分装至不同的离心管，置于-80℃保存。将冷冻干燥后的蛋白干粉用1M的PBS缓冲液溶解，配制成1mg/mL的浓度，作为抗原溶液。抗原溶液与等体积的弗氏完全佐剂混匀，进行第一次免疫，从耳缘静脉采集血清并测定效价。两周后进行加强免疫，抗原量为500μg/mL，从新西兰兔心脏采血，收集多克隆抗体血清，通过间接ELISA检测，发现其效价达到1：51200，分装后置于-80℃冰箱。3.2SWP12多克隆抗体的特异性检验将重组表达的SWP12蛋白经SDS-PAGE后转至PDVF膜上，并干燥，用抗重组SWP12蛋白的兔抗为一抗，通过Western blotting检验多克隆抗体的特异性，具体如下：将SWP12多克隆抗体用TBST溶液按1：5000的体积比稀释后作一抗，在37℃与膜静置孵育2小时，再用TBST溶液洗膜4次，每次5分钟，用TBS溶液漂洗1次，5分钟；用TBST溶液按1：5000的体积比对羊抗兔抗体IgG-HRP进行稀释，将二抗在37℃静置孵育2小时，同样经TBST洗膜后，加显色液于膜上，避光显色；当显色条带清晰后，用去离子水清洗膜3次以终止显色，拍照保存。结果如图2所示，发现在29kDa左右有一条清晰条带，与目的条带大小一致，说明多克隆抗体能与目的蛋白特异结合。4、孢壁蛋白SWP12在虾肝肠胞虫中分布的激光共聚焦定位将10μL纯化后的虾肝肠胞虫孢子液滴至防脱载玻片上，均匀涂布，室温下干燥10分钟。滴加300μL的5％多聚甲醛覆盖虾肝肠胞虫孢子，固定15分钟。小心舍弃上清，用移液器吸取0.1M的PBS缓冲液缓慢冲洗玻片表面三次，再缓慢滴加0.5％Triton-100覆盖、透化15分钟，改变细胞膜通透性；小心弃掉玻片表面液体，滴加1％的BSA室温封闭1小时；小心地用吸水纸吸净封闭液，分别加入用1％BSA稀释500倍的SWP12多克隆抗体和阴性对照血清，于4℃孵育过夜；弃掉玻片表面液体，吸取PBST溶液缓慢冲洗玻片三次；避光加入1％BSA稀释500倍的FITC荧光标记的山羊抗兔二抗，置于湿盒中避光孵育一小时；去掉玻片表面液体，取PBST溶液缓慢冲洗玻片三次；加入10μM的DiI，室温孵育15分钟；弃上清，再用PBST溶液缓慢冲洗玻片三次；加入5μg/mL DAPI，于常温孵育15分钟，弃上清，取PBST溶液缓慢冲洗玻片三次；待玻片稍稍干燥后，于样品区域中心滴加两滴抗荧光淬灭封片剂，盖上盖玻片，边缘涂上指甲油封片；在激光共聚焦扫描显微镜下观察。结果如图3显示，在实验组分别出现蓝色、绿色和红色荧光信号，且绿色和红色荧光分别都形成一个圈，当绿色和红色荧光重叠后，呈现出明显的黄色荧光，说明SWP12蛋白分布在虾肝肠胞虫孢子壁上。而在阴性对照组中无绿色荧光，重叠后也无黄色荧光。5、投喂孢壁蛋白SWP12的对虾抗虾肝肠胞虫感染效果5.1投喂孢壁蛋白SWP12将健康凡纳滨对虾分为空白对照组、无关蛋白组、低浓度蛋白组、中浓度蛋白组、高浓度蛋白组，每组均设置三个平行对照。每天的投喂量约为虾重的10％，每天投喂三次，连续投喂14天，立即停止投喂重组SWP12蛋白。所用的普通饲料，为江门市海众饲料有限公司生产的斑节对虾配合饲料，主要原料包括：进口鱼粉、乌贼粉、虾肉粉、高筋面粉、酵母粉、矿物质及微量元素、多种维生素、防霉剂。5.2EHP攻毒实验投喂14天后，所有组均改为投喂EHP感染的病虾肝胰腺，连续投喂7天。5.3投喂普通饲料及采样和样品处理攻毒阶段结束后，所有组均改为投喂普通饲料，并在所有组均改为投喂普通饲料的第3、第7、第14、第21、第24天共5个时间点取样每个时间点分别对每个平行各取5尾虾，解剖对虾取出肝胰腺，放于冻存管中，用液氮速冻后置于-80℃冰箱保存。5.4投喂重组SWP12蛋白的对虾抗虾肝肠胞虫感染的保护效果对投喂了14天重组SWP12蛋白的对虾，进行7天EHP的人工感染，结果如图4所示，发现停止投喂重组SWP12蛋白10天内，高浓度组的对虾100％未被EHP感染，而在停喂重组SWP12蛋白第14天以后，各组均检测到EHP感染，但高浓度组的感染率为6.67％，显著低于其他各组感染率，随着时间增加各组感染率呈现上升趋势，在第31天达到最高。BSA组、低浓度蛋白组和中浓度蛋白组的感染率与对照组均没有明显差异。结果表明，用高浓度蛋白拌饲料投喂凡纳滨对虾能增强其抗EHP感染能力，停喂重组SWP12蛋白后，可以保证在至少10天之内100％预防虾肝肠胞虫感染；在停止投喂重组SWP12蛋白21天时，虾肝肠胞虫的感染率显著低于不投喂或投喂低浓度SWP12蛋白的对虾，说明投喂高浓度虾肝肠胞虫孢壁蛋白SWP12能显著增强对虾抗虾肝肠胞虫感染的能力。上述实施例为本发明较佳的实施方式，但本发明的实施方式并不受上述实施例的限制，其他的任何未背离本发明的精神实质与原理下所作的改变、修饰、替代、组合、简化，均应为等效的置换方式，都包含在本发明的保护范围之内。序列表＜110＞
