本发明公开一种核酸内切酶及其应用，所述核酸内切酶的氨基酸序列如SEQ ID NO：1或SEQ ID NO：2所示，或所述核酸内切酶的氨基酸序列与SEQ ID NO：1或SEQ ID NO：2所示的氨基酸序列具有至少70％的序列同一性。该核酸内切酶能够直接特异性识别DNA链上的脱氧尿嘧啶，并切割位于脱氧尿嘧啶3’端的第一个磷酸二酯键，且无需与其它具有AP位点裂解酶活性的核酸内切酶共同作用即可单独完成切割，该核酸内切酶在体外分子诊断DNA的损伤突变等领域具有广阔的应用前景。此外，本发明首次提出了位于脱氧尿嘧啶3’端的第一个磷酸二酯键这一切割位点，现有技术中未见报道。

说明书desc
技术领域本发明涉及核酸内切酶领域，尤其涉及一种核酸内切酶及其应用。背景技术核酸内切酶是一种可以水解核酸分子链内部磷酸二酯键生成寡核苷酸的酶。从对底物的特异性来看，可分为DNaseI、 DNaseII等分解DNA的酶；RNase、RNaseT1等分解RNA的酶。一般的核酸内切酶大都不具备碱基特异性，其中一个特例是限制性内切酶，能够识别并切断特定碱基序列。在生物体内参与DNA损伤修复的核酸酶同样能够特异性识别出错的碱基，并水解出错的碱基或是对含有出错碱基的核酸链产生切割作用，从而引起下游一系列的酶联反应进而纠正错误。能够特异性识别受损出错碱基的核酸酶，现有典型代表的有核酸酶内切酶IV和V。核酸内切酶IV是一个脱嘌呤/嘧啶核酸内切酶，识别双链DNA中的AP位点，并切割AP位点5’端的第一个磷酸二酯键，产生3’羟基和 5’脱氧核糖磷酸末端。核酸内切酶V则主要识别DNA中的脱氧次黄嘌呤核苷，即脱氧腺苷的去氨基产物，对其下游3’端的第二个磷酸二酯键进行切割，产生一个3’羟基和5’磷酸末端，它也可识别切割AP位点、尿素、碱基错配等的DNA，但识别能力较弱。受到内在和外在的环境影响，细胞内的DNA损伤每时每刻都在发生着， DNA损伤会导致突变、癌症、甚至是细胞死亡。因此，生物体进化出了许多的DNA修复系统以消除DNA损伤。核酸内切酶就是参与DNA修复的主要分子机器。DNA脱氨是其中一种典型损伤，胞嘧啶、鸟嘌呤和腺嘌呤脱氨基后分别形成尿嘧啶、黄嘌呤和次黄嘌呤。DNA中的尿嘧啶倾向于与腺嘌呤配对，因此在DNA复制过程中，G：C碱基对将转换为A：T碱基对，从而造成基因突变。因此，在胞嘧啶被脱氨基变成尿嘧啶后，就应该立刻被发现并清除。目前研究较为深入的对该类型突变进行修复的是尿嘧啶-DNA糖基化 酶，该酶特异性识别DNA单链或双链中的尿嘧啶残基，并从DNA上水解释放游离该尿嘧啶残基，形成AP位点。含有AP位点的DNA链能被具有 AP位点裂解酶活性的核酸酶内切酶IV等识别切割，然后其它参与DNA修复的酶会进一步作用，从而完成整个碱基切除修复。因此，现有技术中一般能够特异性识别DNA中脱氧尿嘧啶的酶无法单独完成DNA的切割。因此，现有技术还有待于改进和发展。发明内容鉴于上述现有技术的不足，本发明的目的在于提供一种核酸内切酶及其应用，旨在解决现有能够特异性识别DNA中的脱氧尿嘧啶的酶无法单独完成切割该DNA的问题。本发明的技术方案如下：本发明的第一方面，提供一种核酸内切酶，其中，所述核酸内切酶的氨基酸序列如SEQ
