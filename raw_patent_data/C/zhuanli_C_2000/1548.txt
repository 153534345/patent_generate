标题title
免疫抑制分子CD24核酸适体的筛选及应用
摘要abst
本发明公开了一种免疫抑制分子CD24核酸适体的筛选及应用，本发明筛选的免疫抑制分子CD24核酸适体具有高特异性及高亲和力、高稳定性等特点，且利用该核酸适体实现了CD24蛋白特异性结合及CD24阳性肿瘤细胞系选择性识别，验证了其精准检测肿瘤来源CD24分子的可行性。
权利要求书clms
1.免疫抑制分子CD24的核酸适体，其特征在于：所述CD24核酸适体包括如SEQ ID No.1至SEQ ID No.9所示的核苷酸序列。2.根据权利要求1所述的免疫抑制分子CD24的核酸适体，其特征在于：所述CD24核酸适体还包括SEQ ID No.1至SEQ ID No.9所示的核苷酸序列的突变体或截短序列。3.根据权利要求2所述的免疫抑制分子CD24的核酸适体，其特征在于：所述CD24核酸适体突变体为SEQ ID No.1至SEQ ID No.9所示的核苷酸序列具有80％以上同源性的序列。4.根据权利要求2所述的免疫抑制分子CD24的核酸适体，其特征在于：所述CD24核酸适体突变体为SEQ ID No.1至SEQ ID No.9所示的核苷酸序列具有90％以上同源性的序列。5.权利要求1至4任意一项所述的免疫抑制分子CD24核酸适体的应用，所述应用为在肿瘤免疫疗效精准预测研究领域的应用。6.权利要求1至4任意一项所述的免疫抑制分子CD24核酸适体的应用，所述应用包括以下至少一种：1）CD24核酸适体在肿瘤细胞CD24及其糖基化的流式分析检测、成像中的应用；2）CD24核酸适体在组织切片CD24及其糖基化检测、成像中的应用；3）CD24核酸适体在肿瘤来源细胞外囊泡或外泌体CD24的流式分析检测、ELISA检测、成像中的应用；4）CD24核酸适体在肿瘤来源细胞外囊泡或外泌体CD24糖基化的流式分析检测、成像中的应用；5）CD24核酸适体在活体成像及治疗中的应用；6）CD24核酸适体在制备肿瘤靶向药物及免疫治疗药物、肿瘤免疫精准诊断、肿瘤免疫疗效精准预测及预后等研究中的应用。
说明书desc
技术领域本发明涉及肿瘤免疫疗效精准预测领域中的分子探针工具技术领域，具体是免疫抑制分子CD24核酸适体的筛选及应用。背景技术癌症已成为长期严重威胁人类生存的重大疾病。近年来，基于免疫检查点抑制剂的肿瘤免疫疗法被认为是最有潜力的癌症治疗方式，使癌症治疗领域发生了革命性变化，但并不是所有患者都能受益于免疫检查点抑制剂治疗。因为免疫抑制是一个复杂的网络，除PD-L1/PD-1及CD47/SIRPα之外，可能涉及到多条尚未发现的通路及先天免疫检查点，让肿瘤细胞不断地逃出免疫系统的追杀，这可能就是部分患者未能从免疫治疗中获益的原因。因此，免疫检查点抑制剂治疗是具有风险的，而用药之前预测免疫药物对患者是否有效很有必要。为实现快速判断是否需要免疫药物治疗，亟待发展更多的免疫检查点及其分子检测探针，实现肿瘤免疫疗效精准预测的新突破。Nature杂志上的一项研究表明，继CD47后又在肿瘤细胞表面发现一种新的免疫抑制分子CD24，该蛋白通过与Siglec-10相互作用也能向巨噬细胞发出“不要吃我”的信号，以躲避巨噬细胞的吞噬作用进而免疫逃逸，与PD-1机理类似并互补，是癌症免疫疗法很有希望的新靶标。但CD24是一种高度糖基化的糖基磷脂酰肌醇锚定表面蛋白，而CD24抗体尺寸大，易受糖基化修饰位阻影响，检测效率差；且抗体类检测探针具有易失活、不稳定、成本高的不足。发展更有效、更稳定、更低成本的检测探针，将有望为肿瘤免疫疗效精准预测领域带来更加光明的未来。核酸适体是在体外经指数富集配体系统进化技术筛选获得的小段寡核苷酸序列，与抗体类似，可特异且高效地与相应配体结合，具有诸多优点：1)分子量小、尺寸小，糖基化修饰位阻干扰小、识别效率高；2)易于筛选、合成、成本低；3)生物相容性好、无免疫原性。因此，近年来，核酸适体已成为一种具有巨大潜力的革命性分子工具，给肿瘤早期精准诊断、免疫预后及疗效精准预测带来新的希望。所以，如果获得优异性能的CD24核酸适体，就可辅助CD24抗体，为基于CD24的肿瘤免疫疗效精准预测研究带来新的希望和提供机遇。此外，CD24作为PD-L1及CD47的补充免疫检查点，CD24核酸适体在未来免疫疗法中将具有极大潜力。发明内容本发明的目的是在于提供免疫抑制分子CD24的优异性能核酸适体，并提供其在肿瘤临床医学中的应用。本发明中提供的CD24核酸适体，可精准检测CD24肿瘤细胞，将为基于CD24的癌症疗效精准预测提供革命性的新工具。所述核酸适体是通过基于琼脂糖微珠分离-流式细胞术分析的无糖基化蛋白SELEX方法筛选制备。所述核酸适体具有高特异及高亲和、高稳定性等特点。且利用该核酸适体实现了CD24蛋白特异性结合及CD24阳性肿瘤细胞系选择性识别，验证了其精准检测肿瘤来源CD24分子的可行性，将有望作为CD24的分子检测工具，应用于肿瘤免疫疗效精准预测研究领域。实现本发明目的的技术方案是：免疫抑制分子CD24的核酸适体，所述CD24核酸适体包括如SEQ
