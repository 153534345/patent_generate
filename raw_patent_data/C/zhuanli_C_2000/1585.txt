标题title
桑实杯盘菌效应蛋白Cs02526及其应用
摘要abst
本发明公开了桑实杯盘菌效应蛋白Cs02526及其应用，桑实杯盘菌效应蛋白的氨基酸序列如SEQ ID NO:1所示，其核苷酸序列如SEQ ID NO:2所示；本发明通过原核表达Cs02526，利用重组表达的Cs02526蛋白预处理叶片后，植物通过激活自身免疫反应来提高对病原菌抵抗能力，表明桑实杯盘菌效应蛋白Cs02526可作为病原菌生物防治的靶标。
权利要求书clms
1.桑实杯盘菌效应蛋白Cs02526，其特征在于：所述桑实杯盘菌效应蛋白Cs02526编码的氨基酸如SEQ ID NO.4所示。2.桑实杯盘菌效应蛋白Cs02526基因，其特征在于：所述桑实杯盘菌效应蛋白Cs02526基因的核苷酸序列如SEQ ID NO.3所示。3.含有权利要求2所述桑实杯盘菌效应蛋白Cs02526基因的重组表达载体。4.含有权利要求2所述桑实杯盘菌效应蛋白Cs02526基因的宿主。5.桑实杯盘菌效应蛋白Cs02526在提高植物防御免疫能力和/或抗病性的应用。6.根据权利要求5所述的应用，其特征在于：所述植物为烟草。7.根据权利要求5所述的应用，其特征在于：所述抗病为抗桑实杯盘菌。8.一种提高植物对病原菌抵抗能力的方法，其特征在于：通过在植物中过表达桑实杯盘菌效应蛋白Cs02526，或使用桑实杯盘菌效应蛋白Cs02526处理植物；所述桑实杯盘菌效应蛋白Cs02526编码的氨基酸如SEQ ID NO.4所示。9.根据权利要求8所述的方法，其特征在于：所述病原菌为桑实杯盘菌。10.根据权利要求8所述的方法，其特征在于：所述植物为烟草。
说明书desc
技术领域本发明涉及生物防治领域，具体涉及桑实杯盘菌效应蛋白Cs02526，还涉及桑实杯盘菌效应蛋白Cs02526的应用。背景技术植物病原菌通过多种方式来入侵宿主植物。其中包括病原菌会分泌效应蛋白进入植物细胞中，通过调控植物的生长和免疫反应从而提高自身的侵染效率。为了躲避病原菌带来的危害，植物进化出了一系列精细、高效的免疫反应。比如活性氧的产生、植物激素的合成和防御相关基因的表达等。目前对病原菌关键效应蛋白和植物防御相关基因的鉴定，是生物防治药物开发和植物抗病育种的关键所在。桑实杯盘菌是桑椹菌核病主要的病原真菌，其病害具有传播广、病原菌病病势猛，流行频率高，极易扩大感染，并有连年爆发的特点，最终造成桑椹大面积减产。目前防治该病害的主要方式是喷洒化学农药，虽在一定程度上能够降低病原菌的传播，但存在污染环境、极易产生农药耐药性、食品安全等问题。通过筛选出桑实杯盘菌侵染过程中发挥关键作用的效应蛋白，探究其在植物体内的功能，可为生物防治手段的开发及桑椹菌核病抗病育种提供分子靶标和理论基础。发明内容有鉴于此，本发明的目的之一在于提供一种桑实杯盘菌效应蛋白Cs02526；本发明的目的之二在于提供桑实杯盘菌效应蛋白Cs02526基因；本发明的目的之三在于提供含有所述桑实杯盘菌效应蛋白Cs02526基因的重组表达载体；本发明的目的之四在于提供含有所述桑实杯盘菌效应蛋白Cs02526基因的宿主；本发明的目的之五在于提供所述桑实杯盘菌效应蛋白Cs02526在提高植物防御免疫能力和/或抗病性的应用；本发明的目的之六在于提供一种提高植物对病原菌抵抗能力的方法。为达到上述目的，本发明提供如下技术方案：1、桑实杯盘菌效应蛋白Cs02526，所述桑实杯盘菌效应蛋白Cs02526编码的氨基酸如SEQ ID NO.4所示。2、所述桑实杯盘菌效应蛋白Cs02526基因的核苷酸序列如SEQ ID NO.3所示。3、含有所述桑实杯盘菌效应蛋白Cs02526基因的重组表达载体。4、含有所述桑实杯盘菌效应蛋白Cs02526基因的宿主。5、所述桑实杯盘菌效应蛋白Cs02526在提高植物防御免疫能力和/或抗病性的应用。本发明优选的，所述植物为烟草。本发明优选的，所述抗病为抗桑实杯盘菌。6、一种提高植物对病原菌抵抗能力的方法，通过在植物中过表达桑实杯盘菌效应蛋白Cs02526，或使用桑实杯盘菌效应蛋白Cs02526处理植物；所述桑实杯盘菌效应蛋白Cs02526编码的氨基酸如SEQ ID NO.2所示。本发明优选的，所述病原菌为桑实杯盘菌。本发明优选的，所述植物为烟草。本发明的有益效果在于：本发明鉴定到一个桑实杯盘菌效应蛋白Cs02526能够引起植物细胞死亡并触发植物免疫反应。通过原核表达、纯化Cs02526蛋白处理植物叶片后，可以提高植物抗病能力。该发明为后续生物农药的开发提供理论依据和分子靶标。附图说明为了使本发明的目的、技术方案和有益效果更加清楚，本发明提供如下附图进行说明：图1为Cs02526基因扩增图；图2为桑实杯盘菌效应蛋白Cs02526在烟草叶片瞬时表达的表型观察图。图3桑实杯盘菌效应蛋白Cs02526原核表达以及蛋白处理后叶片抗病力分析图。具体实施方式下面结合附图和具体实施例对本发明作进一步说明，以使本领域的技术人员可以更好的理解本发明并能予以实施，但所举实施例不作为对本发明的限定。实施例1、Cs02526编码基因的克隆与表达载体构建Cs02526编码基因的克隆使用Trizol试剂对桑实杯盘菌菌丝组织RNA进行提取，并用gDNA
