一种抗人CD15工程抗体及应用
摘要abst
本发明提供了一种抗人CD15工程抗体及应用，该工程抗体包括重链可变区与轻链可变区；所述的重链可变区的互补决定区包括CDR1、CDR2与CDR3；所述的重链可变区的CDR1的氨基酸序列如SEQ ID NO.1所示；所述的重链可变区的CDR2的氨基酸序列如SEQ ID NO.2所示；所述的重链可变区的CDR3的氨基酸序列如SEQ ID NO.3所示。本发明所述的抗人CD15工程抗体可用于霍奇金淋巴瘤的诊断、白血病的免疫分型、间皮瘤与腺瘤的鉴别诊断。

说明书desc
技术领域本发明属于医药领域，尤其是涉及一种抗人CD15工程抗体及应用。背景技术目前霍奇金淋巴瘤治疗主要以化疗为主、放疗为辅的联合治疗方案，采用控制增殖、诱导凋亡的化疗药物。这些药物在杀死白血病细胞的同时，对正常造血细胞也有严重损伤，故对机体存在严重的毒副作用。霍奇金淋巴瘤是一个独特的淋巴瘤类型，约占所有淋巴瘤的10-20％。包括两种类型的疾病：结节性淋巴细胞为主型HL和经典型HL。这两种类型HL有一些共同的特征：在病变组织中仅有少数肿瘤性大细胞-霍奇金细胞和Reed-Sternberg细胞，瘤细胞周围有大量的反应性非肿瘤细胞。放疗对于正常组织存在着迟发毒性。随着生存时间的延长，严重并发症的比例增加。在20～30％的长期生存者中，会发生放疗所致的第二肿瘤和心脏毒性。这些远期毒性不仅影响患者的生活质量，也是降低早期预后良好患者长期生存率的重要原因。因此，需要尝试使用单纯化疗来治疗早期预后良好的患者，已经开展了一些相关的对照研究来比较单纯化疗和化放联合治疗如EORTC、GELAH9F、CCG、Tata、NCIC、MSKCC等，研究结果不完全一致，但多数研究表明单纯化疗与联合治疗相比，会引起EFS或PFS的轻度下降，而对0S无显著影响。随着HL病理学、生物学及免疫学的发展，一些靶向药物也正在进行临床前期试验和临床试验。SGNd5是一个CD30单抗与人工合成的抗微管作用的化合物。该抗体-药物偶联物在细胞外液是稳定的，一旦进入肿瘤细胞，激活抗有丝分裂的机制，能靶向性地治疗两类表达CD30抗原的淋巴瘤患者，在体外实验中，CD30抗体表现出了较好的疗效。CD15作为一个抗体药物靶点，有望在后期研究中应用于霍奇金淋巴瘤的治疗。发明内容有鉴于此，本发明旨在克服现有技术中的缺陷，提出一种抗人CD15工程抗体及应用。为达到上述目的，本发明的技术方案是这样实现的：本发明提供了一种抗人CD15工程抗体，该工程抗体包括重链可变区与轻链可变区；所述的重链可变区的互补决定区包括CDR1、CDR2与CDR3；所述的重链可变区的CDR1的氨基酸序列如SEQ
