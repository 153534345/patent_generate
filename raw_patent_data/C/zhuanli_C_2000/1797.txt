标题title
FRET荧光蛋白探针及其应用
摘要abst
本发明提供了一种FRET荧光蛋白探针，包括Lssmorange、mKate2和连接多肽，该连接多肽的两端分别连接Lssmorange和mKate2，其中，所述连接多肽包括如SEQ ID NO:1～9所示的氨基酸序列。本发明还提供了一种基于上述FRET荧光蛋白探针的重组质粒及转化体。本发明还提供了一种新型冠状病毒主蛋白酶抑制剂筛选模型及方法，主要利用SARS‑COV‑2 Mpro蛋白液对上述荧光蛋白探针的水解作用，引起的荧光强度的变化来筛选新型冠状病毒主蛋白酶抑制剂。所以，上述FRET荧光蛋白探针容易识别出SARS‑COV‑2 Mpro抑制剂药物筛选中的假阳性；在细胞内可表达，可以在细胞水平对SARS‑COV‑2 Mpro抑制剂进行筛选；从而使得采用上述荧光蛋白探针的新型冠状病毒主蛋白酶抑制剂筛选模型及方法，提高了SARS‑COV‑2 Mpro抑制剂筛选的准确性。
权利要求书clms
1. 一种FRET荧光蛋白探针，其特征在于：包括Lssmorange、mKate2和连接多肽，该连接多肽的两端分别连接Lssmorange和mKate2，且所述连接多肽包括如SEQ ID NO: 1～9所示的氨基酸序列。2.根据权利要求1所述的荧光蛋白探针，其特征在于：它为Lssmorange-连结多肽-mKate2或者mKate2-连结多肽-Lssmorange。3. 根据权利要求2所述的荧光蛋白探针，其特征在于：氨基酸序列如SEQ ID NO: 10和11所示。4.一种重组质粒，特征在于：包括重组载体和克隆入该重组载体的权利要求1～3任一项所述的荧光蛋白探针的编码基因。5.一种转化体，特征在于：包括表达载体和在该表达载体中转化表达的权利要求4所述的重组质粒。6. 一种权利要求1～3任一项所述的荧光蛋白探针在筛选新型冠状病毒SARS-COV-2主蛋白酶抑制剂中的应用。7.一种新型冠状病毒主蛋白酶抑制剂筛选模型，包括：提供一反应体系，包括HEPES缓冲液、1～6 μM SARS-COV-2 Mpro蛋白溶液、1～30 μM权利要求1～3任一项所述的荧光蛋白探针、待测药物，且所述HEPES缓冲液的pH值为6～8；先将所述HEPES缓冲液、SARS-COV-2 Mpro蛋白溶液及所述待测药物加入到多孔酶标板中室温孵育，再加入所述荧光蛋白探针，形成待检测体系；利用荧光检测法检测所述待检测体系的荧光强度，获得检测数据；分析所述检测数据，获取待测药物对SARS-COV-2 Mpro的抑制活性信息，利用该抑制活性信息筛选SARS-COV-2 Mpro抑制剂。8. 根据权利要求7所述的筛选模型，其特征在于：获得检测数据的步骤包括采用多功能酶标仪或荧光酶标仪在激发波长430～440 nm、发射波长500～800 nm，对所述待检测体系进行实时检测，且每次检测前进行震荡处理，获得荧光强度检测数据。9. 根据权利要求7或8所述的筛选模型，其特征在于：所述SARS-COV-2 Mpro蛋白溶液的制备方法包括先对SARS-COV-2 Mpro基因进行PCR扩增处理，再在SARS-COV-2 Mpro基因序列5′端分别加入分子标签，获得Mpro重组质粒，其中所述分子标签为GST标签或HIS标签；将所述Mpro重组质粒转化入大肠杆菌BL21中进行表达和纯化，获得标签化Mpro蛋白浓缩液；向该标签化Mpro蛋白浓缩液中加入TEV酶进行去除标签处理，得到去标签的Mpro蛋白浓缩液，并进行分子筛柱分析；最后采用HEPES缓冲液稀释，获得浓度为1～6 μM的SARS-COV-2Mpro蛋白溶液。10.一种新型冠状病毒主蛋白酶抑制剂的筛选方法，包括步骤：先将待检测药物和SARS-COV-2 Mpro蛋白溶液进行室温孵育，再加入权利要求1～3任一项所述的荧光蛋白探针并混合均匀，形成待检测体系；然后采用荧光检测法检测所述待检测体的荧光强度，获取检测数据，对所述检测数据进行分析处理，获取所述待检测药物对SARS-COV-2 Mpro的抑制活性信息，并根据该抑制活性信息筛选SARS-COV-2 Mpro抑制剂。
说明书desc
技术领域本发明属于生物技术领域，具体涉及一种FRET荧光蛋白探针及其在筛选抗新型冠状病毒药物中应用。背景技术虽然新型冠状病毒疫苗预防重症或死亡病例的保护效果显著，但由于毒株的不断变异和进化，疫苗并不能完全阻止病毒的传播及感染。因此开发安全有效的新型冠状病毒治疗药物仍是重中之重。SARS-CoV-2属于套式病毒目冠状病毒科的亚科中的β属冠状病毒，基因序列的2/3在宿主细胞的内质网中转化成pp1ab与pp1a蛋白,主蛋白酶可切割pp1ab与pp1a蛋白产生非结构蛋白，包括RNA依赖的RNA聚合酶和解旋酶等，这些非结构蛋白负责病毒基因组的复制和转录，还在病毒的其他重要生命过程中发挥作用，如蛋白质翻译、裂解、修饰和核酸合成，其对于新型冠状病毒复制具有重要的生物学功能。目前，对于基于荧光共振能量转移技术对新型冠状病毒Mpro抑制剂的高通量筛选体系多以多肽探针为底物，无法识别由于蛋白质变性而导致得酶失活，从而产生较多的假阳性。此外，多肽探针的FRET成本较高，无法用于细胞水平药物的筛选。发明内容有鉴于此，本发明的目的之一提供一种FRET荧光蛋白探针，该探针通过柔性连接多肽连接荧光蛋白Lssmorange和mKate2，当药物使目的蛋白变性时也可使荧光蛋白变性，可以容易识别出筛选中的假阳性，提高了药物筛选的准确性，并且可以用于细胞水平的抑制剂筛选。本发明的目的之二提供一种基于上述FRET荧光蛋白探针的重组质粒和转化体，使得上述FRET荧光蛋白探针转入现有的表达载体中进行转化表达，制备容易，获得探针的成本低，克服了高通量筛选成本高的缺点。本发明的目的之三通过构建一种基于荧光蛋白探针的FRET高通量筛选模型，实现了FRET荧光蛋白探针在筛选抗新型冠状病毒药物中的应用。为实现上述目的，本发明采用以下方案：一种FRET荧光蛋白探针，包括Lssmorange、mKate2和连接多肽，该连接多肽的两端分别连接Lssmorange和mKate2，其中，所述连接多肽包括如SEQ ID NO: 1～9所示的氨基酸序列。具体地，所述连接多肽的氨基酸序列为：GGSGSGSGGSGGSG、SGSG及类似结构nKTSAVLQSGFRKMn，n为1、2、3、4、5、6、7。根据荧光蛋白基团在上述荧光蛋白探针中的排列位置不同，所述荧光蛋白探针为Lssmorange-连结多肽-mKate2或者mKate2-连结多肽-Lssmorange。一种重组质粒，包括重组载体和克隆入该重组载体的上述荧光蛋白探针的编码基因。优选地，所述重组载体为原核表达载体和真核表达载体，可以大肠杆菌、酵母菌、昆虫细胞或哺乳动物细胞等。一种转化体，包括表达载体和在该表达载体中转化表达的上述重组质粒。优选地，所述表达载体为感受态的大肠杆菌、酵母菌、昆虫细胞或哺乳动物细胞等。一种上述荧光蛋白探针在筛选新型冠状病毒主蛋白酶抑制制中的应用。优选地，所述荧光蛋白探针在筛选SARS-COV-2 Mpro抑制制中的应用。其中，本文中的“SARS-COV-2 Mpro”是指SARS-COV-2主蛋白酶。一种新型冠状病毒主蛋白酶抑制剂筛选模型，包括：提供一反应体系，包括HEPES缓冲液、1～6 μM SARS-COV-2 Mpro蛋白溶液、1～30μM权利要求1～3任一项所述的荧光蛋白探针、待测药物，且所述HEPES缓冲液的pH值为6～8；先将所述HEPES缓冲液、SARS-COV-2 Mpro蛋白溶液及所述待测药物加入到多孔酶标板中室温孵育，再加入所述荧光蛋白探针，形成待检测体系；利用荧光检测法检测所述待检测体系的荧光强度，获得检测数据；分析所述检测数据，获取待测药物对SARS-COV-2 Mpro的抑制活性信息，利用该抑制活性信息筛选SARS-COV-2 Mpro抑制剂。其中，获得检测数据的步骤包括：采用多功能酶标仪或荧光酶标仪在激发波长430～440 nm、发射波长500～800 nm，对所述待检测体系进行实时检测，且每次检测前进行震荡处理，获得荧光强度检测数据。优选地，所述激发波长约437 nm，所述发射波长分别为560～590 nm和620～640 nm，更优选地，所述发射波长约为570 nm和632 nm。所述Mpro蛋白溶液的制备方法包括：先对SARS-COV-2 Mpro基因进行PCR扩增处理，再在SARS-COV-2 Mpro基因序列5′端分别加入分子标签，获得Mpro重组质粒，其中所述分子标签为GST标签或HIS标签；将所述Mpro重组质粒转化入大肠杆菌BL21中进行表达和纯化，获得标签化Mpro蛋白浓缩液；向该标签化Mpro蛋白浓缩液中加入TEV酶进行去除标签处理，得到去标签的Mpro蛋白浓缩液，并进行分子筛柱分析；最后采用HEPES缓冲液稀释，获得浓度为2μM的Mpro蛋白溶液。一种新型冠状病毒主蛋白酶抑制剂的筛选方法，包括步骤：先将待检测药物和SARS-COV-2 Mpro蛋白溶液进行室温孵育，再加入上述荧光蛋白探针并混合均匀，形成待检测体系；然后采用荧光检测法检测所述待检测体的荧光强度，获取检测数据，对所述检测数据进行分析处理，获取所述待检测药物对SARS-COV-2 Mpro的抑制活性信息，并根据该抑制活性信息筛选SARS-COV-2 Mpro抑制剂。本发明提供的上述FRET荧光蛋白探针之所以能够用来筛选SARS-COV-2 Mpro抑制剂的原理如图1所示。上述荧光蛋白探针通过柔性连接多肽连接荧光蛋白Lssmorange和mKate2，荧光蛋白探针作为M pro水解底物，其在连接多肽处被SARS-COV-2 Mpro切，无法产生FRET；采用荧光检测法在荧光供体Lssmorange的发射波长处和上述FRET荧光蛋白探针的发射波长处连续监测这种荧光强度变化，当抑制Mpro酶活性的化合物的存在时荧光蛋白探针的荧光强度不变，如，620～640 nm处荧光强度不变；反之，化合物无法抑制Mpro酶活性时，荧光蛋白探针的荧光强度降低，如，在620～640 nm处荧光强度呈降低趋势。而且不同浓度的SARS-COV-2 Mpro对上述荧光蛋白探针的水解速度是成量效关系的。同时，由于SARS-COV-2 Mpro抑制剂的溶液基质中能够使SARS-COV-2 Mpro蛋白变性的同时也能使上述荧光蛋白探针变性，所以，本发明提供的上述FRET荧光蛋白探针容易识别出SARS-COV-2
