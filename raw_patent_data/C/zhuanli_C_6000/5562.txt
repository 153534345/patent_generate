标题title
一种葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶及其编码基因和应用
摘要abst
本发明公开了一种葡萄糖奎尼酸莽草酸家族PQQ‑依赖型膜结合脱氢酶及其编码基因和应用。本发明提供的葡萄糖奎尼酸莽草酸家族PQQ‑依赖型膜结合脱氢酶可经宿主细胞表达以生产该酶，该酶具有宽广的底物谱，能够将单糖和双糖的异头碳羟基氧化为羧基，生成相应的糖酸。该酶及表达该酶的宿主可作为生物催化剂用于制备一系列高附加值糖酸，目标产物得率高，无副产物，转化时间短，且制备方法简单、条件温和、环境友好，具有产业化前景。

说明书desc
技术领域本发明属于基因工程和酶工程技术领域，具体涉及一种葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶及其编码基因和应用。背景技术单糖中的醛糖和具有还原性的双糖都含有游离的异头碳，其异头碳上羟基可以被催化剂氧化为羧基，生成相应的糖酸。例如，葡萄糖、半乳糖和乳糖的异头碳羟基氧化后产物分别为葡萄糖酸、半乳糖酸和乳糖酸。这些糖酸都具有重要的应用价值。例如，葡萄糖酸和半乳糖酸可以选择性替代柠檬酸作为食品酸味调节剂，可以被用于甜味剂、药物中间体以及分散剂等相关产品的开发；乳糖酸可用于化妆品中的保湿剂、食品行业添加剂、医药行业的药物载体等，具有极广泛的应用领域。上述糖酸的制备方法一般包括化学法和生物法，目前以化学法为主。生物法一般以酶或含有特定酶的微生物细胞为催化剂，反应条件温和、环境友好，可以极大地避免化学法的相关缺陷。但目前相关酶的报道较少，且多数酶存在底物浓度低、转化时间长、辅因子再生困难、转化率低等缺点。NCBI数据库中有大量注释为葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶的蛋白，这些蛋白的具体功能都没有验证，关于该类酶的文献报道非常少。充分挖掘基因资源，筛选出对底物谱宽广、催化能力强的酶用于糖酸生产，是实现生物法糖酸制备工艺工业化的关键。发明内容发明目的：本发明所要解决的技术问题是针对现有技术的不足，提供一种葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶及其编码基因和应用，为生物法糖酸制备工艺提供新的酶源。为了解决上述技术问题，本发明公开了一种葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶，其为具有如SEQ ID No.2所示的氨基酸序列的蛋白质，或与SEQ ID No.2具有至少80％同源性且具有SEQ ID No.2所示的蛋白质功能的蛋白质。本发明进一步提出了编码上述葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶的核苷酸序列。优选地，编码具有如SEQ ID No.2所示的氨基酸序列的核苷酸序列如SEQ ID No.1所示。提供一种基因工程表达载体，包含前述的多核苷酸。本领域的技术人员熟知的方法能用于构建所述重组表达载体。这些方法包括重组DNA技术、DNA转化技术等。可将编码所述葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶的DNA有效连接到载体中的多克隆位点上，以指导mRNA合成进而表达葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶，或者用于同源重组。本发明的较佳案例中，所述表达载体采用pBBRMCS-2。在另一个方面，本发明提出了一种基因工程表达菌株，所述菌株含有前述重组表达载体或基因组中整合有外源的如前述的多核苷酸。所述基因工程表达菌株的出发菌株为恶臭假单胞菌或经遗传改造后的恶臭假单胞菌，其中，所述遗传改造为删除恶臭假单胞菌的基因组上的葡萄糖脱氢酶编码基因gcd。所述的恶臭假单胞菌优选KT2440。在一种实施方式中，可以先扩增所述葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶的编码基因，然后将其与质粒连接起来，将得到的重组质粒电转化至恶臭假单胞菌的感受态细胞，从而得到基因工程表达菌株。同时，还可以宿主恶臭假单胞菌进行遗传改造，删除其葡萄糖脱氢酶编码基因gcd，从而得到基因工程表达菌株。在另外一种实施方式中，也可以将葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶编码基因整合至恶臭假单胞菌基因组，代替其中的葡萄糖脱氢酶编码基因gcd，构建恶臭假单胞菌工程菌株，作为基因工程表达菌株。本发明进一步提出了上述葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶，或编码所述脱氢酶的核苷酸序列，或上述基因工程表达菌株在氧化单糖和双糖的异头碳羟基制备相应糖酸中的应用，其中，所述的单糖为醛糖，所述的双糖为具有还原性的双糖。在应用时，以单糖和双糖为底物，以基因工程表达菌株或所述葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶为催化剂，同时加入酸中和剂，进行转化反应得到相应糖酸化合物。优选地，所述酸中和剂优选碳酸钙或氢氧化钠。具体地，转化反应的条件为：反应温度为25-40℃，反应pH为5.5～7.5，反应体系中含有金属离子Mg2+或Fe3+或Ca2+0.1～2.0mM。所述温度优选30～35℃，所述pH优选6.0～6.5，所述金属离子优选Mg2+，所述Mg2+浓度优选0.5～1.0mM。有益效果：本发明中的葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶底物谱广，催化能力强，与目前文献报道的其他具有氧化单糖和双糖异头碳羟基活力的酶相似度低，具有明显的差异性，为利用基因工程技术构建具有高效糖酸生产能力的基因工程菌提供了更多的基因资源。以本发明中的基因工程菌和本发明中建立的该基因工程菌对系列单糖和双糖的生物转化工艺，可以实现高效制备一系列高附加值糖酸，目标产物得率高，无副产物，转化时间短，且制备方法简单、条件温和、环境友好，具有很好的产业化前景。具体实施方式根据下述实施例，可以更好地理解本发明。然而，本领域的技术人员容易理解，实施例所描述的内容仅用于说明本发明，而不应当也不会限制权利要求书中所详细描述的本发明。下述实施例中，所述的糖酸得率按如下公式计算：实施例1葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶编码基因的PCR扩增和基因工程菌的构建。本申请的葡萄糖奎尼酸莽草酸家族PQQ-依赖型膜结合脱氢酶编码基因筛选自莓实假单胞菌NL20W，该菌株已保藏于中国典型培养物保藏中心，保藏日期为2021年03月17日，保藏编号为CCTCC
