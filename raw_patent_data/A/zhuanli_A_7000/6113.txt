标题title
一种植物抗虫蛋白质及其应用
摘要abst
本发明公开了一种植物抗虫蛋白质，所述蛋白质氨基酸序列如SEQ ID NO:1所示。本发明改造得到的Cry1B.868‑Dv1基因所表达的杀虫蛋白相对于Cry1B.868具有更好的杀虫效果，尤其是本发明得到的Cry1B.868‑Dv1杀虫蛋白在玉米中均获得了较高的表达量和稳定性，对草地贪夜蛾具有良好抗性。
权利要求书clms
1.一种植物抗虫蛋白质，其特征在于，所述蛋白质氨基酸序列如SEQ ID NO:1所示。2.一种植物抗虫基因，其特征在于，所述抗虫基因的核苷酸序列如SEQ ID NO:2所示。3.一种重组表达载体，其特征在于，所述重组表达载体中包含权利要求1所述的抗虫基因。4.根据权利要求3所述的重组表达载体，其特征在于，所述重组表达载体依次包括如下组成元件：来自玉米泛素基因启动子、权利要求2所述的抗虫基因、胭脂碱合成酶的终止子、来自玉米泛素基因启动子、编码膦丝菌素乙酰转移酶基因；来自花椰菜花叶病毒的终止子。5.权利要求1所述植物抗虫蛋白质或权利要求2所述植物抗虫基因或权利要求3-4所述的重组表达载体的应用，其特征在于，所述应用为培育具有鳞翅目害虫抗性的转基因植物；或制备抑制或杀灭鳞翅目害虫的药剂。6.根据权利要求5所述应用，其特征在于，所述鳞翅目害虫为草地贪夜蛾。7.根据权利要求5所述应用，其特征在于，所述植物选自单子叶植物或双子叶植物。8.根据权利要求7所述应用，其特征在于，所述植物为玉米。9.一种抑制或杀灭草地贪夜蛾的方法，其特征在于，采用权利要求1所述的植物抗虫蛋白质喂食草地贪夜蛾；或将权利要求2所述的植物抗虫基因导入植物中，使草地贪夜蛾取食所述植物。
说明书desc
技术领域本申请涉及基因工程生物防治技术领域，具体而言，涉及一种一种植物抗虫蛋白质及其应用。背景技术目前，农业生产上面临的生物胁迫和非生物胁迫造成农作物生长势减弱，产量降低，给全球粮食安全造成巨大威胁。其中，虫害是影响农林生产力的主要生物胁迫因素之一。随着使用化学农药进行害虫防治造成的环境问题愈发严重，生物杀虫剂的使用逐渐进入人们的视野。苏云金芽孢杆菌是一种革兰氏阳性菌，能够产生杀虫晶体蛋白和营养期杀虫蛋白等不同类型的杀虫蛋白。其中，Cry 蛋白是在芽胞形成期于芽胞内形成的一类杀虫晶体蛋白，对大多数鳞翅目害虫具有良好抗性。在中国专利CN201980049875.1中证明Cry1B.868和Cry1Da_7基因共表达可表现出对鳞翅目害虫秋夜蛾、玉米穗蛾、西南玉米蛀虫和甘蔗蛀虫的抗性。2019年，Wang等人研究发现Cry1B.868蛋白对草地贪夜蛾具有明显的杀虫活性，但未达到高抗。目前还未见对Cry1B.868基因进行改造以提高对靶标害虫抗性的相关报道。因此，为使其具有更强的抗虫效果，需对该基因进行合理的改造，进一步提高对鳞翅目害虫的抗性。发明内容本发明提供了一种植物抗虫蛋白质，所述蛋白质氨基酸序列如SEQ ID NO:1所示。本发明还提供了一种编码所述植物抗虫蛋白质的植物抗虫基因，所述抗虫基因的核苷酸序列如SEQ ID NO:2所示。本发明还提供了一种重组表达载体，所述重组表达载体中包含前述的抗虫基因。具体的，所述重组表达载体依次包括如下组成元件：来自玉米泛素基因启动子、权利要求2所述的抗虫基因、胭脂碱合成酶的终止子、来自玉米泛素基因启动子、编码膦丝菌素乙酰转移酶基因；来自花椰菜花叶病毒的终止子。本发明还提供了一种前述植物抗虫蛋白质、植物抗虫基因或或重组表达载体的应用，所述应用为培育具有鳞翅目害虫抗性的转基因植物；或制备抑制或杀灭鳞翅目害虫的药剂。具体的，所述鳞翅目害虫为草地贪夜蛾。所述植物选自单子叶植物或双子叶植物，优选的，所述植物为玉米。本发明还提供了一种抑制或杀灭草地贪夜蛾的方法，采用前述的植物抗虫蛋白质喂食草地贪夜蛾；或将前述的植物抗虫基因导入植物中，使草地贪夜蛾取食所述植物，即可达到抑制或杀灭草地贪夜蛾的目的。本发明的有益效果包括：本发明改造得到的Cry1B.868-Dv1基因所表达的杀虫蛋白相对于Cry1B.868具有更好的杀虫效果，尤其是本发明得到的Cry1B.868-Dv1杀虫蛋白在玉米中获得了较高的表达量和稳定性，对草地贪夜蛾具有良好抗性。附图说明图1为本发明的重组克隆载体LP13-T构建流程；图2为本发明的重组表达载体LP-PT13的构建流程；图3为本发明转Cry1B.868-Dv1基因玉米植株接种草地贪夜蛾的叶片损伤图，其中CK为野生型植株，NGM为经PCR检测为非转基因的玉米植株，Cry1B.868-Dv1为转基因玉米植株。具体实施方式下面结合实施例对本发明进行进一步说明和描述，但所描述的实施例仅仅是本发明的一部分实施例，而不是全部的实施例。基于本发明和实施例中，本领域普通技术人员在没有做出创造性劳动前提下所获得的所有其他发明和实施例，都属于本发明保护的范围。下述实施例中所使用的实验方法如无特殊说明，均为常规方法。下述实施例中所用的材料、试剂等，如无特殊说明，均可从商业途径得到。实施例1、Cry1B.868-Dv1基因的获得和合成1、获得Cry1B.868-Dv1核苷酸序列本发明对Cry1B.868基因进行改造，得到Cry1B.868-Dv1基因，Cry1B.868-Dv1杀虫蛋白质的氨基酸序列，如序列表中SEQ
