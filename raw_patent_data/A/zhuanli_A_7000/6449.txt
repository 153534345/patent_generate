标题title
陆地棉GhCRP21基因及其编码蛋白和应用
摘要abst
本发明公开了陆地棉GhCRP21基因及其编码蛋白和应用。本发明公开了从陆地棉中分离的GhCRP21基因，其多核苷酸序列为SEQ ID No.1所示，其编码蛋白的氨基酸序列为SEQ ID No.2所示。本发明综合采用基因表达模式分析、亚细胞定位、病毒诱导基因沉默和转基因等分子生物学技术，以棉花品种‘R15’为试验材料，验证从陆地棉中分离克隆的GhCRP21基因的生物学功能及其在抗黄萎病方面的功能，试验结果表明陆地棉GhCRP21基因可正向调控棉花抗大丽轮枝菌反应，在提高植物对黄萎病抗性或培育抗黄萎病植物品种等方面具有应用前景。

1.从陆地棉中分离的GhCRP21基因，其特征在于，所述GhCRP21基因的CDS的多核苷酸选自-中的任何一种：SEQ ID No.1所示的多核苷酸；与SEQ ID No.1的多核苷酸序列的互补序列在严谨杂交条件能够进行杂交的多核苷酸；与SEQ ID No.1所示的多核苷酸至少有90%或以上同源性的多核苷酸；在SEQ ID No.1所示的多核苷酸的基础上进行一个或多个碱基的缺失、取代或插入的突变体，且该突变体仍具有调控植物黄萎病抗性的功能或活性。2.权利要求1所述GhCRP21基因的编码蛋白，其特征在于，其氨基酸序列为SEQ ID No.2所示。3.含有权利要求1所述GhCRP21基因的嵌合基因或表达盒。4.含有权利要求1所述GhCRP21基因的重组表达载体；或者含有权利要求3所述的嵌合基因或表达盒的重组表达载体。5.含有权利要求4所述重组表达载体的重组宿主细胞。6.权利要求1所述GhCRP21基因在调控植物抗黄萎病抗性中的应用，包括：通过调控GhCRP21基因在植物中的转录和翻译水平调控植物对于黄萎病的抗性。7.权利要求3所述的嵌合基因或表达盒在调控植物抗黄萎病抗性中的应用；或者权利要求4所述的重组表达载体在调控植物抗黄萎病抗性中的应用。8.一种提高植物对于黄萎病抗性的方法，其特征在于，包括：构建GhCRP21基因超表达重组植物表达载体；将GhCRP21基因在植物中进行超表达，所得到的转基因植物对于黄萎病抗性提高。9.一种培育抗黄萎病植物品种的方法，其特征在于，包括：构建GhCRP21基因超表达重组植物表达载体；将GhCRP21基因在植物中进行超表达，获得转基因阳性植株并得到对于黄萎病抗性提高的植物品种。10.根据权利要求8或9所述的方法，其特征在于，所述的植物是棉花。
说明书desc
技术领域本发明涉及从棉花中分离的基因，尤其涉及陆地棉GhCRP21基因及其编码蛋白和在提高植物抗黄萎病中的应用，属于陆地棉中新分离的基因及其应用领域。背景技术陆地棉属于锦葵科棉属植物，是中国重要的经济作物之一。黄萎病是由大丽轮枝菌和黑白轮枝菌等轮枝菌属引发的土传性的真菌维管束病害，研究表明中国棉花黄萎病的致病菌多为大丽轮枝菌。土壤中大丽轮枝菌的菌核在25℃~30℃的环境下复苏，接触到棉花根系后从根系伤口或幼嫩根尖处侵染并进入维管束，大丽轮枝菌在棉花维管束内繁殖产生大量孢子，依靠维管束运输、导管运输扩散至棉花各个部位致使棉花叶片萎蔫脱落，严重情况下导致整株死亡。病株残体内残留的大丽轮枝菌孢子可继续侵染其他植株或形成微菌核在土壤中，等待适宜条件继续扩散。大丽轮枝菌微菌核可在极端环境下存活80年之久，环境耐受性强，寄主范围广，对菊科、豆科、锦葵科等双子叶植物都有较强的侵染性，能够侵染600多种双子叶植物，其中农作物180余种。随着环境条件的改变及与寄主植物的长期互作过程中，黄萎病菌生理小种也在不断变异。上述黄萎病寄主范围广、存活时间长、发病生育时期多样、致死率高、病菌不断变异等诸多因素的影响，导致目前防治困难，被称为棉花的“癌症”。目前公认的大丽轮枝菌致病机制主要包括导管堵塞学说和毒素学说。导管堵塞学说认为：病菌在植物体内导管、维管束中富集大量的病菌孢子体，此时植物为阻断病菌在植物体内运输，维管束细胞分泌大量胼胝质阻塞导管，限制病菌扩散，阻塞部位因水分营养不足发生萎蔫死亡的现象。有研究发现幼苗病叶在叶片基部，中部，顶部的导管口均被菌丝、胶状物侵染和侵填体的堵塞，水分贮藏传导代谢过程和土壤养分在根茎叶中运输困难，导致了整个棉花的萎蔫。但对成苗的研究发现当成苗导管输水量远大于植株需水量的情况下，黄萎病发病情况依旧严重，说明堵塞导管学说并不是棉花黄萎病致棉花萎蔫的唯一原因。第二类为毒素学说，该学说认为黄萎病菌虽然在维管束富集，堵塞导管，但成熟棉株导管运输能力更强，在部分导管堵塞的情况下导管运输的营养物质仍足以维持棉花生长发育，此时，水分营养缺失不是棉花萎蔫的主要原因。随着导管等运输功能的增强，黄萎病菌在棉株体内快速扩散，繁殖加快，黄萎病菌繁殖过程中产生大量对棉株有害的毒性物质如毒蛋白，毒蛋白起信号传导作用致使棉花细胞膜损伤，细胞内外渗透压失衡，最终导致棉花细胞死亡，植株病变。同时病菌毒素残留在受害棉株上产生多种微生物毒素最终引起棉株严重萎蔫死亡。Taboys等认为，导管堵塞和毒素共同作用导致了棉花患病萎蔫。目前已经鉴定出许多与棉花抗黄萎病相关的基因。GbNRX1、GhMLP28、GbTLP1、GhDIR1等棉花防御相关蛋白基因表达量较高的转基因植株对大丽轮枝菌的抗性增强。沉默跨膜受体蛋白基因GbaVd1和GbaVd2后，对黄萎病的抗性降低，过表达GbaVd1和GbaVd2可显著增加维管束的木质化，增强对大丽轮枝菌的抗性。茉莉酸的生物合成和信号转导途径是植物抗大丽轮枝菌的重要组成部分。经鉴定，GhCDKE和GhCPK33可通过JA途径增强棉花对大丽轮枝菌病菌的抗性。几丁质酶和β-1,3-葡聚糖酶在抗病过程中应用最为广泛，主要是通过降解病原体细胞壁中的主要成分几丁质和葡聚糖，抑制病原体对植物的侵染，从而提高植物的抗性。因此，植物可以通过细胞壁修饰、胞外酶、模式识别受体和信号转导途径等多种机制对大丽轮枝菌产生抗性。虽然许多基因已被鉴定并证实与棉花对大众抗性有关，但对大众抗性的具体调控网络仍不清楚。转录因子作为一种多功能蛋白，可以同时参与多种信号通路的调控，包括应激信号的感知和相应基因的表达，在信号转导网络中发挥重要作用。因此挖掘棉花抗黄萎病相关基因意义重大。发明内容本发明的目的之一是从棉花中分离获得与调控黄萎病抗性的基因及其编码蛋白；本发明的目的之二是提供含有所述从棉花中分离获得与调控抗黄萎病相关的基因的表达载体或重组宿主菌；本发明的目的之三是将从棉花中分离获得的基因应用于调控植物抗黄萎病或用于培育黄萎病植物品种。为实现上述目的，本发明所采取的技术方案包括：本发明的一方面是提供了从陆地棉中分离的与调控抗黄萎病有关的GhCRP21基因，所述GhCRP21基因的CDS的核苷酸序列选自-中的任何一种：SEQ
