标题title
一种植物抗除草剂EPSPS突变型基因及其应用
摘要abst
本发明公开了一种植物抗除草剂EPSPS突变型基因及其应用。本发明还提供了植物抗除草剂EPSPS突变型蛋白、表达盒和重组载体。本申请的发明人通过实验证明，包含本发明的植物抗除草剂EPSPS突变型基因或植物抗除草剂EPSPS突变型蛋白的植物，尤其是水稻，对草甘膦除草剂具有显著的抗性，能够在该除草剂中几乎不受影响地正常生长，而相同条件下野生型植株则生长受到显著抑制，本发明的该基因突变体具有巨大的市场经济前景。

说明书desc
技术领域本发明属于基因生物学领域，具体涉及一种植物EPSPS突变型基因及其应用。背景技术杂草不仅直接造成作物产量和品质的降低，而且是多种病虫害的宿源地，间接影响作物的正常生长发育和产量形成。化学除草是控制稻田杂草的主要措施，常规除草剂因其作用机制不同而有不同的杀草谱，但是杂草稻的防治一直是个难点，主要原因是杂草稻在分类学和生理学上都与水稻十分相似，除草剂使用不当会对作物产生重大药害。草甘膦是一种有机磷类除草剂，具有内吸性，因其除草效率高、范围广以及难溶于水和土壤残留少、环境相对友好等优点，是杀草谱最广的芽后除草剂，能够对一年生及多年生杂草斩草除根。自其1974年商业化以来，很快在抗草甘膦大豆、玉米、棉花、油菜和甜菜田应用，迅速在农药市场占据一席之地成为使用最为广泛的除草剂。但同时草甘膦作为一种非选择性除草剂，在高效除草的同时也不加选择地伤害了农作物。因此，培育及种植抗草甘膦作物品种，不仅可以减轻人工除草的负担，还能够提高作物产量，有利于作物机械化生产。草甘膦主要通过干扰植物莽草酸合成途径来实现除草目标。其作用机理主要是竞争性抑制莽草酸途径中5-烯醇丙酮莽草酸-3-磷酸合成酶的活性。该合成酶是真菌、细菌、藻类、高等植物体内芳香族氨基酸生物合成过程中一个关键酶。一般情况下草甘膦能够很好的结合天然的EPSPS，研究发现个别EPSPS突变体与草甘膦的结合能力下降，能够耐受草甘膦。目前已在抗草甘膦的牛筋草中找到了一些EPSPS蛋白的变异体，但目前难以在作物体内获得天然突变的抗草甘膦的EPSPS基因，只能通过转基因的方式，将突变的EPSPS基因转入作物体内，获得抗草甘膦作物。因此，急需开发不影响植物生长发育，且对植物具有草甘膦抗性的基因，对水稻等作物生产具有重要意义。发明内容本发明所要解决的技术问题是提供可以赋予植物抗草甘膦的抗性的植物突变型基因。为此，本发明的发明人对大量水稻品种进行了研究并尝试不同的突变实验，最终在日本晴水稻的一个突变植株中获得了能够赋予水稻草甘膦抗性的基因。具体而言，本发明提供一种植物抗除草剂EPSPS突变型基因，所述植物抗除草剂EPSPS突变型基因为野生型EPSPS基因发生以下位点突变：所述植物抗除草剂EPSPS突变型基因第467位核苷酸产生了C-T的突变。优选地，本发明的植物EPSPS突变型基因存在的突变类型为：第467位C突变成T、导致了OsEPSPS基因第156位氨基酸由丙氨酸变成了缬氨酸。本发明的EPSPS突变型基因发生突变的核苷酸，在上述位点突变成其他核苷酸的情况也在本发明的保护范围之内，例如：C突变为A、T、G。优选地，EPSPS突变型基因的核苷酸序列包含序列表中SEQ ID NO:1中所示序列。SEQ ID NO:1本发明还提供了一种植物抗除草剂EPSPS突变型蛋白，所述植物抗除草剂EPSPS突变型蛋白为野生型EPSPS蛋白的氨基酸序列第156位氨基酸由丙氨酸突变为缬氨酸。EPSPS突变型蛋白如序列表中SEQ ID NO.2所示.SEQ ID NO.2本发明还提供一种表达盒，所述表达盒包含所述的植物抗除草剂EPSPS突变型基因或所述的植物抗除草剂EPSPS突变型蛋白。本发明还提供一种重组载体，所述重组载体包含所述的EPSPS突变型基因或所述的植物抗除草剂EPSPS突变型蛋白。本发明所述的植物表达盒、重组载体用于提升目标植物的抗除草剂性能。本发明还提供获得具有除草剂抗性的植物的方法，包括如下步骤：使植物包含所述的植物抗除草剂EPSPS突变型基因或所述的植物抗除草剂EPSPS突变型蛋白。进一步地，所述获得具有除草剂抗性的植物的方法为利用CRISPR介导的引导编辑工具载体使目标植物包含所述的植物抗除草剂EPSPS突变型基因或所述的植物抗除草剂EPSPS突变型蛋白。本发明还提供了获得具有除草剂抗性的植物的方法。进一步地，本发明提供了用于增强植物、植物组织、植物细胞对于至少一种除草剂的耐受性的方法，所述除草剂为具有干扰EPSPS酶的活性的除草剂。因此，本发明中除草剂抗性指的是抗草甘膦类除草剂。本发明所述的增强植物、植物组织或者植物细胞对除草剂的抗性方法可以通过植物的遗传转化或者后代杂交、自交和无性繁殖的方法实施，也可通过基因定点突变的方式获得，使改变的植株包括本发明的核苷酸序列SEQ ID No:1。本发明还提供了耐受除草剂的植物，其表达含有抗除草剂突变位点的EPSPS基因，该基因序列不同于野生型植物的EPSPS基因序列，是EPSPS基因含有抗除草剂突变位点的序列。发现携带有EPSPS突变位点的植物可以抗草甘膦除草剂，而野生型植物对草甘膦类除草剂表现敏感。本发明还提供了所述突变后的核酸或基因及蛋白在植物育种中的应用，用于培育具有除草剂抗性的植物，尤其是农作物，还提供了这些蛋白及其编码基因在转基因或者非转基因如水稻等植物中的应用。进一步地，所述应用包括：将水稻种子去壳、灭菌后将胚分离出来，置于愈伤组织诱导培养基上以产生次级愈伤组织；将次级愈伤组织转移至新的愈伤组织诱导培养基预培养；将步骤中获得的愈伤组织与农杆菌接触15分钟获得抗性愈伤，该农杆菌中转入了具有植物抗除草剂EPSPS突变型基因的重组表达载体；将步骤的愈伤组织转移到其上垫有三张无菌滤纸的培养皿中，21-23℃培养48小时；将步骤的愈伤组织置于前筛选培养基上培养5-7天；将步骤的愈伤组织转移筛选培养基上，以获得抗性愈伤组织；将抗性愈伤组织转移到分化再生培养基中分化成苗；和将步骤的苗转移到生根培养基中生根。有益效果通过实验证明，包含本发明的植物抗除草剂EPSPS突变型基因或植物抗除草剂EPSPS突变型蛋白的植物，尤其是水稻，对草甘膦除草剂具有显著的抗性，能够在该除草剂中几乎不受影响地正常生长，而相同条件下野生型植株则生长受到显著抑制。附图说明图1所示为在含除草剂培养基上产生的抗性愈伤和抗性芽。图2为野生型和突变体在突变位点附近的测序图对比。图3所示为在含除草剂的培养液中进行生长时野生型与EPSPS突变株系的生长情况对比。图4为在不含除草剂的培养液中进行生长时野生型与在含有除草剂的培养液中生长时EPSPS突变株系的生长情况对比。具体实施方式以下结合附图及其实施例对本发明进行详细说明，但并不因此将本发明的保护范围限制在实施例描述的范围之中。下面结合实施例及附图对本发明作进一步详细的描述，但本发明的实施方式不限于此。下述实施例中的实验方法，如无特殊说明，均为常规方法。下述实施例中所用的药材原料、试剂材料等，如无特殊说明，均为市售购买产品。实施例1、植物表达载体的构建和阳性转基因植株获得以水稻EPSPS基因的核苷酸序列gaggacggggtgctcctCcgggg位点，作为打靶位点。设计相应靶标的SgRNA、RT和PBS序列，得到的水稻EPSPS的pegRNA序列为：使EPSPS核苷酸序列的第467位C突变成T。此外在靶点的PBS序列和evopreQ1序列之间，添加8bp linker。因此，得到的水稻EPSPS的pegRNA-evopreQ1序列为：分别合成pegRNA-evopreQ1的正向寡核苷酸链和可与之互补的反向寡核苷酸链，退火形成双链。利用已有的引导编辑系统pHUC411-enpPE2，用BsaI内切酶在37℃酶切4小时，65℃失活酶切体系10分钟，作为构建重组载体的骨架片段。用Goldgate法将pHUC411-enpPE2和水稻的EPSPS的pegRNA-evopreQ1首尾相连，转入大肠杆菌中。通过选择具有卡那霉素抗性且不具壮观霉素抗性的菌斑，获得阳性转化子。经测序验证后，提取阳性质粒，构成用于植物OsEPSPS基因的重组载体质粒，命名为pHUC-enpPE-AtV。利用冻融法将植物表达载体转入根癌农杆菌EHA105，经菌落PCR筛选获得阳性克隆，获得含有pHUC-enpPE-AtV的农杆菌。成熟水稻种子去掉颖壳后，用70％酒精浸泡种子1min，倒掉酒精。用含有1滴Tween20的50％次氯酸钠溶液浸泡种子40min。倒掉次氯酸钠，无菌水洗5遍至溶液澄清，无次氯酸钠味道。无菌水浸泡种子过夜。用解剖刀对种子沿糊粉层将胚剥下，将胚接种于愈伤诱导培养基上。30℃下暗培养11天后将愈伤与胚乳及胚芽分离，将去芽的状态良好、分裂旺盛的初级愈伤组织进行预培养3～5天后用于农杆菌转化。采用上述转入了重组表达载体的根癌农杆菌进行农杆菌介导的遗传转化，该遗传转化、转化子筛选及转基因植株再生等参照段永波等提出的方法，在培养基上加入800mg/L的草甘膦异丙胺盐除草剂，获得抗性愈伤，如图1所示进一步再生获得抗性植株。实施例2：EPSPS基因突变植株的草甘膦抗性分析将获得的草甘膦抗性的突变植株选取叶片并提取基因组DNA。采用引物对：5’-GAGGGCACAACAGTGGTGGACA-3’和5’-AATGATTTCGATCTCCACATCC-3’对EPSPS进行扩增测序。测序结果与野生型日本晴EPSPS基因比较，发现在EPSPS基因的目标位点发生了纯合的碱基突变，第467位核苷酸产生了C-T的突变、导致了EPSPS基因第156位氨基酸由丙氨酸变成了缬氨酸，测序结果如图2所示。发明人将野生型日本晴水稻、本发明获得突变植株的种子消毒后，转接至含有800mg/L的草甘膦异丙胺盐除草剂的1/2MS培养基上生长12天，以在正常的1/2MS生长为对照。如图3所示，在含有草甘膦的培养基上，野生型日本晴水稻均已死亡，EPSPS基因突变植株的生长正常，并未明显受到抑制。进而可以证明，本申请发明人所找到的EPSPS基因突变后会使水稻具有草甘膦除草剂抗性，且不影响水稻植株生长，具有重要的生产应用价值。为了进一步验证本发明突变体基因对水稻生长情况的影响，申请人进行了进一步对比。采用上述相同的方法，在含有除草剂的培养基中培育EPSPS突变植株，在不含有除草剂的培养基中培养野生型植株，如图4所示，左侧为野生型植株的生长情况，右侧为含有本发明的突变基因的植株在含有除草剂的培养基中生长相同天数的生长情况，从对比可以看出，即便在除草剂环境中，含有本发明突变基因的植株的生长完全不受影响，与野生型在不含除草剂环境中的生长状况基本相同，可以证明本发明的突变基因是一种优良的，对水稻生长不会带来负面影响的突变基因，具有良好的经济、社会以及研究价值。虽然上面结合本发明的优选实施例对本发明的原理进行了详细的描述，本领域技术人员应该理解，上述实施例仅仅是对本发明的示意性实现方式的解释，并非对本发明包含范围的限定。实施例中的细节并不构成对本发明范围的限制，在不背离本发明的精神和范围的情况下，任何基于本发明技术方案的等效变换、简单替换等显而易见的改变，均落在本发明保护范围之内。
