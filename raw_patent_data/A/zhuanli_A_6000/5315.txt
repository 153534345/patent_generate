标题title
一种用于治疗肥胖相关脂肪肝的蛋白质
摘要abst
本发明公开了一种用于治疗肥胖相关脂肪肝的蛋白质，其氨基酸序列为SEQ ID NO:1，该蛋白质及其过表达重组腺病毒能够用于制备治疗肥胖相关脂肪肝的药物，具有开发应用前景。
权利要求书clms
1.一种用于治疗肥胖相关脂肪肝的蛋白质，其氨基酸序列为SEQ ID NO:1。2.如权利要求1所述蛋白质的编码基因，其核苷酸序列为SEQ ID NO:2。3.用于表达如权利要求1所述蛋白质的重组腺病毒。4.如权利要求1所述的蛋白质、其编码基因或者包含该蛋白质编码基因的重组腺病毒在制备肥胖相关脂肪肝治疗药物中的应用。5.如权利要求4所述的应用，其特征在于，所述药物为药物组合物，含有蛋白质SEQ IDNO:1、多核苷酸SEQ ID NO:2或者用于表达该蛋白质SEQ ID NO:1的重组腺病毒、以及药学上可接受的载体。6.如权利要求5所述的应用，其特征在于，当蛋白质SEQ ID NO:1为药物的活性成分时，药物组合物中还含有蛋白质稳定剂。7.如权利要求5所述的应用，其特征在于，当多核苷酸SEQ ID NO:2或者包含该多核苷酸SEQ ID NO:2的重组腺病毒为药物的活性成分时，药物组合物中还含有核酸稳定剂。8.如权利要求4所述的应用，其特征在于，所述药物为药物组合物，除了药物活性成分蛋白质SEQ ID NO:1、多核苷酸SEQ ID NO:2或者用于表达该蛋白质SEQ ID NO:1的重组腺病毒外，还包含至少一种辅助治疗剂。9.如权利要求4所述的应用，其特征在于，所述肥胖相关脂肪肝是非酒精性脂肪性肝病。10.如权利要求4所述的应用，其特征在于，所述药物的剂型是冻干剂或者注射剂。
