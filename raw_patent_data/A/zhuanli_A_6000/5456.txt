标题title
一种来自水稻的OsMYB44基因及其应用
摘要abst
本发明公开了一种来自水稻的OsMYB44基因及其应用，OsMYB44基因核苷酸序列如SEQ ID No.2所示，编码的蛋白的氨基酸序列如SEQ ID No.3所示。本发明克隆的OsMYB44片段，可有效提高水稻及其它农作物中色胺及衍生物的含量，增强农作物对UV‑B等胁迫的耐受性。
权利要求书clms
1.来自水稻的OsMYB44基因，其核苷酸序列如SEQ ID No.2所示。2.权利要求1所述的OsMYB44基因编码的蛋白，其氨基酸序列如SEQ ID No.3所示。3.一种表达载体，表达权利要求2所述的蛋白。4.根据权利要求3所述的表达载体，其特征在于，含有SEQ ID No.2所示的核酸片段。5.权利要求1所述的OsMYB44基因在提高植物UV-B耐受性或色胺及其衍生物积累上的应用。6.根据权利要求5所述的应用，其特征在于，所述应用是指在植物中超量表达OsMYB44基因，从而提高植物UV-B耐受性或色胺及其衍生物积累。7.根据权利要求5或6所述的应用，其特征在于，所述植物为水稻。
说明书desc
技术领域本发明涉及植物技术工程领域。具体涉及一个参与水稻色胺积累的MYB类型的转录因子OsMYB44克隆、功能验证和应用研究。背景技术紫外线-B作为一种典型的氧化胁迫对植物造成了严重危害，从DNA、信号肽和蛋白质等分子响应到细胞改变，最终导致叶片失绿、锈斑、卷曲、干枯、甚至死亡等形态和生理的变化，严重影响植物的生长和发育。近年来，由于快速而又持续的气候改变，大气层中臭氧层不断破坏。UV-B逐渐成为影响全球农业生产的最具挑战性的威胁之一。在UV-B对农作物产量影响日益严重的背景下，研究农作物对UV-B的响应、调控和耐受机制，是农作物的稳产和增产的重要举措。水稻作为一种重要的粮食作物，在保障我国粮食危机和安全的重大战略需求中具有举足轻重的作用。因此，研究水稻对UV-B的响应、调控以及耐受，对UV-B胁迫下水稻稳产，选育UV-B耐受的水稻品种促进水稻增产，具有重要意义。色胺类物质是色氨酸代谢途径衍生而来的一类重要的次生代谢物，具有较强的抗氧化活性，能够直接、间接清除活性氧功能，从而抑制细胞死亡或凋亡的发生，保护细胞内的脂质、蛋白质、核酸等参与植物的生长、发育、生物和非生物胁迫。研究表明，UV-B在多个物种中显著诱导色胺合成基因和色胺类物质的积累。然而，在植物中响应UV-B的色胺代谢调控网络却鲜有报道，色胺类物质响应和耐受UV-B的遗传基础仍然不是很清楚。水稻作为一种重要农作物，在水稻中解析色胺响应和耐受UV-B的机制，在当前全球臭氧层不断破坏的背景下，为水稻稳产、选育UV-B耐受水稻品种促进水稻增产提供理论支撑。发明内容本发明所要解决的技术问题为：如何提供一种参与水稻色胺积累的MYB类型的转录因子。本发明的技术方案为：一种来自水稻的OsMYB44基因，其核苷酸序列如SEQ No.2所示的核酸片段。上述所述的OsMYB44基因在提高植物UV-B耐受性上的应用。进一步地，所述应用是指在植物中超量表达OsMYB44基因，从而提高植物UV-B耐受性。上述所述的OsMYB44基因在提高植物色胺及其衍生物积累上的应用。进一步地，所述应用是指在植物中超量表达OsMYB44基因，从而提高植物色胺及其衍生物积累。进一步地，所述植物为水稻。申请人通过UV-B胁迫下水稻转录组数据分析，筛选到一个MYB类型的转录因子，与拟南芥的AtMYB44同源，命名为OsMYB44。OsMYB44全长基因组的核苷酸序列为1950bp，编码区核苷酸序列为960bp，编码319个蛋白质的氨基酸序列，该基因不含内含子。以水稻品种ZH11的cDNA为模板，扩增出OsMYB44基因的全长编码区，构建OsMYB44相关载体，通过酵母单杂、烟草瞬时表达、遗传材料中基因表达量、色胺含量以及抗UV-B胁迫的表型分析，验证OsMYB44调控色胺的积累以及UV-B胁迫的耐受性。水稻中超量表达OsMYB44显著增强了色胺及色胺衍生物的积累，并显著增强UV-B的耐受性。通过对OsMYB44的开发和利用，可有效提高水稻及其它农作物中色胺含量，增强农作物对UV-B等非生物胁迫的耐受性。与现有技术相比，本发明具有以下有益效果：本发明克隆的OsMYB44片段，可有效提高水稻及其它农作物中色胺及衍生物的含量，增强农作物对UV-B等胁迫的耐受性。附图说明图1：UV-B诱导OsMYB44表达水平。标准差基于三个生物学重复，每个样品为3株材料的混合样。**代表**P＜0.01。图2：OsMYB44的亚细胞定位。利用35S:OsMYB44-GFP与35S:OsGhd7-RFP作为细胞核marker载体共转化于烟草叶片中，标尺为20μm。图3：遗传转化超量表达OsMYB44载体的结构示意图。图4：CRISPR/Cas9介导OsMYB44的突变。图5：OsMYB44水稻遗传材料中OMYB44和色胺合成基因表达量。野生型水稻植株为ZH11；OsMYB44超量表达植株为OE-1和OE-2；OsMYB44突变体植株CR-1和CR-2。标准差基于三个生物学重复，每个样品为3株材料的混合样。图6：OsMYB44水稻遗传材料中的色胺及衍生物的含量。野生型水稻植株为ZH11；超量表达植株为OE-1和OE-2；OsMYB44突变体植株CR-1和CR-2。图7：OsMYB44水稻遗传材料耐受UV-B的表型分析。野生型水稻植株ZH11；超量表达植株为OE-1和OE-2；OsMYB44突变体植株CR-1和CR-2。图8：UV-B处理后OsMYB44水稻遗传材料的存活率和地上部分鲜重。野生型水稻植株为ZH11；超量表达植株为OE-1和OE-2；OsMYB44突变体植株CR-1和CR-2。标准差基于三个生物学重复，每个样品为3株材料的混合样。*和**分别代表*P＜0.05和**P＜0.01。具体实施方式下述实施例中的实验方法，如无特殊说明，均为常规方法。下述实施例中所用的试验材料，如无特殊说明，均为从商业渠道购买得到的。本发明人通过UV-B胁迫处理水稻幼苗叶片的转录组数据分析和筛选，从水稻中成功克隆了参与色胺积累并增强UV-B耐受性的MYB类型的转录因子OsMYB44。实施例1、OsMYB44基因的获得1、OsMYB44基因确定和结构分析UV-B胁迫处理水稻幼苗后，位于水稻9号染色体一个MYB转录因子的表达量明显受到上调。进一步通过定量PCR检测显示，该转录因子在UV-B处理1h、3h和12h均受到UV-B诱导。其中，在UV-B处理1h时，该转录因子诱导最为显著。在水稻基因组数据库TIGR中，这个MYB转录因子与拟南芥AtMYB44同源，其编号是LOC_Os09g01960，命名为OsMYB44。OsMYB44全长基因组序列为1950bp，编码序列为960bp，编码319个氨基酸，该基因没有内含子。2、OsMYB44基因的扩增申请人根据OsMYB44基因在粳稻“日本晴”基因组中的位置和结构，以OsMYB44基因两侧的基因组序列为模板，设计了两条PCR引物即：OsMYB44-F和OsMYB44-R。为了获得OsMYB44基因全长cDNA序列，申请人提取水稻品种中花11的总RNA，反转录成cDNA。PCR扩增目的片段与带有通用接头attB1和attB2载体的pDONR207发生BP反应，得到的入门载体的克隆中。将反应产物转化大肠杆菌DH5a，加1mL LB复苏1h，取150μL复苏的菌液涂于含抗性LA培养基的平板上，37℃温箱培养12h。单克隆进行PCR检测，提取质粒测序。对PCR产物进行测序分析，获得OsMYB44基因的cDNA序列，从ZH11中扩增得到960bp的OsMYB44基因序列。序列比对正确后，再分别与目的载体进行LR反应。实施例2、OsMYB44亚细胞定位分析1、OsMYB44亚细胞定位瞬时表达载体的构建从ZH11中扩增得到960bp的OsMYB44基因片段通过与目的载体进行LR反应。将反应产物转化大肠杆菌DH5a，加1mL LB复苏1h，取150μL复苏的菌液涂于含抗性LA培养基的平板上，37℃温箱培养12h。单克隆进行PCR检测，提取质粒即为目的载体。2、烟草瞬时转化种植本氏烟草。在14h光照/10h黑暗，温度25℃，相对湿度70％条件下培养大概4-5周。将含有目地基因的载体转入农杆菌EHA105中。挑取含有目的载体的农杆菌单克隆至含有相应抗生素的5mL LB培养基中，在28℃，200rpm的条件下培养2d。转移1mL培养的农杆菌菌液至20mL含有相应抗生素的LB培养基中扩大培养。在28℃，200rpm的条件下，培养至农杆菌生长到对数期。在室温，5000rpm离心10min以收集菌体，用浸染液悬浮农杆菌菌体至OD600＝1.0，室温静止2-3h。用1mL的注射器拔去掉针头吸取含有目的质粒的菌液，轻轻地注射到叶片中。同时，记号笔标记烟草叶片水渍状区域。注射好的植株放在21℃左右在黑暗条件下培养1d，再放在光照条件下培养1d，即可在激光共聚焦显微镜下观察GFP的表达情况。具体结果表明，OsMYB44与OsGhd7共同定位到烟草叶片的细胞核内。实施例3、OsMYB44的遗传材料对UV-B的耐受性分析1、OsMYB44遗传转化载体的构建和遗传转化从ZH11中扩增得到960bp的OsMYB44基因片段通过与目的载体pJC034，进行LR反应，获得目的载体，载体图谱见图3。该载体携带具有组成型和超量表达特征的玉米泛素基因启动子的农杆菌介导的遗传转化载体。同时，通过CRISPR/Cas9介导的基因突变，构建OsMYB44的突变体载体，具体突变位点见图4。2、OsMYB44遗传转化通过农杆菌EHA105介导和水稻遗传转化体系，将正确克隆的质粒导入到水稻品种ZH11中，经过预培养、侵染、共培养、筛选具有潮霉素抗性的愈伤组织、再进行分化、生根、炼苗和移栽等步骤，得到转基因植株。3、OsMYB44遗传材料中色胺基因表达量和色胺含量分析首先，对T0代转基因水稻植株DNA水平的检测，确认构建OsMYB44的阳性植株。对生长至4周大的超量表达OsMYB44和敲除突变OsMYB44的T1代阳性水稻植株和野生型水稻植株进行强度为12.8μW cm-2UV-B胁迫处理。当UV-B处理12h时，取UV-B处理和未处理的这些植株水稻叶片的RNA样和代谢样，分析植株中色胺合成基因的表达量和色胺类物质的含量。结果显示，与对照野生型ZH11相比，在超量表达OsMYB44阳性转化植株体内的OsMYB44基因的表达量获得大幅度提高，在突变OsMYB44阳性转化植株体内的OsMYB44基因的表达量并没有明显差异，说明构建的遗传材料是没有问题的，可以用于进一步的研究分析。进一步定量PCR检测显示，在OsMYB44超量表达的水稻植株中色胺合成途径关键基因OsTSα和OsTDC1的表达量明显受到上调，并且UV-B处理后OsTSα和OsTDC1的表达量上调更加显著；在OsMYB44突变的水稻植株中UV-B处理后OsTSα和OsTDC1的表达量相对于野生型明显受到下调，这些结果暗示OsMYB44参与水稻中色胺的调控依赖于UV-B。进一步色胺及色胺衍生物含量分析表明，tryptophan、tryptamine、N-Benzoyltryptamine、N-Acetyltryptamine、N-p-Coumaroyltryptamine和N-Cinnamoyltryptamine在OsMYB44超量表达的水稻植株中显著受到上调；tryptophan、N-Benzoyltryptamine、N-Acetyltryptamine、N-p-Coumaroyltryptamine和N-Cinnamoyltryptamine在OsMYB44突变的水稻植株中显著受到下调。4、OsMYB44遗传材料对UV-B的耐受性分析同时，结合前期的实验条件，对生长至4周大的超量表达OsMYB44和敲除突变OsMYB44的T1代阳性水稻植株和野生型水稻植株进行强度为12.8μW cm-2UV-B胁迫处理。当UV-B处理72h时，移至正常生长的光照培养箱中，观察OsMYB44遗传材料对UV-B的耐受性。待正常生长至10d后，统计水稻植株的存活率和地上部分鲜重。结果表明，超量表达OsMYB44水稻植株展现出对UV-B的耐受性；敲除突变OsMYB44水稻植株展现出对UV-B的敏感性。而且，UV-B处理后超量表达OsMYB44水稻植株存活率和地上部分鲜重明显高于野生型植株；敲除突变OsMYB44水稻植株存活率和地上部分鲜重明显低于野生型植株。
