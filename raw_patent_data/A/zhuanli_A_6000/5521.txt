标题title
增加番茄根毛长度的基因及应用
摘要abst
本发明公开了一种增加番茄根毛长度的基因及应用，属于作物分子遗传学技术领域。所述基因的核苷酸序列如SEQ ID NO.1所示。本发明通过一系列实验，发现借助转基因技术在“Ailsa Craig”生态型番茄中过量表达Solyc04g077960后，显著增加番茄根毛的长度，有效提高番茄对土壤的固着能力，明显改善番茄在营养物质贫瘠条件下的生长状况。本发明建立的方法为培育番茄新品种的育种工作提供了重要的分子标记，为提高植物生物学产量新品种的创制提供了重要参考。

1.基因Solyc04g077960在如下-任一项中的应用：增加番茄根毛长度；提高番茄对土壤的固着能力；改善番茄在营养物质贫瘠条件下的生长状况；番茄育种；所述基因Solyc04g077960的核苷酸序列如SEQ ID NO.1所示。2.包含Solyc04g077960的重组表达载体、重组菌和/或转基因细胞系在增加番茄根毛长度、提高番茄对土壤的固着能力和/或改善番茄在营养物质贫瘠条件下的生长状况中的应用；所述基因Solyc04g077960的核苷酸序列如SEQ ID NO.1所示。3.基因Solyc04g077960编码的蛋白在如下-任一项中的应用：增加番茄根毛长度；提高番茄对土壤的固着能力；改善番茄在营养物质贫瘠条件下的生长状况。4.根据权利要求3所述的应用，其特征在于，基因Solyc04g077960编码蛋白的氨基酸序列如SEQ ID NO.2所示。5.一种增加番茄根毛长度的方法，其特征在于，包括使番茄中SEQ ID NO.2所示蛋白的表达量增加和/或活性提高的步骤。6.一种培育根毛长度增加的转基因植物的方法，其特征在于，包括如下步骤：向受体植物中导入基因Solyc04g077960，使基因Solyc04g077960在受体植物中过表达，得到转基因植物；所述转基因植物与所述受体植物相比根毛长度增加。7.根据权利要求6所述的方法，其特征在于，所述受体植物为番茄。8.根据权利要求6所述的方法，其特征在于，利用番茄根毛特异性启动子ProLeEXT1构建的表达载体使基因Solyc04g077960在受体植物中过表达。
说明书desc
技术领域本发明涉及作物分子遗传学技术领域，具体涉及一种增加番茄根毛长度的基因及应用。背景技术根毛是植物根系的重要组成部分，它是植物根表皮细胞凸起形成的组织，是根的一种特殊结构。根毛在植物吸收水分和营养物质、植物响应逆境胁迫以及植物与微生物互作等方面发挥着关键作用。研究表明：长根毛有利于提高植物对土壤的固着能力，改善植物对营养物质的吸收。因此，深入研究植物根毛，对提高作物养分和水分的吸收利用、增加作物产量、改善作物品质具有重要的理论和实践意义。番茄是全世界栽培最广泛的蔬菜作物之一，同时番茄也是植物科学研究的重要模式作物之一。由于不同地区土质和营养成分的差异，番茄的产量受到较大影响。番茄根毛发育直接受环境因素的影响，根毛对于改善番茄吸收水分、营养物质，提高番茄对土壤的固着能力具有重要作用。然而，目前调控番茄根毛生长的分子机制并不明确。阐明调控根毛生长和发育的分子机制不仅有助于我们深入了解植物细胞发育分化的规律，而且能更好的服务于生产实践。因此，通过基因工程结合遗传育种方法，寻找增加番茄根毛长度的基因具有重要的研究和应用价值。发明内容针对上述现有技术，本发明的目的是提供一种增加番茄根毛长度的基因及应用。本发明发掘鉴定出增加番茄根毛长度的基因Solyc04g077960，在番茄中过量表达Solyc04g077960后，能显著增加番茄根毛的长度，有效提高番茄对土壤的固着能力，明显改善番茄在营养物质贫瘠条件下的生长状况。为实现上述目的，本发明采用如下技术方案：本发明的第一方面，提供基因Solyc04g077960在如下-任一项中的应用：增加番茄根毛长度；提高番茄对土壤的固着能力；改善番茄在营养物质贫瘠条件下的生长状况；番茄育种；所述基因Solyc04g077960的核苷酸序列如SEQ
