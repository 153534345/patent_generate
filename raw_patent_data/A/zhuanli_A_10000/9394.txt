标题title
迟缓爱德华氏菌外膜蛋白疫苗及其制备和应用
摘要abst
本发明涉及分子疫苗学领域，具体的说是一种迟缓爱德华氏菌外膜蛋白疫苗及其制备方法和应用。外膜蛋白疫苗含SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原。本发明所构建的重组蛋白疫苗可以保护牙鲆应对迟缓爱德华氏菌的感染，诱导产生特异性抗体，该蛋白对鱼类免疫细胞的免疫活性具有显著促进作用。
权利要求书clms
1.一种迟缓爱德华氏菌外膜蛋白疫苗，其特征在于：外膜蛋白疫苗含SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原。2.按权利要求1所述的迟缓爱德华氏菌外膜蛋白疫苗，其特征在于：所述外膜蛋白疫苗为含SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原的重组诱导表达质粒pETOmp1。3.按权利要求1或2所述的迟缓爱德华氏菌外膜蛋白疫苗，其特征在于：所述重组质粒pETOmp1是以氨基酸序列如SEQ ID No.1为模板，通过同源重组连接至商品化质粒pET28a中，即得到重组质粒pETOmp1。4.一种权利要求1所述的迟缓爱德华氏菌外膜蛋白疫苗的制备方法，其特征在于：所述外膜蛋白疫苗为含有SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原与佐剂混合。5.按权利要求4所述的迟缓爱德华氏菌外膜蛋白疫苗的制备方法，其特征在于：所述抗原为将SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原通过同源重组连接至商品化质粒pET28a中，获得含有抗原的重组诱导表达质粒pETOmp1。6.按权利要求4或5所述的迟缓爱德华氏菌外膜蛋白疫苗的制备方法，其特征在于：将上述制备的重组蛋白用PBS稀释至400μg/ml，而后将稀释液与佐剂等体积混合，作为重组蛋白疫苗用于后续实验。7.按权利要求6所述的迟缓爱德华氏菌外膜蛋白疫苗的制备方法，其特征在于：所述佐剂的制备方法是将体积比为2：5的NaOH和Al23混合液，悬浮于PBS中至浓度为8mg/ml。8.按权利要求7所述的迟缓爱德华氏菌重组蛋白疫苗的制备方法，其特征在于：所述NaOH浓度为质量比5％；Al23浓度为质量比5％。9.一种权利要求1所述的迟缓爱德华氏菌外膜蛋白疫苗的应用，其特征在于：所述外膜蛋白疫苗在制备具有预防和治疗迟缓爱德华氏菌疫苗制剂中的应用。
说明书desc
技术领域本发明涉及分子疫苗学领域，具体的说是一种迟缓爱德华氏菌外膜蛋白疫苗及其制备方法和应用。背景技术迟缓爱德华氏菌是一种革兰氏阴性病原菌，能感染包括鱼类和哺乳动物在内的多种动物。迟缓爱德华氏菌能够引发海水养殖鱼类的迟缓爱德华氏菌病，给我国水产养殖行业造成了严重的经济损失。目前，多种养殖鱼类都有关于迟缓爱德华氏菌引起大规模死亡的报道。针对迟缓爱德华氏菌的肆虐，韩国已发明一种基于灭活全菌的商业化疫苗，在我国由于相关研究起步较晚，迄今为止，仅有1种迟缓爱德华氏菌减毒疫苗获得国家新兽药证书，但尚未大规模应用。因此，开展针对迟缓爱德华氏菌的其他种类的疫苗研发刻不容缓。基于细菌外膜蛋白开发的重组亚单位疫苗是一种有效的手段。亚单位疫苗抗原蛋白在大肠杆菌中异源表达，重组的疫苗抗原蛋白经分离纯化后得到，能够诱发高效的特异性免疫应答。发明内容本发明目的在于提供一种迟缓爱德华氏菌外膜蛋白疫苗及其制备方法和应用。为实现上述目的，本发明采用的技术方案为：一种迟缓爱德华氏菌外膜蛋白疫苗，外膜蛋白疫苗含SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原。所述外膜蛋白疫苗为含SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原的重组诱导表达质粒pETOmp1。所述重组质粒pETOmp1是以氨基酸序列如SEQ ID No.1为模板，通过同源重组连接至商品化质粒pET28a中，即得到重组质粒pETOmp1。所述质粒的构建：重组质粒pETOmp1以氨基酸序列如SEQ ID No.1为模板，碱基序列经过生工生物工程股份有限公司合成后，连接至pET28a中，即得到重组质粒pETOmp1。一种迟缓爱德华氏菌外膜蛋白疫苗的制备方法，所述外膜蛋白疫苗为含有SEQ IDNo.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原与佐剂混合。所述抗原为将SEQ ID No.1中氨基酸所示的迟缓爱德华氏菌外膜蛋白疫苗抗原通过同源重组连接至商品化质粒pET28a中，获得含有抗原的重组诱导表达质粒pETOmp1；其中，质粒pETOmp1经转化至大肠杆菌BL21中，得到转化子BL21/pETOmp1，大肠杆菌BL21/pETOmp1用含有卡那霉素的5ml LB液体培养基培养过夜，将过夜培养的菌液转移到500ml LB液体培养基中，37℃培养，待菌液培养至吸光度OD600≈0.5后，加入终浓度为1mM的IPTG并于16℃培养16个小时。培养完成的菌液经离心收集，速冻解冻，超声破碎后加入裂解液，室温裂解2小时，裂解完全的菌液经离心收集上清，上清液经过镍柱纯化后，即可得到具有序列表SEQ ID No.1中的氨基酸序列的重组蛋白，用于后续作为疫苗。将上述制备的重组蛋白用PBS稀释至400μg/ml，而后将稀释液与佐剂等体积混合，作为重组蛋白疫苗用于后续实验。所述佐剂的制备方法是将体积比为2：5的NaOH和Al23混合液，悬浮于PBS中至浓度为8mg/ml。所述NaOH浓度为质量比5％；Al23浓度为质量比5％。一种迟缓爱德华氏菌外膜蛋白疫苗的应用：所述外膜蛋白疫苗在制备具有预防和治疗迟缓爱德华氏菌疫苗制剂中的应用。本发明具有如下优点：1.高保护性。本发明疫苗对迟缓爱德华氏菌的免疫保护效率达75％。2.无需其他商业化佐剂，本发明所构建的重组蛋白疫苗可以保护牙鲆应对迟缓爱德华氏菌的感染，诱导产生特异性抗体，该蛋白对鱼类免疫细胞的免疫活性具有显著促进作用。附图说明图1为本发明实施例提供的纯化的rOmp1Et电泳图谱，其中，泳道1，分子量标准，泳道2，rOmp1Et蛋白。具体实施方式下面结合实施例对本发明作进一步说明。实施例旨在对本发明进行举例描述，而非以任何形式对本发明进行限制。在本发明实施例中所涉及到的常规性实验方法均采用如下方法：1.DNA序列合成及质粒连接由生工生物工程股份有限公司完成；2.质粒提取使用“美国，Omega
