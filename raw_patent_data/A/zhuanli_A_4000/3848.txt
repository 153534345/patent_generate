标题title
GsHSP基因及其在提高大豆耐盐性和提高豆苗菜中异黄酮的含量的应用
摘要abst
本发明公开了一种GsHSP基因及其在提高大豆耐盐性和提高豆苗菜中异黄酮的含量的应用，该GsHSP基因的序列如SEQ ID NO.1所示，本申请还公开了含有该GsHSP基因的重组载体、转基因细胞或重组菌及其应用，所述GsHSP可提高大豆根系的耐盐能力，过表达可促进大豆在盐碱条件下的根系生长和豆苗菜的产量，并且可应用于提高盐胁迫下豆苗菜中异黄酮含量，可应用于培育高品质的耐盐的GsHSP重组大豆植株，具有很好的市场前景。
权利要求书clms
1.GsHSP基因，其特征在于，序列如SEQ ID NO.1所示。2.含有如权利要求1所述的GsHSP基因的重组载体、转基因细胞或重组菌。3.如权利要求1所述的GsHSP基因、及含有该GsHSP基因的重组载体、转基因细胞或重组菌在提高豆苗菜异黄酮含量中的应用。4.如权利要求1所述的GsHSP基因、及含有该GsHSP基因的重组载体、转基因细胞或重组菌在提高大豆耐盐性中的应用。5.如权利要求1所述的GsHSP基因、及含有该GsHSP基因的重组载体、转基因细胞或重组菌在提高豆苗菜在盐碱条件下的产量中的应用。
说明书desc
技术领域本发明涉及大豆功能基因技术领域，尤其涉及一种GsHSP基因及其在提高大豆耐盐性和提高豆苗菜中异黄酮的含量的应用。背景技术大豆作为我国重要的农业作物，富含优质植物性蛋白质、不饱和脂肪酸、钙、B族维生素和异黄酮等物质，在国民营养中发挥重要作用。近些年来，随着人民生活水平的提高和对物质要求的提升，我国大豆的产量已不能满足人们日益增长的市场需求。因此，提高我国大豆的产量和品质是当前所急需解决的重大民生问题。盐胁迫等逆境因素是影响大豆的产量和品质的主要因素，给大豆生产造成了巨大损失，因此提高大豆品质和抗逆性是解决问题的关键。采用的传统育种方法来改良大豆的不良性状，因为受大豆基因资源匮乏而受到限制；化学药剂的乱使用和过度使用所造成的植物的抗性问题和环保问题，也使化学药剂的应用受到限制。大豆作为一种重要的农作物，含有丰富的营养成分。大豆中含有充足的优质蛋白质、不饱和脂肪酸、钙、异黄酮以及各种维生素等，在我国居民膳食中占据重要位置并且是优质植物性蛋白质的重要来源。大豆异黄酮是一种浅黄色的粉末物质，其气味略苦并且伴有略微的涩味。异黄酮是一种化合物，在结构和生物学上与人体雌激素类似。大豆及其制品在进入人体后，所含有的大豆异黄酮主要在肠道内被代谢和吸收，随之转化成雌马酚等物质，通过毛细血管进入人体的血液循环体系中。大豆异黄酮被公认为是天然活性物质植物雌激素，是一种被广泛接受的HRT天然替代品或HRT补充剂。大豆异黄酮作为一种生理活性物质，在人体的生命活动中发挥重要的生理功能，在减轻女性衰老、改善更年期症状、保护心血管、抵抗肿瘤、预防骨质疏松、保护神经系统、降低血糖、保护肝脏、调节免疫系统等方面发挥重要作用。由于异黄酮的种种营养和保健生理功能，越来越受到人们的认可。而其中，大豆苷作为大豆异黄酮重要组成部分，在预防和治疗心血管疾病、抗肿瘤等具有药理功效。而大豆黄苷作为大豆异黄酮的另一种重要组成成分，其在抗过敏和抗血栓中发挥重要作用。提高大豆的耐盐性有利于扩大大豆种植面积、提高大豆相关产品产量，以及如何提高豆苗菜等大豆类食品中异黄酮的含量，提高大豆类食品的营养保健品质，都是目前研究者重点关注的问题。因此，现有技术有待进一步改进。发明内容针对上述问题，本发明提供了一种GsHSP基因及其在提高大豆耐盐性和提高豆苗菜中大豆苷和大豆黄苷的含量的应用，首次发现GsHSP可提高大豆根系的耐盐能力，其过表达可促进大豆的根系生长，GsHSP过表达能进一步提高盐胁迫下豆苗菜中大豆苷与大豆黄苷的含量，可应用于培育耐盐性能的GsHSP重组大豆植株，提高其栽种面积、产量和品质。为解决上述问题，本申请提供的技术方案如下：第一方面，本发明提供GsHSP基因，其序列如SEQ ID NO.1所示。申请人首次从极耐盐野大豆品种中挖掘到盐胁迫响应基因GsHSP，对它进行了克隆和生物信息学分析，通过实时荧光定量PCR分析了该基因的组织表达特性和盐胁迫表达模式，通过构建基因过表达载体GsHSP和过表达大豆株系，发现该GsHSP能提高大豆根系中的耐盐能力，提高盐胁迫下大豆苗内豆苗菜异黄酮代谢物含量，可用于培育优质大豆株系。例如，可采用上述基因构建GsHSP过表达重组大豆，实验证明该基因的过表达重组大豆具有优良的奶盐能力，其在盐胁迫下的根系长势优于野生型，促进豆苗菜的增产，体内大豆苷与大豆黄苷的含量显著高于野生型，更具营养价值。第二方面，本发明提供含前述的GsHSP基因的重组载体、转基因细胞或重组菌。所述重组载体为现有的克隆质粒或表达质粒等载体。优选地，所述重组菌为农杆菌。第三方面，本申请还提供上述GsHSP基因、及含有该GsHSP基因的重组载体、转基因细胞或重组菌在提高豆苗菜异黄酮含量中的应用。实验证明，GsHSP过表达豆苗菜中，大豆苷和大豆黄苷的含量得到明显提升，尤其是GsHSP过表达能进一步提高盐胁迫下豆苗菜中大豆苷与大豆黄苷的含量，可利用该基因的过表达培育优质豆苗菜。实验证明，GsHSP的过表达可提高盐胁迫下豆苗菜中大豆苷与大豆黄苷的含量；而GsHSP的基因敲除大豆的染料木苷和大豆黄素的含量显著提升。第四方面，本申请提供所述GsHSP基因在提高大豆耐盐性中的应用。实验证明，在盐胁迫条件下，GsHSP过表达组根系长势、侧根根长和数量要高于WT组，这说明GsHSP可以提高大豆根系的耐盐能力，促进盐碱条件下大豆根系的生长。第五方面，本申请还提供前述的GsHSP基因在提高豆苗菜在盐碱条件下的产量中的应用。由于发现GsHSP的过表达促进豆苗菜在盐胁迫条件下的根系生长，进而有助于提高豆苗菜在盐胁迫下的产量。第六方面，本申请还提供一种GsHSP重组大豆植株的培育方法，其包括以下步骤：设计第一引物对扩增前述的GsHSP的基因序列，在GsHSP的基因序列两端增加酶切位点，并将其扩增产物进行酶切后将其连接至植物过表达载体上，得到GsHSP植物过表达载体；将构建好的GsHSP植物过表达载体转化发根农杆菌K599，并进行大豆根系侵染，得到GsHSP重组大豆植株。优选地，所述表达载体采用pSuper1300。优选地，第一引物对的上游引物的序列如SEQ ID NO.2所示；第一引物对的下游引物的序列如SEQ ID NO.3所示。该GsHSP重组大豆植株为GsHSP过表达重组大豆植株，其具有优良的性状，具有较好的耐盐能力，在盐胁迫条件下，其根系长势、侧根根长和数量要高与野生型，且体内大豆苷与大豆黄苷的含量远高于野生型，具有更好的品质，具有更好的应用前景。第七方面，本申请还提供一种GsHSP基因敲除大豆植株的培育方法，其包括以下步骤：将设计的GsHSP基因敲除靶向位点序列引物对进行混合反应，形成oligo二聚体，再将oligo二聚体插入到Cas9/gRNA载体中，转化大肠杆菌感受态细胞，挑取阳性单菌落送测序，选择测序正确的载体，得到CRISPR/Cas9-GsHSP基因敲除载体；将构建好的CRISPR/Cas9-GsHSP基因敲除载体转化发根农杆菌K599，并进行大豆根系侵染，得到GsHSP基因敲除大豆植株。优选地，所述第二引物对的上游引物的序列如SEQ ID NO.4所示；第二引物对的下游引物的序列如SEQ ID
