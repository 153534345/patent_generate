标题title
一种非编码RNA的新用途以及由其构建的链球菌
摘要abst
本发明涉及口腔疾病治疗与预防技术领域，具体涉及一种非编码RNA的新用途以及由其构建的链球菌。一种非编码RNA在制备致龋能力降低的变异链球菌中的应用，所述非编码RNA的序列如SEQ ID NO.1所示。本方案研究发现非编码RNA在无龋儿童牙菌斑中表达量增加，发明人构建了ASvicK的过表达株，证实ASvicK可调控vicK基因表达，并发现ASvicK可改变变异链球菌生长形态、干扰变异链球菌胞外多糖代谢及致龋性。本方案解决了现有技术中缺少具有理想龋病治疗效果以及制备方法简单的变异链球菌突变株的技术问题，获得的菌株可用于调整口腔菌群失调，减少高致龋能力的变异链球菌数量，预防龋病的发生及发展。

说明书desc
技术领域本发明涉及口腔疾病治疗与预防技术领域，具体涉及一种非编码RNA的新用途以及由其构建的链球菌。背景技术龋病是一种发生于牙齿硬组织的慢性进行性的感染性疾病，其发生发展与牙菌斑生物膜密切相关。变异链球菌是龋病发生发展过程中最重要的致病菌，其可利用食物中的糖类，产生有机酸，合成胞外多糖；从而引起耐酸的致龋菌群活性增加及微生态失衡，导致龋病的发生。变异链球菌合成的胞外多糖是重要的毒力因子与物质能量来源，胞外多糖的量、组分与结构与龋病发生发展息息相关。同时胞外多糖也为细菌的生存、细菌对环境变化的耐受性、细菌对防龋药物的抵抗性都提供了重要的保护作用。因此，针对变异链球菌胞外多糖的研究在龋病防治中具有重要的意义。双组份信号转导系统，是变异链球菌适应性、生存和毒力表达调控中的必需组成部分。其中，双组份信号转导系统VicRK和CovR信号系统与变异链球菌生物膜形成、胞外多糖合成以及产酸等致龋特性密切相关。VicRK双组份系统中，组氨酸蛋白激酶VicK可感受环境刺激信号，传递磷酸化信号，活化同源反应调节因子VicR，VicR被活化后调节细菌生物学特性。如果能够获得VicK或VicR表达水平降低的变异链球菌菌株，就可以控制该菌株的生长速度以及生物膜形成能力，并降低细菌粘附能力和胞外多糖含量，从而降低该菌株的致龋相关能力。但是，现有技术构建vicK或vicR缺陷株的操作较为复杂，且获得的缺陷株的治疗效果不理想，亟需研发一种制备过程简单且具有更好的防龋能力的变异链球菌菌株。发明内容本发明意在提供一种非编码RNA在制备预防龋病的变异链球菌中的应用，以解决现有技术中缺少具有理想龋病治疗效果以及制备方法简单的变异链球菌突变株的技术问题。为达到上述目的，本发明采用如下技术方案：一种非编码RNA在制备致龋能力降低的变异链球菌中的应用，所述非编码RNA的基因序列如SEQ ID NO.1所示。本方案还提供了一种致龋能力降低的变异链球菌，其保藏号为CCTCC NO.M20211257。本方案还提供了含有一种致龋能力降低的变异链球菌的微生物制剂。本方案还提供了一种致龋能力降低的变异链球菌在制备预防或者治疗龋病的制剂中的应用。本方案的原理及优点是：序列如SEQ ID NO.1所示的非编码RNA具有降低变异链球菌vicK基因表达的功效，从而可以改变变异链球菌生长形态、干扰变异链球菌胞外多糖代谢及致龋性。针对龋病高发，且没有特效药物防治的现状，本方案通过构建反义vicK基因突变株，获得提供一种产胞外多糖能力、生物膜形成能力、产酸能力显著下降的变异链球菌。该突变株可以应用在预防龋病方面，并制成微生态制剂，作为变异链球菌的替代菌株，调整口腔菌群失调，减少高致龋能力的变异链球菌数量，预防龋病的发生。该ASvick变异链球菌现保存于中国典型培养物保藏中心，保藏号为CCTCC NO.M 20211257，株系名称为ASvicKpDL278。发明人前期筛选和实验发现，AsvicK在无龋儿童牙菌斑中表达量增加，该非编码RNA与变异链球菌vicK基因部分编码区mRNA互补。在此基础上，发明人构建了ASvicK的过表达株，证实ASvicK可调控vicK基因表达，并发现ASvicK可改变变异链球菌生长形态、干扰变异链球菌胞外多糖代谢及致龋性。龋病属于生态失衡性疾病，其产生与牙菌斑生物膜的沉积密切相关，变异链球菌是龋病发生发展过程最重要的致病菌。当耐酸环境的致龋菌群活性增加，胞外多糖合成代谢作用加强，牙菌斑生物膜结构致密性及其对牙面的黏附作用增加，从而打破了菌群微生态平衡导致龋病的发生。胞外多糖是致龋菌最重要的毒力因子以及物质能量来源，是其生存过程必须的“保护衣”。牙菌斑生物膜细菌在胞外多糖的保护作用下，使得药物制剂的效果大打折扣。而本技术方案所获得的ASvicK的过表达株具有生物膜形成能力下降和胞外多糖含量减少等特性，可以作为微生物制剂，与普通变异链球菌形成种内竞争关系，从而抑制牙菌斑生物膜结构的形成，起到防龋护牙的效果。综上所述，本技术方案的有益效果在于：本方案通过利用AsvicK的性能构建过表达株，显著降低胞外多糖的含量以及相关基因的表达，通过靶向牙菌斑生物膜最基本的基质成分，从而调控菌斑微生物生态环境。本方案是控制菌斑、降低其致龋性的理想方法，对龋病的防治和口腔健康的维护有重要的意义。将本方案反义vicK基因突变株制成微生态制剂，可以调整口腔菌群失调，减少高致龋能力的变异链球菌数量，预防龋病的发生及发展。进一步，所述非编码RNA用于降低变异链球菌中的VicK蛋白的含量。VicRK双组份系统中，组氨酸蛋白激酶VicK可感受环境刺激信号，传递磷酸化信号，活化同源反应调节因子VicR，VicR被活化后调节细菌生物学特性。本技术方案的非编码RNA能够降低VicK蛋白表达水平，从而控制变异链球菌菌株的生长速度以及生物膜形成能力，并降低细菌粘附能力和胞外多糖含量，从而降低该菌株的致龋相关能力。进一步，致龋能力降低的变异链球菌由如下方法制备：制备表达所述非编码RNA的重组质粒；再将重组质粒转入转化前变异链球菌中，获得所述变异链球菌。通过构建能够表达本方案的非编码RNA的表达质粒、并将表达质粒转入普通变异链球菌中的方法，可以获得稳定表达ASvick的变异链球菌株系，是一种高效且易操作的方式。比起现有技术的基因敲除方法，本技术方案更为简单可行且适合于广泛应用。进一步，将序列如SEQ ID NO.1的所述非编码RNA的基因片段整合在空载质粒上，获得重组质粒。序列如SEQ ID NO.1的核苷酸片段为ASvick的编码基因，将其整合在空载质粒上获得表达载体，可在变异链球菌内大量表达Asvick，从而实现降低vick基因和vick蛋白的含量水平的目的。进一步，所述转化前变异链球菌的菌株为UA159。变异链球菌UA159为本技术领域常见菌株，易于获取并且对其进行分子操作的技术较为成熟，在此基础上进行变异株系的研究和改造，可获得较高的变异株系构建的成功率。进一步，在所述微生物制剂中，所述变异链球菌的OD600值为0.15-0.2，浓度为108CFU。控制变异链球菌的浓度为108CFU且OD600值为0.15-0.2，可保证细菌具有较为理想的活性，能够更好地预防龋病的发生。进一步，所述微生物制剂在下颌磨牙区的接种量为200μl每侧，并且所述微生物制剂用于涂抹于牙齿表面。将微生物制剂涂抹于牙齿表面，并且将其同时接种于下颌磨牙区，可以有效实现细菌定值，进而实现防龋病的功效。附图说明图1为本发明实施例1的无龋儿童及重度低龄儿童龋的牙菌斑ASvicK相对表达量。图2为本发明实施例2的ASvicK的二级结构预测结果示意图。图3为本发明实施例2的pDL278/ASvicK及pDL278/vicK重组质粒酶切凝胶电泳验证结果。图4为本发明实施例2的UA159、ASvicK及SmuvicK+突变株的显微镜观察图片。图5为本发明实施例2的UA159、ASvicK及SmuvicK+突变株的ASvicK的相对表达量的检测结果。图6为本发明实施例2的UA159、ASvicK及SmuvicK+突变株的生长曲线和CFU曲线。图7为本发明实施例3的结晶紫含量测定生物膜总量的实验结果。图8为本发明实施例3的CLSM观察生物膜形貌的实验结果。图9为本发明实施例3的SEM观察生物膜形貌的实验结果。图10为本发明实施例3的生物膜中水不溶性葡聚糖和水溶性葡聚糖的含量检测结果。图11为本发明实施例3的生物膜状态下调控胞外多糖代谢和调控乳糖代谢相关基因表达水平检测结果。图12为本发明实施例3的菌株生物膜乳酸产量以及乳酸脱氢酶活性检测结果。图13为本发明实施例4的动物实验流程示意图。图14为本发明实施例4的体式显微镜观察大鼠窝沟龋损情况的图像。图15为本发明实施例4的对大鼠龋齿的Keyes评分统计结果。图16为本发明实施例4的激光共聚焦显微镜观察大鼠牙菌斑生物膜的实验结果。图17为本发明实施例4的扫描电镜观察大鼠牙菌斑生物膜的实验结果。具体实施方式实施例1：临床分离株分析从无龋/重度低龄儿童龋中分离采集牙菌斑，在通过现有技术的常规手段培养分离变异链球菌，使用常规RT-qPCR法检测了分离获得的变异链球菌中AsvicK的相对表达量，实验结果如图1所示，图中展示了无龋儿童及重度低龄儿童龋中的牙菌斑ASvicK相对表达量。ASvicK在无龋儿童中表达增加，提示ASvicK与致龋性密切相关。RT-qPCR引物具体如下：第一链合成翻译引物：5’-TCACGTATTGATAATCAGACCAGTC-3’；正义链PCR引物：3’-GTCGTTAGTTGATTATTCCAAATGC-5’。实施例2：ASvicK的过表达株的构建重组质粒的构建本方案用到的长链非编码RNA，将其命名为反义vicK
