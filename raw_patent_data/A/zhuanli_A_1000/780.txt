标题title
抗人神经生长因子的抗体
摘要abst
本发明涉及特异性结合NGF的中和性抗体或其抗原结合片段，以及产生这种抗体的制备方法。本发明还涉及所述抗体在治疗与NGF相关的疾病或病症，尤其在缓解或改善个体疼痛中的应用。
权利要求书clms
1.特异性地结合人神经生长因子的人抗体或其抗原结合片段在制备用于减轻、缓解疼痛或关节炎的药物中的用途，其中所述抗体或其抗原结合片段包含序列如SEQ IDNO：7所示的重链可变区和序列如SEQ IDN O：8所示的轻链可变区。2.特异性地结合人神经生长因子的人抗体或其抗原结合片段在制备用于减轻、缓解疼痛或关节炎的药物中的用途，其中所述抗体或其抗原结合片段包含重链和轻链，所述重链包含序列如SEQ ID NO：7所示的重链可变区和序列如SEQ ID NO:9所示的重链恒定区，所述轻链包含序列如SEQ IDN O：8所示的轻链可变区和序列如SEQ ID NO:10所示的轻链恒定区。3.权利要求1或2所述的用途，其中所述抗原结合片段选自Fab、Fab’-SH、Fv、scFv或2片段。4.权利要求1或2所述的用途，其中所述疼痛是病理生理性疼痛。5.特异性地结合NGF的人抗体或其抗原结合片段在制备用于与第二治疗剂共治疗应用以减轻、缓解疼痛或关节炎的药物中的用途，其中所述抗体或其抗原结合片段包含序列如SEQ ID NO：7所示的重链可变区和序列如SEQ IDN O：8所示的轻链可变区。6.特异性地结合NGF的人抗体或其抗原结合片段在制备用于与第二治疗剂共治疗应用以减轻、缓解疼痛或关节炎的药物中的用途，其中所述抗体或其抗原结合片段包含重链和轻链，所述重链包含序列如SEQ ID NO：7所示的重链可变区和序列如SEQ ID NO:9所示的重链恒定区，所述轻链包含序列如SEQ IDN O：8所示的轻链可变区和序列如SEQ ID NO:10所示的轻链恒定区。7.权利要求5或6所述的用途，其中所述第二治疗剂选自其他神经营养素、止痛剂、抗炎剂、化学治疗剂或免疫治疗剂。8.权利要求5或6所述的用途，其中所述疼痛是病理生理性疼痛。
