标题title
亚洲小车蝗UAP基因及其在蝗虫防治中的应用
摘要abst
本发明公开了一种亚洲小车蝗UAP基因及其在蝗虫防治中的应用。其中亚洲小车蝗UAP基因，核苷酸序列如SEQ ID NO:1所示。亚洲小车蝗UAP基因RNA干扰序列片段，核苷酸序列如SEQ ID NO:2所示。亚洲小车蝗UAP基因的RNA干扰序列片段合成的dsRNA。该dsRNA在亚洲小车蝗防治中的应用。本发明通过RNA干扰亚洲小车蝗几丁质代谢，使亚洲小车蝗因蜕皮困难死亡，从而达到生物防治的目的，它可减少化学杀虫剂使用，具有极大的生产应用价值。
权利要求书clms
1.一种亚洲小车蝗UAP基因，其特征在于：核苷酸序列如SEQ ID NO :1所示。2.根据权利要求1所述的亚洲小车蝗UAP基因RNA干扰序列片段，其特征在于：核苷酸序列如SEQ ID NO :2所示。3.根据权利要求2所述的亚洲小车蝗UAP基因的RNA干扰序列片段合成的dsRNA。4.根据权利要求3所述的dsRNA在亚洲小车蝗防治中的应用。
说明书desc
技术领域本发明涉及生物技术领域，具体涉及亚洲小车蝗UAP基因及其在蝗虫防治中的应用。背景技术目前，亚洲小车蝗仍然主要依靠化学防治，开发新型环保、可替代化学防治的方法变得尤为紧迫。RNA干扰是一种由双链RNA分子引起的特异性转录后基因沉默的现象，具有效率高、特异性强、操作简便等特点被应用于动植物表型和抗性基因功能鉴定的相关研究中，沉默必要基因导致昆虫生长迟缓、发育缺陷，甚至死亡。因此，学者将其称为第四代杀虫剂。几丁质是昆虫表皮及围食膜等部位的重要组成成分，其含量受到几丁质代谢途径中相关酶基因的调控，由于人和其他高等动物并没有几丁质，因此昆虫几丁质代谢中的关键酶基因已被公认为新型杀虫剂作用的靶标。UDP-N-乙酰氨基葡萄糖焦磷酸化酶基因催化可逆反应GlcNAc-1-P+UTP变成UDP-GlcNAc+PPi，由此产生几丁质合成酶的前体物质UDP-GlcNAc，进一步合成几丁质，在昆虫生理代谢中发挥着重要作用，通过RNA干扰亚洲小车蝗几丁质代谢，使亚洲小车蝗因蜕皮困难死亡，从而达到生物防治的目的，提供环保治蝗技术，是目前亟需进行的研究课题。而基于RNAi进行害虫防治的前提是筛选靶标序列获得对昆虫具有高致死作用的dsRNA。因此，现有技术中亟需一种亚洲小车蝗UAP基因、亚洲小车蝗UAP基因RNA干扰序列片段及亚洲小车蝗UAP基因RNA干扰序列片段合成的dsRNA在亚洲小车蝗防治中的应用。发明内容本发明的目的是为了克服现有技术存在的不足，提供一种亚洲小车蝗UAP基因、亚洲小车蝗UAP基因RNA干扰序列片段及亚洲小车蝗UAP基因RNA干扰序列片段合成的dsRNA在亚洲小车蝗防治中的应用。为实现以上目的，本发明是通过以下技术方案实现的：一种亚洲小车蝗UAP基因，核苷酸序列如SEQ ID NO :1所示。设计相应的简并引物后，通过PCR扩增获得其保守区，产物经琼脂糖凝胶电泳检测其片段长度为1504 bp，同时经测序获得其全长，并命名为OasiUAP2，其全长为2157 bp，开放阅读框为1482 bp，编码493个氨基酸。亚洲小车蝗UAP基因RNA干扰序列片段，核苷酸序列如SEQ ID NO :2所示。是根据SEQ ID NO:1设计上下游引物经PCR扩增获得。进一步利用相关试剂盒，根据相应的干扰序列片段合成dsRNA。本发明提供一种亚洲小车蝗UAP基因的RNA干扰序列片段合成的dsRNA。上述dsRNA在亚洲小车蝗防治中的应用。注射dsRNA到昆虫2-3腹节，结果表明，dsRNA可以特异性地沉默基因OasiUAP2的mRNA表达，根据畸形表型及存活率可以直观看到干扰后的相应基因对亚洲小车蝗的影响。本发明的有益效果是：本发明提供一种亚洲小车蝗UAP基因、亚洲小车蝗UAP基因RNA干扰序列片段及亚洲小车蝗UAP基因RNA干扰序列片段合成的dsRNA在亚洲小车蝗防治中的应用，通过RNA干扰亚洲小车蝗几丁质代谢，使亚洲小车蝗因蜕皮困难死亡，从而达到生物防治的目的，减少化学杀虫剂使用，具有极大的生产应用价值。附图说明图1为琼脂糖凝胶检测亚洲小车蝗OasiUAP2转录组cDNA序列长度；图2为注射dsOasiUAP2后亚洲小车蝗表型变化；图3为注射亚洲小车蝗dsOasiUAP2在12、24、48和72
